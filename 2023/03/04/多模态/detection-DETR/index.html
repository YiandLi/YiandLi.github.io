DETR : DEtection TRansformer

DETR: DETR: End-to-End Object Detection With Transformers
代码: DETR网络结构详解（结合代码）

模型架构

  • 输入图像维度 [3, H0, W0] ， CNN 下采样得到 [C,H,W] ，然后降维到 [d,H,W] ，最后拉直为 [d, HW] 。
    这里降维下采样的尺寸大小决定最终保留多少个 token ，决定了最终 attention 的计算效率，这里使用 1/32 的下采样尺寸。

  • transformer encoder
    位置编码使用 sin/cos 初始化，并且固定住；
    位置编码是 2d 的，分别在x和y方向做 positional encoding ，考虑两者拼接后和原始输入一致，这里初始化为输入维度的 1/2 。
    和 Transformer 不同的是，这个位置编码不仅仅加在第一层（如图），而是每一层 attention layer 都加入这个静态的编码，不是动态变化的。

  • transformer decoder

    • Attention 不进行掩码
    • 输入可以学习的 query token ( initially set to zero ) + learnable positional encoding ( also called object queries)
    • 第一层 Self-Attention 中，：学习位置编码是直接加在 QK 上的；
      第二层中 Attention 中：可学习位置编码 加在 query token 的 Q 上，query token 的 K 是需要加上 sin 位置编码的。
    • 总之，V 都不需要加位置编码，只有 query token 的 Q 的位置编码需要学习；其他的都是固定的 sin 位置编码。
    • 第一层的 Self Attention 层可以删除。

  • FFN 分类器：

    • a 3-layer perceptron with ReLU activation function ：输出 bbox四个值（中心点坐标+宽度高度）
    • a linear projection layer ： 输出输出物体类别预测值

  • 推理：
    最终每一个 query token 都可以对应一个 object ，然后使用二分图匹配，使每一个 object 仅对应一个 query token 的 bbox，就不需要 NMS 来降低冗余了。

Deformable DETR

DEFORMABLE DETR: DEFORMABLE TRANSFORMERS FOR END-TO-END OBJECT DETECTION
代码解读：https://zhuanlan.zhihu.com/p/372116181 ； https://zhuanlan.zhihu.com/p/596303361

‘Deformable’ ： 可变形的

DETR 问题

  1. slow convergence - 收敛慢，这主要是因为DETR中的self-attention，初始状态下对每个位置的attention都是平均的，所以需要更长时间让attention层学会注意到那些稀疏的但重要的位置 。
  2. limited feature spatial resolution - 小目标性能差，这是因为没有用类FPN的多尺度结构，而且self-attention的计算量和输入尺寸的平方成正比，如果使用小 patch，则效率极差 。

两者其实都是 transformer 架构的弊端，
于是作者借鉴 Deformable Convolution ，提出 可变形注意力机制 – 每个特征像素不必与所有特征像素交互计算，只需要与部分基于采样获得的其它像素交互，并且这些采样点的位置是可学习的。这是一种局部(local)和稀疏(sparse)的高效注意力机制，能够解决DETR收敛慢与能够处理的特征分辨率受限的问题。

Method

Multi-Scale Features

在提取特证的部分，与 DETR 仅仅下采样 32 不同，该工作使用了不同的下采样率：

  • 骨干网络提取特征（左）：一共有 3 层， C3C4C5C_3 C_4 C_5C3​C4​C5​ 来自于 ResNet 。
  • 颈部网络提取特征（右）：一共有 4 层，用于降维 。
  • 特征序列化：将最后统一的特征全部拼接到一起，组成最终的序列 。

作者将下采样得到的输出称为 feature map 。

Scale-Level Embedding

因为平行采样了多个 feature map 结果，我们需要区分不同的 feature map ，所以这里引入 Scale-Level Embedding 。

同一特征层中的所有特征点会对应相同的 scale-level embedding，于是有几层特征就使用几个不同的 scale-level embedding 。

这个scale-level embedding是随机初始化并且是随网络一起训练的、是可学习的 。

在实际使用中 这个 embedding 与基于三角函数公式计算的 position embedding 相加在一起作为位置信息的嵌入。

Deformable Attention

Standard Attention：

MultiHeadAttn⁡(zq,x)=∑m=1MWm[∑k∈ΩkAmqk⋅Wm′xk]\operatorname{MultiHeadAttn}\left(z_q, \boldsymbol{x}\right)=\sum_{m=1}^M \boldsymbol{W}_m\left[\sum_{k \in \Omega_k} A_{m q k} \cdot W_m^{\prime} x_k\right] MultiHeadAttn(zq​,x)=m=1∑M​Wm​[k∈Ωk​∑​Amqk​⋅Wm′​xk​]

zqz_qzq​ 看作query, 由 xxx 经过线性变换生成, qqq 是对应的索引; Ωk\Omega_kΩk​ 即所有的 kkk 集合, kkk 是key的索引
mmm 代表是第几个注意力头部, WmW_mWm​ 是对注意力施加在value后的结果进行线性变换从而得到不同头部的输出结果, Wm′W_m^{\prime}Wm′​ 用于将 xkx_kxk​ 变换成value, AmqkA_{m q k}Amqk​ 代表归一化的注意力权重。

可以看到，每一个 q 都需要和其他所有 k,v 进行交互。

可变形注意力 ：query不是和全局每个位置的key都计算注意力权重，而是仅在附近位置中采样 k 个 Key + Value（k=3），然后进行稀疏 Attention 。

Deformable Attention：

DeformAttn⁡(zq,pq,x)=∑m=1MWm[∑k=1KAmqk⋅Wm′x(pq+Δpmqk)]\operatorname{DeformAttn}\left(z_q, \boldsymbol{p}_q, \boldsymbol{x}\right)=\sum_{m=1}^M W_m\left[\sum_{k=1}^K A_{m q k} \cdot W_m^{\prime} x\left(p_q+\Delta \boldsymbol{p}_{m q k}\right)\right] DeformAttn(zq​,pq​,x)=m=1∑M​Wm​[k=1∑K​Amqk​⋅Wm′​x(pq​+Δpmqk​)]

变化为：

  1. 输入部分增加了 pqp_qpq​ ，表示当前 zqz_qzq​ 的位置，注意仍然是 2d 位置向量，作者称其为参考点(reference points)。

  2. 对每个输入向量，现在在每一个 head 下，选择 KKK 个其他向量进行 attention ， Δpmqk\Delta p_{m q k}Δpmqk​ 表示采样集合点相对于参考点的位置偏移（offsets）。
    和可变卷积一样， Δpmqk\Delta p_{m q k}Δpmqk​ 是可以学习的，通过所有 query token 经过一个全连接层得到。
    通过 pqp_qpq​ 和 Δpmqk\Delta p_{m q k}Δpmqk​ ，可以计算出最终的目标采样点。

  3. 此外，注意力权重 AAA 也是一样，直接由query经过全连接层得到，同时在K个采样点之间归一化。这种情况下，其实就没有了 qk 交互了。

Multi-scale Deformable Attention Module

为了将上述可变注意力应用到多尺度特征上：

MSDeformAttn⁡(zq,p^q,{xl}l=1L)=∑m=1MWm[∑l=1L∑k=1KAmlqk⋅Wm′xl(ϕl(p^q)+Δpmlqk)]\operatorname{MSDeformAttn}\left(z_q, \hat{\boldsymbol{p}}_q,\left\{\boldsymbol{x}^l\right\}_{l=1}^L\right)=\sum_{m=1}^M \boldsymbol{W}_m\left[\sum_{l=1}^L \sum_{k=1}^K A_{m l q k} \cdot W_m^{\prime} \boldsymbol{x}^l\left(\phi_l\left(\hat{\boldsymbol{p}}_q\right)+\Delta \boldsymbol{p}_{m l q k}\right)\right] MSDeformAttn(zq​,p^​q​,{xl}l=1L​)=m=1∑M​Wm​[l=1∑L​k=1∑K​Amlqk​⋅Wm′​xl(ϕl​(p^​q​)+Δpmlqk​)]

变化为：

  1. {xl}l=1L\left\{\boldsymbol{x}^l\right\}_{l=1}^L{xl}l=1L​ 表示不同的下采样率得到的 feature map 。

  2. p^q\hat p_qp^​q​ 表示归一化到了 [0,1][0, 1][0,1] ，可以理解为先不考虑具体的采样率，首先得到偏移量的百分比形式 。

  3. ϕ\phiϕ 为将百分比坐标映射到各个 feature map 中去，用 lll 表示不同的 feature map 。
    每一个 feature map 的尺寸都不一样，但是可以通过百分比来得到偏移位置 。
    这样，每个参考点在所有特征层都会有一个对应的(归一化)坐标，从而方便计算在不同特征层进行采样的那些点的位置 。

在每个特征层采样 KKK 个点，共有 LLL 层特征，从而每层采样 LKLKLK 个点，注意力权重也是在这 LKLKLK 个点之间进行归一化 。
