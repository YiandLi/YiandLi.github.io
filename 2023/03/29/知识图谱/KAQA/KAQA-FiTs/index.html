Paper: FiTs: Fine-grained Two-stage Training for Knowledge-aware Question Answering

Knowledge-aware question answering (KAQA): 根据一个 knowledge base 来回答问题

当前 SOTA KAQA 可以被分成两类：

  1. semantic parsing-based (SP-based) methods：对问题进行句法分析，生成逻辑符号
  2. information retrieval-based (IR-based) methods：先从 KG 中抽取相关知识（KG retrieval），然后进行联合推理（joint reasoning）。

但是 IR-based 存在两个问题：

  1. Modality Gaps：两种模态的语意和分布存在差异
  2. Difficulties in Joint Reasoning：对所提供的两种模式的知识进行联合推理比较困难

为了解决上述问题，作者提出了 a Fine-grained Two-stage training framework (FiTs) ，一种两阶段的训练框架：

  1. knowledge adaptive post-training：将 KG 和 PLM 的表征信息进行对齐 -> a better starting point to the joint reasoning module
  2. knowledge-aware fine-tuning：在对齐的表征上继续训练，增强推理能力。
    除了有监督的QA任务外，还额外使用到两个自监督辅助任务，用于弥补从 KG 中抽取出的知识不完整，并且存在噪音。

Modality Interaction

用 modality interaction layer (MInt) 来将 token h~intl\tilde{\mathbf{h}}_{i n t}^lh~intl​ 和 node e~intl\tilde{\mathbf{e}}_{i n t}^le~intl​ 这两个表征向量进行融合。

然后将表示文本的 interaction token 和表示图的 interaction node 进行拼接，使用一个 two-layer MLP 来交互这两个表征：

[hintl;eintl]=MINT⁡([h~intl;e~intl])\left[\mathbf{h}_{i n t}^l ; \mathbf{e}_{i n t}^l\right]=\operatorname{M INT}\left(\left[\tilde{\mathbf{h}}_{i n t}^l ; \tilde{\mathbf{e}}_{i n t}^l\right]\right) [hintl​;eintl​]=MINT([h~intl​;e~intl​])

至此更新完了一层，使用 interaction token 和 interaction node 进行交互，相当于排两个代表去进行信息融合。

Dis-advantages

这种表征方式的缺点：

  1. PLM 和 KG 是在不同语料上训练的，所以他们的表征有时候会相互冲突，给 joint reasoning 带来困惑
  2. 两者分布不一致，即使是相近的语意，因为训练方式不同，其向量也有可能是相似度很低的。

Knowledge Adaptive Post-training

为了弥补这种 gap，得到一种更好的初始化，作者提出了一种 knowledge adaptive (KA) loss，并且给 PLM 加上了 MLM loss( 15% mask )，组合起来进行联合 post-training。

首先使用 cross-modality encoder 编码后得到 token representation HM={hintM,h1M,…,hTM}\mathbf{H}^M= \left \{\mathbf{h}_{i n t}^M, \mathbf{h}_1^M, \ldots, \mathbf{h}_T^M \right \}HM={hintM​,h1M​,…,hTM​} 和 KG node representation EM={eintM,e1M,…,eKM}\mathbf{E}^M=\left\{\mathbf{e}_{i n t}^M, \mathbf{e}_1^M, \ldots, \mathbf{e}_K^M\right\}EM={eintM​,e1M​,…,eKM​} 。

然后通过 token pooling 得到句子中的实体向量 EL={eintL,e1L,…,eJL}\mathbf{E}^L=\left\{\mathbf{e}_{i n t}^L, \mathbf{e}_1^L, \ldots, \mathbf{e}_J^L\right\}EL={eintL​,e1L​,…,eJL​} ， eiL\mathbf{e}_{i}^LeiL​ 表示输入 Q−AQ-AQ−A 中的每一个实体。
同时从所有节点表征 EM\mathbf{E}^MEM 中挑选出实体表征 EG={eintG,e1G,…,eJG}\mathbf{E}^G=\left\{\mathbf{e}_{i n t}^G, \mathbf{e}_1^G, \ldots, \mathbf{e}_J^G\right\}EG={eintG​,e1G​,…,eJG​} 。

然后作者使用对比学习框架将 EL\mathbf{E}^LEL 和 EG\mathbf{E}^GEG 对齐，这里没有使用点乘相似度，而是 concat + MLP 来得到概率分数：

Knowledge-aware Fine-tuning

在微调阶段，为了更好地利用两个子编码模型的知识，这里额外引入两个自监督学习目标，和 QA supervision 一起进行训练。

QA Supervision

对一个 n-way/choice 的样本 (c,q,A)(c,q,A)(c,q,A) ，选项 ai∈Aa_i\in Aai​∈A 被选中的概率为：

p(ai∣q,c)∝exp⁡(MLP⁡(hintM,eintM,g))p\left(a_i \mid q, c\right) \propto \exp \left(\operatorname{MLP}\left(\mathbf{h}_{i n t}^M, \mathbf{e}_{i n t}^M, \mathbf{g}\right)\right) p(ai​∣q,c)∝exp(MLP(hintM​,eintM​,g))

LSup=−∑i=1nyilog⁡(p(ai∣q,c))\mathcal{L}_{S u p}=-\sum_{i=1}^{n} y_{i} \log \left(p\left(a_{i} \mid q, c\right)\right) LSup​=−i=1∑n​yi​log(p(ai​∣q,c))

Knowledge Source Distinction

KG retrieval model 的精度很重要，既需要抽取出正确的能引导答案的子图结构，又需要避免抽取出其他无关的结构带来噪音。
但是实际上，这个检索器，很容易抽取出无关的模型。这个时候我们就需要让模型可以自行辨别抽取出来的节点到底是否有效。

由于缺少监督数据，这里作者使用一种启发式方法。首先手动在子图中添加 kirrk_{irr}kirr​ 个无关实体，然后得到节点编码为：

EG=(e1G,e2G,…,em+kirrG)\mathbf{E}^G=\left(\mathbf{e}_1^G, \mathbf{e}_2^G, \ldots, \mathbf{e}_{m+k_{i r r}}^G\right) EG=(e1G​,e2G​,…,em+kirr​G​)

然后使用双层 MLP，进行四分类：

LKSD=−∑i=1m+kirr∑j=14yijlog⁡(y^ij)y^i=softmax⁡(W3ReLU⁡(W2eiG+b1))\begin{gathered} \mathcal{L}_{K S D}=-\sum_{i=1}^{m+k_{i r r}} \sum_{j=1}^4 y_{i j} \log \left(\hat{y}_{i j}\right) \\ \hat{\mathbf{y}}_i=\operatorname{softmax}\left(\mathbf{W}_3 \operatorname{ReLU}\left(\mathbf{W}_2 \mathbf{e}_i^G+\mathbf{b}_1\right)\right) \end{gathered} LKSD​=−i=1∑m+kirr​​j=1∑4​yij​log(y^​ij​)y^​i​=softmax(W3​ReLU(W2​eiG​+b1​))​

(1) linked to entities in the question
(2) linked to entities in the candidate answer
(3) an entity in the retrieved multi-hop neighborhood but not directly linked to mentioned entities
(4) an irrelevant entity.
