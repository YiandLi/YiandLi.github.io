PCBERT

Posted on 2023-05-28 In NLP , Named entity recognition
Symbols count in article: 1.9k Reading time ≈ 9 mins.

本文适用于 中文低资源NER 任务

  1. 对于中文场景：前面有很多工作尝试融入外部词典信息，取得了很好的效果(Flat，Lattice LSTM，LEBERT) ，在此基础上，作者额外为每个索引出来的词进行entity分类，额外融入entity label 信息。
  2. 对于低资源场景：前面有工作使用 prompt tunning 缓解数据压力，作者这里先在高资源数据集上进行prompt tunning ，得到 parent model，然后在低资源数据集上进行 prompt inference，得到伪标签，将这一部分信息给到 child 模型，作为额外的伪标签特征。

总之就是特征工程玩到头了，给我们启发迁移学习模型也可以用于提供额外特征。

下面是正文：

Using Transfer Learning for Low-Resource NER task, two challenges:

  1. the semantic distribution difference between the source and target domains;
  2. the same category labels have different definitions in different datasets.

Author proposes a Parent and Child BERT(PCBERT) : few-shot / zero-shot NER

  1. a prompt-based label transfer approach:
    1. train an annotating model on high- resource datasets
    2. discover more implicit labels on low-resource datasets
  2. introduce an enhanced lexicon feature
    1. leverage the lexicon feature of transferring model to enhance Chinese word boundary distinction ability
    2. design a label extension strategy to achieve label transferring from high-resource datasets

Label Extension Strategy

DP(X)\mathcal{D}_P\left(X\right)DP​(X) 表示输入文本的语义分布
E(L)\mathcal{E}\left(L\right)E(L) 表示标签的语义分布

则迁移学习范式下，模型的表现能力 ppp 和语义分布的相关性可以表示为：

p∝DP(XS)∩DP(XT)DP(XS)−DP(XT)p∝DE(LS)∩DE(LT)DE(LS)−DE(LT)\begin{aligned} p & \propto \frac{\mathcal{D}_P\left(X_S\right) \cap \mathcal{D}_P\left(X_T\right)}{\mathcal{D}_P\left(X_S\right)-\mathcal{D}_P\left(X_T\right)} \\ p & \propto \frac{\mathcal{D}_{\mathcal{E}}\left(L_S\right) \cap \mathcal{D}_{\mathcal{E}}\left(L_T\right)}{\mathcal{D}_{\mathcal{E}}\left(L_S\right)-\mathcal{D}_{\mathcal{E}}\left(L_T\right)} \end{aligned} pp​∝DP​(XS​)−DP​(XT​)DP​(XS​)∩DP​(XT​)​∝DE​(LS​)−DE​(LT​)DE​(LS​)∩DE​(LT​)​​

即和相交的语义分布呈正比，和不相交仅在原始source数据集中的分布呈反比。

作者认为 DE(LS)∩DE(LT){\mathcal{D}_{\mathcal{E}}\left(L_S\right) \cap \mathcal{D}_{\mathcal{E}}\left(L_T\right)}DE​(LS​)∩DE​(LT​) 中反映了 source 和 target 标签集合语义的并集，有可能包含了 Target label set 没有，但是 Source label set 中存在的语义信息。

所以本文考虑拓展 target 集 TTT 中的标签特征，最直接的就是用在 Source 集合上训练的标注模型在 Target 集合上打伪标签，但是存在 label noise 问题：

  1. poor annotation model performance
  2. the same category labels may explain the different meanings between S and T .

Method

Parent-Bert : a prompt-based model , to extract the implicit label extension features
Child-Bert : a lexicon-based model (inspired by LEBERT) , to incorporates multi-label features of each lexicon

Training steps:

  1. P-BERT pre-trains on the label extension dataset.
  2. froze P-BERT , provide label extension features to fine-tune the C-BERT

P-BERT

在打上伪标签的集合上进行 prompt tunning，用于给 C-BERT 提供 prompt feature 。

首先将文本放入一个 prompt 模版中： [CLS] X [SEP] TP(X,Y)\text{[CLS] X [SEP] TP(X,Y)}[CLS] X [SEP] TP(X,Y)
TP(X,Y)\text{TP(X,Y)}TP(X,Y) 为 label templates ，形式为 [Index] is [Z]\text{[Index] is [Z]}[Index] is [Z] ， [Index]\text{[Index]}[Index] 为token position id， Z\text{Z}Z 为 label text slot。每个 label slot 都被 Pad 成为相同的长度。

在 Prompt tunning 过程中，将 label text slot Z\text{Z}Z 的输入token 设置为 [MASK]\text{[MASK]}[MASK] ，目标是重构 annotation 。

C-BERT

中文 ner 的难点在于没有 explicit entity boundary ， 研究已经证明为每个 token 融入词典信息可以缓解。

输入部分为 P-BERT 和 自己的 word embedding ，每个 token 编码维度为 HHH 。

融入辞典特征

  1. 融入外部词典特征：为每个 token 构造词典集合，每个词都来自外部词典，并且是输入序列的子序列，并且进行编码：

融入 P-BERT 信息

实验

Randomly sample a small train set from each original dataset
The train set sampling sizes K are 250, 500, 1000, and 1350 (the max size of Weibo is 1350)

The high-resource dataset is integrated with multiple datasets, includ- ing CLUENER (Xu et al., 2020), CNERTA (Sui et al., 2021), RenMinRiBao (Xia et al., 2005), and datasets from unknown sources.
从这一点看来，为了充分发挥 P-BERT 的能力，这里 source 数据集越大越丰富越好。

Baseline：BERT,BERT-LC,Lattice,LSTM,FLAT,LEBERT,LEBERT-LC

均超过 baseline

vs ablation models

removing the P-BERT component : 去掉 pbert 后效果下降，但是仍然优于 lebert，表明 label 特征有效

removing the label extension strategy (LEA)：效果下降，表明额外的 pbert 伪标签有效

replace the label extension dataset with the high-resource dataset to train the P-BERT (LEB)：效果下降明显，表明当语义gap大时，直接迁移效果不好，这里用 prompt 的方式迁移效果比较好。

总结

  1. 在融入外部词典的基础上，额外融入了标签特征，这个标签特征可以是离线准备好的
  2. 使用 prompt tunning 构造伪标签，思路简单但是确实没见到过
  3. 这里 attention 区别传统点乘，是用了一个纺射矩阵
