Sparrow

DeepMind - Improving alignment of dialogue agents via targeted human judgements

A information-seeking dialogue agent, using:

  1. reinforcement learning form human feedback
  2. break down the requirements for good dialogue into natural language rules : collect more human judgements, and reward model
  3. our agent provides evidence from sources supporting factual claims when collecting preference judgements over model statements

method

collect human data for

  1. rule violations

    → train a rule reward model (rule RM) → predicts whether a given rule was violated

  2. per-turn response preferences

    → train preference reward models (preference RMs)

然后在训练时，使用强化学习在上面两个模型给出的信号上训练LM模型

Rule

helpful , correct , harmless

divide each rule into more detailed rules

  • Helpfulness rules : include answering user questions, staying on topic, and avoiding
    common problems such as repetition, and are combined with an overall per-turn response preference
  • Correctness rules : cover types of incorrect statements which raters might not otherwise
    penalize, such as the agent claiming to have a physical body or take real-world actions, and are combined with the evidence-specific rewards
  • Harmless : 查阅了现有文献来确定潜在的失败模式，然后编写规则脚本并寻找模型无法遵循规则的示例

Generating dialogue turns

Base-moel/Agent : DPC (Dialogue-Prompted Chinchilla) Chinchilla-70B

组件：User， Agent，Search Query，Search Result

后面两个用于从数据库（Google Search）中检索事实依据

首先拼接 the prompt, dialogue history, and participant name ，得到输入上下文

然后进行检索，抓取 Search Result ，然后和上面内容进行拼接，生成最终的输出

根据是否在知识库中进行搜索，有几种不同的方式：

Evidence

Learning to search:

对每个输入，进行四次生成。两次是不检索直接生成，两次是检索生成。

然后人工选择最好的，这样可以提供监督信号：1. 是否检索 2. 如何生成是更好的

Retrieval： Search Query 是由模型生成的，Search Result 是每个网页搜索答案中截取最多500字

human feedback for evaluation

  1. 模型检索的频率
  2. 模型检索出的内容对回答有效的频率

作者使用人工进行标注

Reward models

both fine-tuned from Chinchilla 70B

Response preference reward models (preference RMs) ：

在几个候选生成中人工选择更好的，然后训练一个preference RM 去预测分数

除了候选分数之外， 额外增加一个其他prompt的生成回复 ： penalize off-topic answers

增加两个辅助损失函数：

  1. a classification loss predicting whether evidence conditioned answers were supported and plausible
  2. ask raters to indicate when all responses in a comparison are low quality and regularize the corresponding Elo scores to be negative.

Rule Violation Reward Model (Rule RM) ：检验是否 break the rule

使用指令微调，人工诱导模型打破规则，生成 specific failure

a conditional classifier 𝑟(𝑥, 𝑦) ∈ [0, 1] to show whether 𝑦 was violated by Sparrow

ReRanking

当使用 model@N 进行生成时，会先让模型生成 N 个回复，然后使用两个 RM model 打分，将分数最高的模型输出

训练部分

  1. 首先 fine tune → a better initial starting point for RL

  2. Reinforcement learning：使用上述训练的两个 RW Model 进行RL

    每个 episode 是一个 a single statement conditioned on a preceding dialogue context

    每个 action 是一个 token，reward 在每个 episode 结束时进行计算得出

这里训练的每一个 episode 为 self-play 模式，即 episode 来自于 dialogue context ，

All statements after the initial dialogue context are generated by Sparrow

Sparrow generates multiple turns for a dialogue, playing the role of User, Agent, and Search Query (Search Results are retrieved programmatically) .
