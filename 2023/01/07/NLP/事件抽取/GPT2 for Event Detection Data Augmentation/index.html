GPT2 for Event Detection Data Augmentation

Posted on 2023-01-07 In NLP , Event Extraction
Symbols count in article: 1.4k Reading time ≈ 7 mins.

Paper: Unleash GPT-2 Power for Event Detection

任务：
Event Detection (ED)：识别出句子中的触发词，并且判断事件类型。但是本文中指代的 ED 任务其实是下方的 EC。
Event Identification (EI)：识别出句子中的触发词，本文解决方法为 token 级别的二分类任务。
Event Classification (EC)：根据给定的触发词判断事件类型，多分类任务。

针对标注数据不足的情况，作者尝试使用 GPT-2 去生成训练数据；并且提出了新的 teacher-student 模型框架和新的目标函数，来减少生成样本的噪音。

Data Augmentation

使用原始训练集 $\mathcal{O} $ 去 fine-tune GPT-2，然后让 GPT-2 去生成样本。
为了降低误差，这里生成的句子仅仅标注出 trigger，不会生成具体的事件类型。

训练集中的句子 $ S\in \mathcal O$ ， S=w1,w2...,wnS = w_1,w_2...,w_nS=w1​,w2​...,wn​ 中只有一个 trigger。
首先对其进行拓展，使用两个特殊 token 来标记 trigger，得到 S′=[BOS,w1,...,TRGs,wt,TRGe,...,wn,EOS]S'=[BOS,w_1,...,TRG_s,w_t,TRG_e,...,w_n,EOS]S′=[BOS,w1​,...,TRGs​,wt​,TRGe​,...,wn​,EOS] ， wtw_twt​ 为 trigger， TRGsTRG_sTRGs​ 和 TRGeTRG_eTRGe​ 为特殊标记，用来提示模型具体的 trigger 信息。

然后使用 S′S'S′ 微调 GPT-2 ，使用微调后的 GPT-2 生成新的增强样本数据集 $\mathcal{G} $ ，该数据集和原始训练数据集大小一致。
然后得到了新的数据集 A=O∪G\mathcal A = \mathcal O \cup \mathcal GA=O∪G 。

在生成时，进行后处理，只有同时包含 TRGsTRG_sTRGs​ 和 TRGeTRG_eTRGe​ 的样本才会被选择。

Task Modeling

生成出来的样本大量的噪音，所以这里作者提出 teacher-student 架构进行训练。
该架构包含两个模型，teacher model 和 student model 的模型架构一致。

Base Model

使用 Bert-base + Bi-LSTM 进行编码，得到：

H=[h1,h2,...,hn]H=[h_1,h_2,...,h_n] H=[h1​,h2​,...,hn​]

ED

使用 Max pooling 得到句子表征：

h=MAX_POOL(H)h=\text{MAX\_POOL}(H) h=MAX_POOL(H)

然后拼接 trigger 的表征 ttt 和句子表征 hhh 得到 VVV ，放入到线性分类器中判断事件类型。

训练损失函数为

Lpred=−log⁡P(l∣S,t)\mathcal L_{pred}= -\log P(l|S,t) Lpred​=−logP(l∣S,t)

lll 为标签

EI

因为生成的数据集中没有事件类型标签 lll ，所以这里引入一个辅助任务 EI：对 HHH 中的每个token进行二分类，损失函数为CE损失：

Laux=−(lauxlog⁡(FFaux(V))+(1−laux)log⁡(1−FFaux(V)))\mathcal L_{aux} = -(l_{aux} \log(\mathrm{FF}_{aux}(V) ) + (1-l_{aux}) \log(1-\mathrm{FF}_{aux}(V) )) Laux​=−(laux​log(FFaux​(V))+(1−laux​)log(1−FFaux​(V)))

FFaux\mathrm{FF}_{aux}FFaux​ 为二分类器

在原始数据 $\mathcal{O} $ 上得到 Lpred\mathcal L_{pred}Lpred​ ，在全部数据 A\mathcal AA 上得到 Laux\mathcal L_{aux}Laux​ ，构成最终的损失。

Knowledge Consistency

因为生成的数据存在噪音，所以这里使用 teacher-student 框架进行训练：

  1. 首先在 O\mathcal OO 上训练得到 teacher model，用于存储 anchor knowledge。
  2. 然后冻结 teacher model，在 A\mathcal AA 上让 teacher model 指导 student model 进行训练。

leverage the consistency guidance; prevent drastic divergence from anchor knowledge

  1. 预测时使用 student mode

训练中 teacher model 和 student model 都使用两个任务 ED + EI进行训练，

  • Teacher model 在原始数据集 O\mathcal OO 上进行训练，损失为：

Lpred+τLaux\mathcal L_{pred} + \tau \mathcal L_{aux} Lpred​+τLaux​

  • Student model
    • 在混合数据集 A\mathcal AA 上进行 EI 任务的训练
    • 在原始数据集 O\mathcal OO 上进行 ED 任务的训练

训练 student model 时，除了上面两个任务之外，针对 ED 和 EI，作者又提出了两个新的目标函数来确保 knowledge consistency。

ED

两个模型都是在 O\mathcal OO 上进行训练的
给定句子 S∈OS\in \mathcal OS∈O 和 trigger 标注，让两个模型分别预测对应的事件类型，得到 Pt(⋅∣S,t)P_{t}(\cdot \mid S, t)Pt​(⋅∣S,t) 和 Ps(⋅∣S,t)P_{s}(\cdot \mid S, t)Ps​(⋅∣S,t) ，下标表示不同的模型。

我们想让最小化 student model 和 student model 的 KL散度：

LKL=−Σl∈LPt(l∣S,t)log⁡(Pt(l∣S,t)Ps(l∣S,t))\mathcal{L}_{K L}=-\Sigma_{l \in L} P_{t}(l \mid S, t) \log \left(\frac{P_{t}(l \mid S, t)}{P_{s}(l \mid S, t)}\right) LKL​=−Σl∈L​Pt​(l∣S,t)log(Ps​(l∣S,t)Pt​(l∣S,t)​)

这里不是对称的，注意顺序，是想让 student 去学 teacher。

实现上和蒸馏一致，但是目标是不一样的，我们这里是想要确保 knowledge consistency，减少噪音的影响。

EI

因为 G\mathcal GG 是生成出来的，所以作者想使用「原始数据和生成数据之间的差异」作为训练标准，即模型能够识别出两者的差异。
具体地，student model 应该和 teacher model 有相同的「判别差异」的能力。
对于离散的分布 X\mathcal XX 和 Y\mathcal YY ，他们的距离为：

 Dist W=Σx∈XΣy∈Yπ∗(x,y)C(x,y)\text { Dist }_{W}=\Sigma_{x \in \mathcal{X}} \Sigma_{y \in \mathcal{Y}} \pi^{*}(x, y) C(x, y)  Dist W​=Σx∈X​Σy∈Y​π∗(x,y)C(x,y)

π∗(x,y)\pi^{*}(x, y)π∗(x,y) 表示两个样本之间的联合概率分布， CCC 为距离度量函数。

使用 dG,OTd_{\mathcal G,\mathcal O}^TdG,OT​ 和 dG,OSd_{\mathcal G,\mathcal O}^SdG,OS​ 表示两个模型判别出的 G\mathcal GG 和 O\mathcal OO 的距离，则损失函数为：

Ldist =∣dO,GT−dO,GS∣∣B∣\mathcal{L}_{\text {dist }}=\frac{\left|d_{\mathcal{O}, \mathcal{G}}^{T}-d_{\mathcal{O}, \mathcal{G}}^{S}\right|}{|B|} Ldist ​=∣B∣∣∣∣​dO,GT​−dO,GS​∣∣∣​​

∣B∣|B|∣B∣ 为 batch size


如何定义两个数据之间的距离呢？
作者将其定义为将 O\mathcal OO 转换为 G\mathcal GG 的代价（the cost of transforming O\mathcal OO to G\mathcal GG ）。

这里使用 Optimal Transport (OT)，给定两个句子 Xo∈OX_o\in \mathcal OXo​∈O 和 Yg∈GY_g\in \mathcal GYg​∈G ，他们之间的距离为句子表征之间的欧式距离，句子表征使用 每个token 表征的 max-pooling 。

如何定义两个数据的联合概率？
使用 FFaux\mathrm{FF}_{aux}FFaux​ + softmax 表示某个句子的概率分布，联合概率应该是向乘吧。

Loss

总的 loss function 为：

L=Lpred +αLaux +βLKL+γLdist \mathcal{L}=\mathcal{L}_{\text {pred }}+\alpha \mathcal{L}_{\text {aux }}+\beta \mathcal{L}_{K L}+\gamma \mathcal{L}_{\text {dist }} L=Lpred ​+αLaux ​+βLKL​+γLdist ​
