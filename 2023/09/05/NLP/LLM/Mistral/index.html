  2. Sliding Window Attention (SWA) ： handle longer sequences at smaller cost

    因为上面每个 input token 仅仅 attend 前面 4096 个 token ，那么 KV cache 也就只需要维护前面 4096 个 token 即可。

    当当前的 position id 大于窗口 w=4096 的时候，将当前计算的值覆盖 i % w 位置 ，注意这里 position id 就不是从前向后顺序增加了。
