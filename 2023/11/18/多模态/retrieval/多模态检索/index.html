Method

Inter-modal Consistent Multi-modal Contrastive Loss

传统方法使用对比学习损失，但是仅仅考虑到跨模态样本间的相似度，忽略了同模态 文本相似度。
作者提出 Consistent Multimodal Contrastive (CMC) loss ，同时考虑 intra-modal 和 inter-modal ranking loss。
给定 image-text pair (I,T)(I,T)(I,T) ，可以通过单模空间内的最近邻居，得到 hard negatives sample，比如可以找到距离 III 最近的其他图像 Iv−I_{v-}Iv−​ ，从而得到 hard negative pair，语言模态的 TTT 也是一样，最终得到两个 hard negative pair ：
(Iv−,Tv−)\left(I_{v^{-}}, T_{v^{-}}\right)(Iv−​,Tv−​) 和 (Il−,Tl−)\left(I_{l^{-}}, T_{l^{-}}\right)(Il−​,Tl−​) 。
对比损失为 ：

Lr=max⁡(0,δ−S(I,T)+S(I,Tl−))+max⁡(0,δ−S(I,T)+S(Iv−,T))\begin{aligned} L_{\mathrm{r}} & =\max \left(0, \delta-S(I, T)+S\left(I, T_{l^{-}}\right)\right) \\ & +\max \left(0, \delta-S(I, T)+S\left(I_{v^{-}}, T\right)\right) \end{aligned} Lr​​=max(0,δ−S(I,T)+S(I,Tl−​))+max(0,δ−S(I,T)+S(Iv−​,T))​

一个 hinge loss，让负样本相似度比正样本相似度大 δ\deltaδ ，这个损失仅仅考虑了跨模态。

Intra-modal Consistent Multi-modal Contrastive Loss
将 image-text pair (I,T)(I,T)(I,T) 看作一个整体，为了保证图文对表征之间的一致性，这里定义单一模态下的对比学习（the contrastive loss within the same modality）：

La=max⁡(0,∣S(I,Il−)−S(T,Tl−)∣−σ)+max⁡(0,∣S(I,Iv−)−S(T,Tv−)∣−σ)\begin{aligned} L_{\mathrm{a}} & =\max \left(0,\left|S\left(I, I_{l^{-}}\right)-S\left(T, T_{l^{-}}\right)\right|-\sigma\right) \\ & +\max \left(0,\left|S\left(I, I_{v^{-}}\right)-S\left(T, T_{v^{-}}\right)\right|-\sigma\right) \end{aligned} La​​=max(0,∣S(I,Il−​)−S(T,Tl−​)∣−σ)+max(0,∣S(I,Iv−​)−S(T,Tv−​)∣−σ)​

σσσ 是一个松弛变量，保证了不同模态下，不同样本之间的相对距离可以具有特定的差距，并且可能不完全一致，以放宽样本之间距离的约束。
如何理解呢？也就是对于两样本对 (I,T),(I′,T′)(I,T),(I',T')(I,T),(I′,T′) ，如果单模态下相似度 S(I,I′)S(I,I')S(I,I′) 比较小，那么我们希望另一模态下的相似度 S(T,T′)S(T,T')S(T,T′) 也要比较小才好。

The training loss combines both global and local retrieval tasks and jointly learns both global and local representations of two modalities (image and text) in an end-to-end trainable network.

Two-Stage Inference
three ways to calculate the similarity between two samples:

  1. Use only global features : Sg(X,Y)S_g(X, Y)Sg​(X,Y)
  2. Use only local features : Sl(X,Y)S_l(X, Y)Sl​(X,Y)
  3. Use two features at the same time to obtain mixed similarity

Sgl(X,Y)=(1−θ)Sg(X,Y)+θSl(X,Y)S_{g l}(X, Y)=(1-\theta) S_g(X, Y)+\theta S_l(X, Y) Sgl​(X,Y)=(1−θ)Sg​(X,Y)+θSl​(X,Y)

2-Stage inference:

  1. Use global similarity SgS_gSg​ to obtain the top K candidate samples quickly
  2. Use Mixed similarity SglS_{g l}Sgl​ to re-rank only the top K samples

More specifically, the pre-trained VinVL [13] is used to extract image and text features. It should be emphasized that the VinVL encoder employs cross-attention operation and requires image-text pairs as the input. In contrast, our approach com- prises two independent branches which receive image and text features, respectively. For simplicity, <Image,Text> denotes an image-text pair. We use <Mask,Image> and <Text,Mask> as the input of the VinVL encoder to extract image and text features, respectively, where Mask operation masks off all the elements for this modality. Through dual transformer encoders ITR and TTR, global and local representations for each modality are subsequently learned.

Retrieval-Enhanced Contrastive Vision-Text Models

retrieval-enhanced contrastive (RECO) vision-text models：a method to equip existing vision-text models with the ability to use relevant external knowledge in a cross- modal manner for improved zero-shot predictions.

动机：CLIP 在罕见的或甚至在预训练数据集中不存在的细粒度实体上仍然存在困难。本文作者探究 Retrieval-Augmented CLIP-like model，具体让 CLIP 根据跨模态的检索结果调整目标文档/图像的 embedding，提升 Zero-shot 能力。

Method

在使用VLM时，根据输入，在外部知识库中检索结果，然后通过跨模态的结果对输入表征进行调整。

Visual Semantic Loss (VSL) - ICME2023

Image-text Retrieval via Preserving Main Semantics of Vision

图像中存在大量信息冗余，会导致错误匹配。本文提出 Visual Semantic Loss ，让模型关注图像的重要内容。

给定三元组 (v,t,t−)(v,t,t^-)(v,t,t−) / (t,v,v−)(t,v,v^-)(t,v,v−) ， t−t^-t− 和 v−v^-v− 分别表示负样本，使用 triplet loss 作为基本目标函数：

mmm : the margin parameter

该 loss 仅仅在正负样本层面进行训练，并没有考虑到正样本中更细粒度的信息所带来的干扰。
比如文本“A man walking a dog.” ，如果正样本图片中存在其他在目标文本中没有出现的元素（secondary content of the image），比如汽车，这个 loss 就没有让模型将注意力完全集中在人和狗的身上。

Pre-Screening Mechanism

Efficient Image-Text Retrieval via Keyword-Guided Pre-Screening

a coarse-grained gallery pre-screening：
如果查询text 是关于 dog 的，那么图片中主体为人类的图片就可以直接被排除了。
所以作者首先预测出 text 和 images 的关键词，然后通过匹配计算来预先筛出明显不匹配的样本。

于是作者推出一个 pre-screening 机制，该机制可以用作任何现有检索方法的前向步骤，用于召回关键词一致的候选，之后我们就可以在召回的候选的基础上进行双塔 / 单塔的检索了。

作者将这个关键词预测问题转化为一个多标签问题，问题来到：

  1. 如何保证分类器的正确性 -> use the booming image-text retrieval technique
  2. 如何减小分类器的时间复杂度 -> use the inverted index in the search engine, with O(1)O(1)O(1) querying time complexity
