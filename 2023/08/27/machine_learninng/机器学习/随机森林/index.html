随机森林

Posted on 2023-08-27 In Machine Learning
Symbols count in article: 1.7k Reading time ≈ 8 mins.

属于集成学习（Ensemble）下的Bagging（Bootstrap Bagging）方法：

  • 采样 T 个容量为 m 的训练集，设置 T 棵决策树，产生 T 个分类结果。
  • 对分类任务：使用投票法；对生成任务：使用平均法。

如果 base model 复杂度为 O(m)O(m)O(m) ，采样+投票/平均 复杂度为 O(s)O(s)O(s) ，则总复杂度为 T(O(m)+O(s))T(O(m)+O(s))T(O(m)+O(s)) ， O(s)<<O(m)O(s)<<O(m)O(s)<<O(m) ，那么训练一个集成的模型的复杂度和训练一个 base model 的复杂度是 同阶 的。

关于模型，我们希望：

  1. base model 之间有比较大的差异 <- 采样集合之间差异较大。
  2. 每个 base model 不太差 <- 采样的数据不能太少。

于是采样策略为：
如果训练集大小为N，对于每棵树而言，随机且有放回地从训练集中的抽取N个训练样本（bootstrap sample），作为该树的训练集。
每棵树的训练集都是不同的，而且里面包含重复的训练样本，不容易过拟合。

如果样本量无限大，则某个样本不会被抽到的概率为 lim⁡n→∞(1−1n)n\lim_{n\rightarrow \infty} (1-\frac 1n)^nlimn→∞​(1−n1​)n ，这个值的极限是e，为 0.360.360.36 。
所以对于某一次采样，整个样本约有 36.8%36.8\%36.8% 的数据不会被抽到，这部分数据被称为外包数据(Out of Bag, OOB)。同样地，每一个样本不会参与 36.8%36.8\%36.8% 的树的训练。

这种采样策略带来另一个优点：包外估计（out-of-bag estimate）：

  1. base model 为 决策树：使用包外样本来辅助剪枝，或者使用包外样本辅助对零训练样本节点的处理 ？
  2. base model 为神经网络：充当 dev 辅助训练过程 。
  3. 得到 base model 的权重。

bagging 的优点：

  1. 泛化能力提升，组合多个很好的 base model 。
  2. 计算：缓解 base model 进入局部极小值 。
  3. 表示：最优解可能不在表示空间，通过平均方法让假设空间变大 -> swa 。

如何 bagging ？
在 base model 效果差不多时候，使用简单平均；效果有差距时，使用加权平均。

训练树

  • 设每个样本有 d 个属性，在决策树的每个节点需要分裂时， 随机从这 d 个属性中选取出 k 个属性 ，满足条件 k << d。
  • 然后从这 k 个属性中选择 1个最优属性 作为该节点的分裂属性，直到无法继续分裂。
  • 每棵树都尽最大程度的生长，没有剪枝过程。

一般的 bagging 算法的「多样性」仅仅来自样本扰动，即使用「随机采样的数据」来训练。而随机森林引入了属性扰动，即「随机选择 k 个属性」，使得 base model 之间差异度进一步增加。

参数

决策树的数量 和 抽取的特征数量

  • 随机森林分类效果（错误率）与两个因素有关：
    • 森林中任意两棵树的相关性：相关性越大，错误率越大；
    • 森林中每棵树的分类能力：每棵树的分类能力越强，整个森林的错误率越低。
  • 减小特征选择个数m，树的相关性和分类能力也会相应的降低；增大m，两者也会随之增大。
    所以关键问题是如何选择最优的m（或者是范围）

包外误差

训练每棵树时，都有一部分样本没有参加训练，这部分样本可以作为测试集，统计预测误差，称为包外误差。
和交叉验证类似，这部分验证的误差可以作为泛化误差的估计。

简化流程

根据实验，随着决策树的数量增加，包外误差和测试误差都会下降。
1）对每个样本，计算它作为oob样本的树（约1/3的树木）对它的预测；
2）然后以简单多数投票作为该样本的分类结果；简单平均作为回归结果。
3）最后用误分个数占样本总数的比率作为随机森林的oob误分率。

这个包外误差可以用来确定决策树的数量，即通过观察误差何时停止下降来决定何时终止训练。

特征重要性

随机森林可以在训练过程中计算特征的重要性，即置换法。
原理：如果某个特征对分类很重要，那么对于某个样本，改变这个特征值之后，样本的预测结果就会出现错误。
即分类结果对这个特征值很敏感。反之，如果某特征值对分类不重要，则随便改变也不会影响分类结果。

对于每棵决策树，使用相应的OOB(袋外数据)数据来计算它的袋外数据误差,记为 err1err_1err1​ 。
随机地对袋外数据OOB所有样本的某一特征X加入噪声干扰(就可以随机的改变样本在特征X处的值)，再次计算它的袋外数据误差,记为 err2err_2err2​ 。

一些总结

  • 随机森林属于Bagging算法。
  • 超参数为：
    • 训练单棵树的特征数量
    • 决策树的数量
    • 每棵决策树的设置，如最少的叶节点数量
  • 当数据总特征很多，但是有效特征（relevant predictors）比较少的时候，随机森林效果不好。
    因为每棵树的都会弱化这些有效特征，导致每棵树都是弱分类器。
  • 增加树的数量不会降低方差，即不会导致过拟合。
    但是如果树的数量太多，会导致树木相关性增强，从而增大方差。
  • 融合模型的鲁棒性不会低于单棵决策树。
