方法 - Differentiable Discrete Proxy

目标：preserve the top-K important tokens，decrease Flops，retain performance

组件结构： a token importance metric + a re-parameterization trick

Token Sorting

pruning：使用 cls attention 过程中计算得到的 qk attention score 来度量 token 的重要性，然后排序，然后直接裁剪掉第 l 层分数最低的 NαpN \alpha_pNαp​ 个 visual token ，即剩余 N(1−αp)N(1 - \alpha_p)N(1−αp​) 个 token 。

merging：接下来使用余弦相似度来衡量「 N(αm−αp)N(\alpha_m - \alpha_p)N(αm​−αp​) 不重要的 token」与「 N(1−αm)N(1 - \alpha_m)N(1−αm​) 其余 token」之间的相似度。对于相似的 token 对，通过直接取平均值来生成一个新 token。

Compression Rate Re-parameterization

使用 重参数技巧 让 pruning rate 和 merging rate 变得可导 。

设置 rate 集合 C={C1,C2,…,CN}\mathbf{C}=\left\{C_1, C_2, \ldots, C_N\right\}C={C1​,C2​,…,CN​} ，其中 Ck=k−1NC_k=\dfrac{k-1}{N}Ck​=Nk−1​ 表示删除了 k−1k-1k−1 个 tokens 。

然后设置集合中元素 CkC_kCk​ 被采到的概率 ρk\rho_kρk​ ， ∑k=1Nρk=1\sum_{k=1}^N \rho_k=1∑k=1N​ρk​=1 。

最终的压缩率为 α=∑k=1NCkρk\alpha=\sum_{k=1}^N C_k \rho_kα=∑k=1N​Ck​ρk​ 。

于是问题变成了求最优概率值 ρ\rhoρ 。

Token-Level Mask

通过累加采样概率，
第 k 个 important token 被 mask 的概率为：

π1=0,πk=ρN+2−k+⋯+ρN−1+ρN,k≥2\pi_1=0, \pi_k=\rho_{N+2-k}+\cdots+\rho_{N-1}+\rho_N, k \geq 2 π1​=0,πk​=ρN+2−k​+⋯+ρN−1​+ρN​,k≥2

重要性最高的 token 永远会被保留，其余的如果大于 α\alphaα 才进行保留。

为保证训练推理一致性，这里通过 概率 得到 mask value：

mk={0,πk≥α1,πk<αm_k=\left\{\begin{array}{l}0, \pi_k \geq \alpha \\ 1, \pi_k<\alpha\end{array}\right. mk​={0,πk​≥α1,πk​<α​

Dynamically Prune Vision Tokens

参数：

  • one ranking function f\mathcal ff
  • two parameters: filtering layer KKK , filtering ratio R%R\%R%

过程：

  • 在 LVLM（即llama/vicuna）的第 k 层，使用 ranking func 进行排序，删除 top-R% 个 visual token 。
  • 排序的依据为：the average attention-score one token received from all other tokens
