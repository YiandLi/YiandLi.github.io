TabPedia

Posted on 2024-10-24 In MultiModal
Symbols count in article: 689 Reading time ≈ 5 mins.

Intro

Visual Table Understanding (VTU):
(i) 表格检测（Table Detection, TD），从 doc images 中分类出 table image；
(ii) 表格结构识别（Table Structure Recognition, TSR），解析以表格为中心的图像中的表格结构；
(iii) 表格查询（Table Querying, TQ），从指定位置的整张图像中识别表格结构
(iv) Table Question Answering (TQA)

当前使用 LVLM 解决 VTU 的问题：
(i) 表示格式之间的差异（二维结构与一维序列）
(ii) 所需的图像分辨率

本文：
(i) 提出 TabPedia ，a novel large vision-language model ，统一 TAU Task 。
(ii) 引入 concept synergy mechanism (概念协同机制)，以协调表格感知(table perception)和理解任务(comprehension tasks)。
通过在框架中引入冥想 token (meditative token)，TabPedia 自适应地利用 multi-source visual embeddings 和 task instructions ，生成准确且合理的响应。
(iii) 提出 a new table VQA benchmark, ComTQA

Method

Model Architecture

High-Resolution Vision Encoder.

high-resolution image 很重要，作者这里使用 Swin-B 作为 backbone，来编码高分图片。

  1. 给定输入图片 III ，首先 scale 到固定分辨率 H×W=2,560×1,920H × W=2,560 × 1,920H×W=2,560×1,920 上 ，得到 IhI_hIh​ 。
    在调整图像大小的过程中保持了长宽比，以防止表格内容和结构的失真。
  2. 然后通过 Swin-B 进行编码，最终得到降采样 1/32 后的 visual token VhV_hVh​ ，维度为 1024 。

Low-Resolution Vision Encoder.

为得到全局信息，同样 scale 到 224 × 224 的低分表示上，然后用 Clip ViT L/14 编码，得到 VlV_lVl​ ，即 256 个 1024 的 visual token 。

Projections.

高分辨率部分 VhV_hVh​ ，token 数量过多，Projector 是一个卷积层（2D,KernelSize3,Step3），来继续降采样，然后 flatten ，得到 H64×W64\frac{\mathrm{H}}{64} \times \frac{\mathrm{W}}{64}64H​×64W​ 个 visual token，记作 V^h\hat V_hV^h​ 。

底分辨率部分 VlV_lVl​ ，参考 LLaVA，Projector 是一个 a linear layer 进行维度转换，记作 V^l\hat V_lV^l​ 。

Concept Synergy.

输入： V^h, V^l, Instruction Q\hat V_h ,\ \hat V_l,\text{ Instruction } QV^h​, V^l​, Instruction Q
LLM： Vicuna-7B

考虑到表格感知任务与理解任务的差异，引入 meditative tokens MMM ，用于实现概念协同，即感知不同维度的 visual token 和 Instruction 。

X=[Q,<IMG_S>;V^l;<IMG_SEP>;V^h;< IMG_E> ; M] X =\left[\mathrm{Q},<\mathrm{IMG} \_\mathrm{S}>; \hat{V}_l ;<\mathrm{IMG} \_\mathrm{SEP}>; \hat{V}_h ;<\text { IMG\_E> } ;\right. \text { M] } X=[Q,<IMG_S>;V^l​;<IMG_SEP>;V^h​;< IMG_E> ; M] 

<IMG_S>, <IMG_E>, <IMG_SEP>\text{<IMG\_S>, <IMG\_E>, <IMG\_SEP>}<IMG_S>, <IMG_E>, <IMG_SEP> 为可学习的 special token 。
