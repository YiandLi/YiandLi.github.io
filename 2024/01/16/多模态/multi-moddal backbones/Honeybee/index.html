Honeybee: Locality-enhanced Projector for Multimodal LLM

Visual Projector 的两个特点：

  1. flexibility in number of visual tokens ：决定推理速度
  2. preservation of visual features ：决定最终的视觉理解能力

基于此，作者提出一个新的 flexible and locality-enhanced projector 。

Introduction

MLLM

多模态大型语言模型 (MLLM) 的目标: 多模态输入 + 输出生成遵循指令的响应

多模态大模型的架构一般为：1) vision encoder, 2) projector, 3) large language model (LLM)

影响效率的组件一般不是 visual encoder 和 projector ，而是 visual token 的数量 。

projector

projector 的作用是将 visual features 转化为 visual token，和 LLM 对齐。

两种类型：

  1. linear projector：使用单个线性层直接进行降维

    • 优点：擅长通过一对一转换来保留视觉特征的局部上下文。 这种对局部性的强烈保护可以实现有效的空间理解。
    • 劣势：token 数量是固定的，降低效率。

  2. abstractor：Q-Former，Re-Sampler

    • 优势：可以自主选择 visual token 的数量
    • 劣势：在空间理解上效果不如 linear projector，因为更倾向于从主要区域抽取上下文，局部感知能力就下降了。

于是作者使用 convolution 和 deformable attention ，提出了两种 locality-enhanced abstractors ， C-Abstractor 和 D-Abstractor ，将局部感知设计注入 abstractor，在效率和效果之间进行了平衡。

同时还提出训练方法，如何根据两种数据：

  1. GPT-assisted instruction-following dataset(LLaVA)
  2. vision-language task datasets with instructization process (Flamingo) ，

最大限度地提高指令数据的效益。

Locality-enhanced Projector

C-Abstractor : Convolutional Abstractor

直接使用 L 个 ResNet block 进行编码，然后根据需要决定 avg pooling 的大小，然后再接 L 个 ResNet 。

比较暴力，没有任何 attention 操作。

D-Abstractor : Deformable attention-based Abstractor

有可能 CNN 引入了了过于严格的局部归纳偏差，所以提出了基于 「Deformable attention 可变形注意力」 的抽象器，能够保证局部上下文信息。

通过 2-D coordinate-based sampling，每一个可学习的 query token 可以聚合周围 reference points 的信息。

公式为：

zl+1=∑k=1KAkl⋅Xfeat (p+Δokl)\mathbf{z}^{l+1}=\sum_{k=1}^K A_k^l \cdot X_{\text {feat }}\left(p+\Delta o_k^l\right) zl+1=k=1∑K​Akl​⋅Xfeat ​(p+Δokl​)

  • ppp ：learnable 2-D reference points
  • Δo\Delta oΔo ：learnable 2-D sampling offsets
  • AAA ：learnable attention weights
  • z\mathbf{z}z ：query token
  • KKK ：手动选择的 reference points 的数量
  • lll ：attention layer 层数

所有的 reference points, sampling offsets, attention weights 都是直接由 learnable query z\mathbf{z}z 加一个线性层得到的，所以全部都是可以学习的向量。

其实 attention weights 就不是 qk 交互的结果了，而是训练到的线性层决定谁的权重比较高；
不过前面有一个 self attention，所以也能够交互，这里线性层更像是一个 filter 。

同时作者提出创新：

  1. v-pooled Q： 使用 adaptive average pooling 对 feature map 进行初始化，得到 qqq ，而不是随机初始化。这一步操作就已经决定了 query token 的数量。
  2. M-RP：手动初始化 reference points，并且在整个特征空间中均匀分布。

Training

Model：
CLIP ViT-L/14 + 7B and 13B Vicuna-v1.5
resolutions of 224 and 336 for 7B- and 13B-LLM
use the features from the second-last layer of CLIP instead of the last layer

Two-stage pipeline:

  1. 冻结 ViT 和 LLM，重点关注所提出的 Locality-enhanced Projector 的训练。
  2. 对 Locality-enhanced Projector 和 LLM 进行训练，以增强更深层次的视觉理解和生成能力。
