TextMonkey

TextMonkey: An OCR-Free Large Multi-modal Model for Understanding Document

场景：a large multi-modal model，text-centric tasks

  • Scene Text-Centric tasks
  • Document-Oriented tasks
  • Key Information Extraction

Intro

Related Work

  1. Fig. 1 (a)：之前的工作使用 vit 进行编码，但是他们的分辨率只有 224 / 336 ，原始的图片会被降采样，无法分辨 small texts，只能看到 large texts 。
  2. Fig. 1 (b)：UReaer 和 Monkey 使用采取了一种裁剪策略来扩展输入分辨率。然而，这种裁剪策略可能会无意中将相关单词分割开，导致语义不连贯。 -> missing cross-window
    context
  3. Fig. 1 (c)：DocPedia 直接在频域而不是像素空间处理视觉输入。由于频域的特性，它可以在不丢失信息的情况下快速扩展分辨率，但是无法利用 VIT 了。

This Work

  1. 提出 Split Module：为了处理高分图片，作者将原始的大图切分为 non-overlapping 的多个小图。
  2. 提出 Shifted Window Attention：小图内部的 patch 还是使用 vit 自己的 self-attention；同时提出 zero-initialization Shifted Window Attention 来处理 cross-window patches。
  3. 提出 Image Re-Sampler
  4. 提出 Token Re-Sampler：滑动窗口机制导致最终的输出序列长度变长，所以提出一个 re-sampler 剔除冗余 patch，仅仅保留有价值的 patch 。

METHODOLOGY

把原始的大图切分为多个非重叠的 448×448448\times 448448×448 小图，每小图内继续切为 14×1414\times 1414×14 的 patch 。

Attention

  1. Within Window: 每一个 window 内部的 patch 直接用 VIT attention 编码。
  2. Cross Window: 参考 Swin transformer ，对于不同 window 的 patch，设置一个 attention window，来滑动覆盖多个子图，然后进行覆盖范围内的 attention，但是这次每个 patch token 不 attend 原始小图内的 token 。为了更好地初始化训练，这个 Shifted Window Attention 被初始化为 0 ，这里使用了 Lora 的初始化方法。

Image Resampler

参考 Qwen-VL ：
就是一个 Q-Former 结构，随机初始化多个 query token（256 个），去 cross attend VIT 的输出。
为了保证位置信息，还给 Key value 引入 2D 绝对位置编码。

Token Resampler

参考 Token merging: Your vit but faster
使用 image tokens 的 cosine sim 表示相似程度，进而可以表示 importance：和其他 token 越相似，就越不重要。

为了保留全局信息，使用选择出来的 important token 去 attend 原始的 256 token。

Position-Related Task

希望 LLM 不仅仅能提供答案，还能指出这个答案位于图片的哪一个位置。

  1. 作者人工标注了「位置线索」，加入到原始的答案中。
  2. 增加了额外的训练任务，以提高模型对文本位置的感知，例如文本定位（text spotting）和阅读文本（reading text）。
    为了标准化不同比例的图像，使用范围 (0,1000)(0, 1000)(0,1000) 来表示位置信息。因此，在分辨率为 (Hr×Wr)(H_r \times W_r)(Hr​×Wr​) 的图像中， xxx 将被标准化为 x/Hr×1000x/H_r \times 1000x/Hr​×1000 ， yyy 同理。

基座模型

一共9.7B

视觉编码器：1.9B，来自Qwen-VL的视觉编码器（Vit-BigG，听说基于OpenCLIP）。

图片重采样器：90M，256个可学习的 queries，不同分辨率的图片设置不同的采样率。

大语言模型：Qwen-7B

移位窗口注意力模块：45M
