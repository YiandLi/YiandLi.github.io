【机器学习】降维——PCA（非常详细）

主成分分析

主成分分析 Principal components analysis（PCA）

  • 定义： 设法将原来变量重新组合成一组新的互相无关的几个综合变量，同时根据实际需要从中可以取出几个较少的综合变量尽可能多地反映原来变量的信息的统计方法叫做主成分分析或称主分量分析，也是数学上用来降维的一种方法。
  • 问题描述：将n维特征映射到k维上 k<nk \lt nk<n ，这k维是全新的正交特征。这k维特征称为主成分，是重新构造出来的k维特征，而不是简单地从原始特征中删除一些特征。
  • 超平面性质：
    • 最近重构性：样本点到这个超平面的距离足够近
    • 最大可分性：样本点在这个超平面上的投影能尽可能的分开
      将 nnn 维降到 n^\hat nn^ 维，我们希望新维度数据尽可能代表原始数据集，即损失尽可能小。

方差

那么如何选择一个基向量 PPP 才是最优的呢？

我们需要降维后损失最小，就是想让数据之间的差距仍然明显，理解为投影后的数据尽可能的分开。
就是要求方差越大越好。所以选择令方差最大的基向量。

协方差 Covariance

协方差表示的是两个变量的总体的误差，这与只表示一个变量误差的方差不同。如果两个变量的变化趋势一致，也就是说如果其中一个大于自身的期望值，另外一个也大于自身的期望值，那么两个变量之间的协方差就是正值。如果两个变量的变化趋势相反，即其中一个大于自身的期望值，另外一个却小于自身的期望值，那么两个变量之间的协方差就是负值。

当完成第一个方向（基）选择后，如果不加约束条件，第二个方向（基）会和第一个完全一致，这显然是没有用的，因此要有其它的约束条件。
我们希望求得的两个新特征/字段尽可能表示更多的信息，使其不存在相关性，也就是正交基之间 协方差为 0 → uncorrelated 。
即对于一系列正交基：

{w1,w2,…,wd} \left\{\boldsymbol{w}_1, \boldsymbol{w}_2, \ldots, \boldsymbol{w}_d\right\} {w1​,w2​,…,wd​}

最理想的情况下就是：两两垂直：

∥wi∥2=1,wiTwj=0\left\|\boldsymbol{w}_i\right\|_2=1, \boldsymbol{w}_i^{\mathrm{T}} \boldsymbol{w}_j=0 ∥wi​∥2​=1,wiT​wj​=0


根据上面描述，通过点乘来进行投影，即 x→wixx \rightarrow w_i xx→wi​x

问题变成了如何选择这一系列正交基。

选取的目标是使降维后的点（某一维度上） wixw_i xwi​x 尽量的分散，也就是方差尽量的大，方差公式为 E[(wiX−Xˉ)2]E[(w_i X-\bar X)^2]E[(wi​X−Xˉ)2] ，减去均值后的平方的期望。我们做过了中心化操作，所以均值为 0 ，则每一个正交基的方差变为了 E[(wiX)2]E[(w_i X)^2]E[(wi​X)2] ，总体方差变成 $E[(W X)^2] $ 。

其对角元素表示了降维后的同一纬度上元素的方差：越大越大
其他元素表示不同维度上元素的方差：我们希望为 0，表示了不同维度之间相互不相关。


数学上使用协方差来表示两个不同的特征 A,BA,BA,B 的相关性，

协方差矩阵对角化

我们的目标即使协方差矩阵的对角元素（方差）变大，其余元素（协方差）等于0，就是使协方差矩阵对角化。
并且因为各个主成分的重要性越来越小，即各个新特征方差越来越小，所以在对角线上的元素是按大小从上到下排列的。

我们进一步看下原矩阵与基变换后矩阵协方差矩阵的关系：
设原始数据矩阵 XXX 对应的协方差矩阵为 CCC ，而 PPP 是一组基向量按行组成的矩阵。
新数据特征矩阵为 Y=PXY=PXY=PX ，设 YYY 的协方差矩阵为 DDD 。
现在推导下 DDD 和 CCC 的关系：

D=1mYY⊤=1m(PX)(PX)⊤=1mPXX⊤P⊤=P(1mXX⊤)P⊤=PCP⊤=P(1m∑i=1mai21m∑i=1maibi 1m∑i=1maibi1m∑i=1mbi2)P⊤\begin{aligned} D&=\frac{1}{m} Y Y^{\top} \\\\ &=\frac{1}{m}(P X)(P X)^{\top} \\\\ &=\frac{1}{m} P X X^{\top} P^{\top} \\\\ &=P\left(\frac{1}{m} X X^{\top}\right) P^{\top} \\\\ &=P C P^{\top} \\\\ &=P\left(\begin{array}{cc} \frac{1}{m} \sum_{i=1}^{m} a_{i}^{2} & \frac{1}{m} \sum_{i=1}^{m} a_{i} b_{i} \\\\ \ \frac{1}{m} \sum_{i=1}^{m} a_{i} b_{i} & \frac{1}{m} \sum_{i=1}^{m} b_{i}^{2} \end{array}\right) P^{\top} \end{aligned} D​=m1​YY⊤=m1​(PX)(PX)⊤=m1​PXX⊤P⊤=P(m1​XX⊤)P⊤=PCP⊤=P⎝⎛​m1​∑i=1m​ai2​ m1​∑i=1m​ai​bi​​m1​∑i=1m​ai​bi​m1​∑i=1m​bi2​​⎠⎞​P⊤​

那么 P 的前 K 行就是要寻找的基，用 P 的前 K 行组成的矩阵乘以 X 就使得 X 从 N 维降到了 K 维并满足上述优化条件。

找到了 PPP ，用 PPP 的前R行乘 XXX ，就可以得到 YYY ，完成降维。

将特征向量按对应特征值大小从上到下按行排列成矩阵，取前k行就组成基向量矩阵 PPP 。

一个 n 行 n 列的实对称矩阵 CCC 一定可以找到 n 个单位正交特征向量，设这 n 个特征向量为 e1,e2,...ene_1,e_2,... e_ne1​,e2​,...en​ ，将其按列组成矩阵 EEE ，那么有如下性质 :

补充：计算特征值和特征向量

Sv=λvSv = \lambda v Sv=λv

对矩阵S，称 λ\lambdaλ 为其特征值（eigenvalues）， vvv 为其特征向量（eigenvectors）。
如果S为 m∗mm*mm∗m 的方阵，则S最多有m个不同的特征值，满足 ∣S−λI∣=0|S-\lambda I|=0∣S−λI∣=0 。
对称矩阵不同的特征矩阵是相互正交的。
半正定矩阵的特征向量是非负的。

总结

  • 是一种无监督学习算法
  • 优点
    • 仅仅需要以方差衡量信息量，不受数据集以外的因素影响。
    • 各主成分之间正交，可消除原始数据成分间的相互影响的因素。
    • 计算方法简单，主要运算是特征值分解，易于实现。
  • 缺点
    • 不会提升模型的推理能力
    • 主成分各个特征维度的含义解释性不强
    • 方差小的非主成分也可能含有对样本差异的重要信息，在降维操作中被丢弃可能对后续数据处理有影响。

非负矩阵分解

非负矩阵分解 Non-Negative Matrix Factorization（NMF）

将非负的大矩阵 XXX 分解成两个非负的小矩阵相乘 WHWHWH ：

Xn∗m≈Wm∗r⋅Hr∗mX_{n*m} \approx W_{m*r} \cdot H_{r*m} Xn∗m​≈Wm∗r​⋅Hr∗m​

分解前后可理解为：原始矩阵X的列向量是对左矩阵W中所有列向量的加权和，而权重系数就是右矩阵H对应列向量的元素。
所以，W 被称为基矩阵，H 被称为系数矩阵。
一般情况下 r<nr\lt nr<n ，所以 (m+n)∗r<mn(m+n)*r \lt mn(m+n)∗r<mn ，这样就可以完成降维。

求解

非负矩阵分解是个NP问题，可以划为优化问题用迭代方法交替求解W和H。

应用

在重构图片上，不如PCA好，因为PCA直接找到最优的方向。
但是NMF能够找到非负特征。
