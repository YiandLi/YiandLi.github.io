LinearRegression

sklearn.linear_model.LinearRegression

初始化方法

  • fit_intercept : bool, default=True：是否计算截距
  • copy_X : bool, default=True：默认为True, 否则X会被改写。
  • n_jobs : int, default=None：数据量太大时，考虑多核？
  • positive : bool, default=False：设置为True时，强制系数为正值。仅密集阵列支持此选项。

属性

  • coef_ : array of shape (n_features, ) or (n_targets, n_features)： feature的系数（coefficient
  • rank_ : int： 矩阵的秩
  • singular_ : array of shape (min(X, y),)：矩阵X的奇异值
  • intercept_ : float or array of shape (n_targets,)：截距（intercept，线性模型中的独立项
  • n_features_in_ : int
  • feature_names_in_ : ndarray of shape (n_features_in_,)：返回特征的名称（传入的X为dataframe时

方法

  • score(X, y, sample_weight=None)：评估模型，返回 R−squared=sum[(ytrue−ypred)2]sum[(ytrue−mean(ytrue)]2)R-squared = \frac {sum[(y_{true} - y_{pred})^2]} {sum[(y_{true} - mean(y_{true})]^2)}R−squared=sum[(ytrue​−mean(ytrue​)]2)sum[(ytrue​−ypred​)2]​ ，最好的值为1。

注意

如果有非结构化数据，则使用数值替代label。

  • 对于单纯的二分类的特征，将其中一个值看作一个baseline，使用一个虚拟数据(dummy data)来表示二元值，是或者不是。
    用dataframe.parameter.map({'feature1':1, 'feature2':0})将属性转换为[0,1]。
    可以将其中的feature2理解为baseline，即系数表示有了feature1这个特征，输出值能增加多少。如果将[0,1]取反，则系数会变为负数。
  • 但是对于多分类的特征，不能简单地使用[1,2, …n]，因为不存在倍数关系。
    解决方法是对该特征中的每一个值，使用一个[0,1]。
    即使用pd.get_dummies()方法，可以直接使用这种one-hot表示，不公更通常的做法是使用n-1种表示n种，因为全为0也是一种表示，可看作baseline。
