介绍

对比学习 Contrastive Learning

自监督学习的一种，通过样本增强的方式自动构造相似实例和不相似实例，习得一个表示学习模型。
通过这个模型，在投影空间中拉近相似实例的距离，增大不相似实例在投影空间中的距离。

正样本对

  • 多种不同的构造方式：使用样本增强技术，对输入进行处理；对相同的输入使用不同的编码器；使用next sentence当作正样本 …
  • 本文采用了最简单的样本增强技术 （ minimal data augmentation
    bert中有dropout层。即在前向传播的时候，让某个神经元的激活值以一定的概率p停止工作，这样可以使模型泛化性更强，因为它不会太依赖某些局部的特征。
    使训练过程变得嘈杂，迫使图层中的节点对输入承担或多或少的责任。

    所以，同一句话输入到相同的带有dropout的模型中，输出的向量表征是不同的。

对比学习评判标准

论文：《Understanding Contrastive Representation Learning through Alignment and Uniformity on the Hypersphere》

  • Alignment：近似例子/正例在超球面上的距离应该比较近。

ℓalign ≜E(x,x+)∼ppos ∥f(x)−f(x+)∥2\ell_{\text {align }} \triangleq \underset{\left(x, x^{+}\right) \sim p_{\text {pos }}}{\mathbb{E}}\left\|f(x)-f\left(x^{+}\right)\right\|^{2} ℓalign ​≜(x,x+)∼ppos ​E​∥∥​f(x)−f(x+)∥∥​2

  • Uniformity：系统保留了尽可能多的信息，即 ziz_izi​ 被均匀地映射到超球面上。越均匀说明信息被保留得越充分，不同样本间两两有差。

ℓuniform ≜log⁡E(x,y)∼i.i.d.pdatae−2∥f(x)−f(y)∥2\ell_{\text {uniform }} \triangleq \log \underset{\left(x, y\right) \sim^{i.i.d.} p_{\text {data}}}{\mathbb{E}} e^{-2\|f(x)-f(y)\|^{2}} ℓuniform ​≜log(x,y)∼i.i.d.pdata​E​e−2∥f(x)−f(y)∥2

实验

  1. 不同的augmentation
    作者从维基百科中随机抽取十万个句子来微调BERT模型，并在STS-B开发集上测试。

    None是作者提出的随机dropout mask方法, 追加显式的数据增强方法都会大幅降低性能。
    作者也对比了这样的自监督方式和NSP目标的训练结果，如下表所示，将NSP作为正样本的构建方式明显很不科学，另外，做语义相似度计算一定要共享编码器。
  2. dropout rate的讨论

    也可见，同样是没有随机mask，0 mask和fixed mask差别却很大。
  3. Alignment and Uniformity
    作者在训练中，每10步记录一个checkpoint。
    同时将l“delete one word”作为对比。

    可见，所有模型的uniformity都提升了，但是两种特殊参数的alignment却显著降低了。
    庆幸的是Unsupervised SimCSE的alignment保持稳定。
    同样说明在一个预训练好的模型基础上进行对比学习很重要，因为可以保证alignment。
    最后，“delete one word”的alignment提升很大，但是uniformity却不多。

Supervised SimCSE


在NLI(SNLI+MNLI)数据集中，entailment作为正样本，conradiction作为负样本，加上原样本premise一起组合为 (xi,xi+,xi−)\left(x_{i}, x_{i}^{+}, x_{i}^{-}\right)(xi​,xi+​,xi−​) 。

In NLI datasets, given one premise, annotators are required to manually write one sentence that is absolutely true (entailment), one that might be true (neutral), and one that is definitely false (contradiction).

训练损失函数为：

−log⁡esim⁡(hi,hi+)/τ∑j=1N(esim⁡(hi,hj+)/τ+esim⁡(hi,hj−)/τ)-\log \frac{e^{\operatorname{sim}\left(\mathbf{h}_{i}, \mathbf{h}_{i}^{+}\right) / \tau}}{\sum_{j=1}^{N}\left(e^{\operatorname{sim}\left(\mathbf{h}_{i}, \mathbf{h}_{j}^{+}\right) / \tau}+e^{\operatorname{sim}\left(\mathbf{h}_{i}, \mathbf{h}_{j}^{-}\right) / \tau}\right)} −log∑j=1N​(esim(hi​,hj+​)/τ+esim(hi​,hj−​)/τ)esim(hi​,hi+​)/τ​

作者分别尝试在不同的语义匹配数据集上(QQP、Flickr、ParaNMT)训练模型，并在STS-B上测试模型，结果如下表所示，其中sample列表示采样相等样本量训练的结果，作者发现NLI训练出来的模型性能是最好的，这是因为NLI数据集的质量本身很高，正样本词汇重合度非常小(39%)且负样本足够困难，而QQP和ParaNMT数据集的正样本词汇重合度分别达到了60%和55%。

模型拟合NLI数据集的难度更大，训练出来的模型能更多的学习到语义上的相似度，而不是依靠简单的词汇匹配信息来估计相似度


同时，作者发现，增加负样本对可以将结果从84.9提高到86.2。

Connection to Anisotropy

语言模型生成的语义向量分布存在各向异性的问题，这极大地限制了语义向量的表达能力，缓解这个问题的一种简单方法是加入后处理步骤，比如BERT-flow和BERT-whitening将向量映射到各向同性的分布上，而本文作者证明了对比学习的训练目标可以隐式地压低分布的奇异值，提高uniformity。
可以看成模型坍塌问题：表示空间应该均匀分布。所以对比学习中的推开负样本对可以解决这个问题。
Wang and Isola (2020)证明当负样本数量趋于无穷大时，对比学习的训练目标可以渐近表示为：
