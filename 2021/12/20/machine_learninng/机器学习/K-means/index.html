Hard K-means

  • 要选择代表点，因为如果某些样本点很接近，那么就可以用一个中心点来代表它们。
    这就是 聚类（cluster） 的思想，从所有N个样本点中选择少数几个代表作为中心点。

  • 聚类（clustering）问题是一种典型的非监督式学习（unsupervised learning）。
    它的优化问题有两个变量需要确定：

    • 一个是分类的分群值 SmS_mSm​ ，每一类可表示为 S1,S2,⋯,SmS_1,S_2,⋯,S_mS1​,S2​,⋯,Sm​ ；
    • 另外一个是每一类对应的中心点 μ1,μ2,⋯,μmμ_1,μ_2,⋯,μ_mμ1​,μ2​,⋯,μm​ 。

  • 找到最优聚类是一个NP-hard问题。

损失函数

Ein(S1,⋯ ,SM;μ1,⋯ ,μM)=1N∑n=1N∑m=1M⟦xn∈Sm⟧∥xn−μm∥2E_{i n}\left(S_{1}, \cdots, S_{M} ; \mu_{1}, \cdots, \mu_{M}\right)=\frac{1}{N} \sum_{n=1}^{N} \sum_{m=1}^{M} \llbracket \mathbf{x}_{n} \in S_{m} \rrbracket\left\|\mathbf{x}_{n}-\mu_{m}\right\|^{2} Ein​(S1​,⋯,SM​;μ1​,⋯,μM​)=N1​n=1∑N​m=1∑M​[[xn​∈Sm​]]∥xn​−μm​∥2

能够发现这是一个组合最佳化的问题，既要优化分群值 SmS_mSm​，又要求解每一类的中心点 umu_mum​。

优化方式

对 SSS 和 μμμ 分别进行最优化求解：
首先，如果中心点 μ1,μ2,⋯,μmμ_1,μ_2,⋯,μ_mμ1​,μ2​,⋯,μm​ 是固定的，因为一个点只能属于一个群，所以点 xnx_nxn​ 一定属于距离自己最近的样本点的群。
然后，如果 S1,S2,⋯,SMS_1,S_2,⋯,S_MS1​,S2​,⋯,SM​ 是固定的，目标就是只要找出每个类的中心点 μμμ 。使用梯度下降算法，得到中心点 μmμ_mμm​ 就等于所有属于类群 SmS_mSm​ 的平均位置处：

μm=1Nm∑n∈Smxnμ_m = \frac{1}{N_m} \sum_{n \in S_m}x_n μm​=Nm​1​n∈Sm​∑​xn​

NmN_mNm​ 为该聚类内点的个数
SmS_mSm​ 为某一个聚类
xnx_nxn​ 为聚类内的点

经过以上的推导，我们得到了一个非常有名的一种unsupervised learning算法，叫做k-Means Algorithm。这里的k就是代表中心点个数和类群的个数，即上面的M。

流程

  1. 首先，随机选择 kkk 个中心点 μ1,μ2,⋯,μkμ_1,μ_2,⋯,μ_kμ1​,μ2​,⋯,μk​；
  2. 然后，再由确定的中心点得到不同的类群 S1,S2,⋯,SkS_1,S_2,⋯,S_kS1​,S2​,⋯,Sk​ （Assignment step）；
  3. 接着，再由确定的类群计算出新的不同的k个中心点 （Refitting step）；
  4. 继续循环迭代计算，交互地对μ和S值进行最优化计算，不断更新μ和S值，直到程序收敛，实现 EinE_{in}Ein​ 最小化。

这将同时考虑多个参数来最小化损失转化为分别考虑参数最小化损失的方式叫做 交替最小化 Alternating minimization 。

讨论

  • K-means 算法的弊端：
    • 需要事先指定聚类数量
    • 不适合非凸形状的聚类
    • 对噪声和偏离对象敏感
  • K-means 提升方法：
    • Repeated K-means，多试几次择优
    • 更好初始化，使用一些启发式方法来分配向量起始分布

GMM

混合高斯模型

对于单高斯模型，由于可以明确训练样本是否属于该高斯模型：
  对于任意输入x，使用 p(x∣z=k)=N(x∣μk,Σ)p(x|z=k)=\mathcal N(x|\mu_k,\Sigma)p(x∣z=k)=N(x∣μk​,Σ) 表示该样本为正样本/负样本的概率。然后根据提前设置的阈值来进行分类。

混合高斯模型（Gaussian Mixture Model）假设每个聚类分布都是符合高斯分布的，当前数据分布为多个高斯分布的叠加。
通过使⽤⾜够多的⾼斯分布，并且调节它们的均值和⽅差以及线性组合的系数，⼏乎所有的连续概率密度都能够以任意的精度近似。

混合高斯分布

假设模型由 K 个 高斯分布组成（即数据包含K个类），每个高斯分布称为一个“Component”，这些 Component 线性加成在一起就组成了 GMM 的概率密度函数。

p(x)=∑k=1KπkN(x∣μk,Σk)p(x)=\sum_{k=1}^{K} \pi_{k} \mathcal N\left(x \mid \mu_{k}, \Sigma_{k}\right) p(x)=k=1∑K​πk​N(x∣μk​,Σk​)

πk∈[0,1]\pi_k \in [0,1]πk​∈[0,1] 被称为混合系数（mixing coefficients）。
πk=pk\pi_k=p_kπk​=pk​ 理解为选择第k个分布的先验概率，∑k=1Kπk=∑k=1Kpk=1\sum_{k=1}^K \pi_k=\sum_{k=1}^K p_k= 1∑k=1K​πk​=∑k=1K​pk​=1

Soft K-means

  • 硬聚类指的是个体只能属于某一种聚类。
    软聚类允许个体同时属于多个聚类

  • soft assignments：每个聚类的大小不一定相同。
    在refitting时，一个聚类可以使用更多数据信息。

  • 迭代过程：

    • 点 xnx^nxn 属于以 mkm_kmk​ 中心的聚类的概率为：

    rn=softmax(−β{∣∣mk−xn∣∣}k=1K)r^n = softmax(-\beta\{||m_k-x^n||\}_{k=1}^K ) rn=softmax(−β{∣∣mk​−xn∣∣}k=1K​)
