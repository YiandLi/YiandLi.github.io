径向基神经网络

Posted on 2021-12-11 In Machine Learning
Symbols count in article: 2.5k Reading time ≈ 13 mins.

https://blog.csdn.net/red_stone1/article/details/101302684

机器学习的任务大多都是输入输出的映射问题，网络就是需要学习这种映射函数。
所以神经网络的设计可以看做一种曲线拟合问题 curve-fitting或者是函数近似问题 function approximation。
曲线拟合问题就像是寻找一个插值平面（interpolation surface） 在高维空间中拟合样本，即插值问题interpolation problem。

径向基函数 Radial Basis Function

定义：函数的值依赖于空间中点 xxx 到中心点 ccc 之间距离。是一个单调函数，可记为 ϕ(x,c)=ϕ(∣∣x–c∣∣)\phi(x,c)=\phi(||x–c||)ϕ(x,c)=ϕ(∣∣x–c∣∣) 。
径向基函数用于度量某点到指定中心点c的距离，用距离来表示相似性。
取值一般和距离呈反比：当 xxx 远离 ccc 时函数取值很小；当 xxx 靠近 ccc 时，取值变大，来保证只有中心区域被少部分激活。

常见的径向基函数

  1. 多二次函数 multiquadratics

ϕ(x)=(x2+c2)1/2     for some c>0\phi(x) = (x^2 + c^2)^{1/2}\ \ \ \ \ for\ some\ c>0 ϕ(x)=(x2+c2)1/2     for some c>0

  1. 逆二次函数 inverse multiquadratics

ϕ(x)=1(x2+c2)1/2     for some c>0\phi(x) = \frac{1}{(x^2 + c^2)^{1/2}}\ \ \ \ \ for\ some\ c>0 ϕ(x)=(x2+c2)1/21​     for some c>0

  1. 高斯函数 Gaussian

ϕ(x)=exp(−x2/2σ2)     for some σ>0\phi(x) = exp(-x^2 / 2\sigma^2)\ \ \ \ \ for\ some\ \sigma>0 ϕ(x)=exp(−x2/2σ2)     for some σ>0

描述


径向基函数主要用于解决多变量拟合/插值问题。
可通过多个径向基函数的和来逼近某个给定的数，这一逼近过程可被看作是简单的神经网络。用来表示插值函数F：

F(x)=∑i=1Mwi ϕ(∣∣x−xi∣∣)F(x) = \sum_{i=1}^M w_i\ \phi (|| x-x_i||) F(x)=i=1∑M​wi​ ϕ(∣∣x−xi​∣∣)

MMM 表示中心点的个数，是人为决定的。

Full RBFNetwork

将每一个样本点都看作中心点时， M=NM=NM=N 。
Full RBFNetwork（或称 Regularization Networds），将插值函数 interpolation function F 定义成一个多元高斯函数的线性叠加 a linear superposition of multivariate Gaussian functions，高斯函数的数量和输入样本数量相同：

F(x)=∑i=1Nwi exp(−∣∣x−xi∣∣22σ2)F(x) = \sum^N_{i=1} w_i\ exp(\frac{-||x-x_i||^2} {2\sigma^2}) F(x)=i=1∑N​wi​ exp(2σ2−∣∣x−xi​∣∣2​)

  • 特点：
    是一个通用的逼近器 universal approximator，能够很好的近似到任意的多元连续函数。
  • 缺点：
    • 这样的话每一个新来的点都会受到样本点（N中的点）的投票表决，离新来的的点越近的所持有的票数 ϕ\phiϕ 越大。并且由于高斯函数衰减的很快，这就导致了距离新来的点最近的那个点会掌握大部分的票数 ϕ\phiϕ ，就会近似成KNN算法。
    • 需要大量的计算 computationally inefficient：因为一个高斯函数对应一个样本，所以通常需要大量高斯函数。
    • 求权重 www 需要很大的矩阵 ϕTϕ\phi^T \phiϕTϕ 。

径向基网络 RBF Network

不把所有N个样本点都拿来作中心点，而是只选择其中的M个样本点作为中心点。

  • 三层结构：
    • 输入层
    • 隐藏层
      应用非线性转移函数 a non-linear transformation function
      扩充维度/映射到高维度隐空间 high-dimensional hidden space中
    • 输出层：线性相加 linear transformation
  • 这里和神经网络不同的是，神经网络前后两步计算都是线性组合；而RBF网络隐藏层是计算距离的RBF，然后输出层是线性组合。
  • 基本原理：
    • 当样本被映射到高维空间后，更容易被超平面分割/线性可分。
    • 非线性函数可以将非线性可分的插值问题转换为一个映射的线性可分问题。

求中心点 : K-means 聚类

  • 要选择代表点，因为如果某些样本点很接近，那么就可以用一个中心点来代表它们。
    这就是 聚类（cluster） 的思想，从所有N个样本点中选择少数几个代表作为中心点。
  • 聚类（clustering）问题是一种典型的非监督式学习（unsupervised learning），找到最优聚类是一个NP-hard问题。
  • 它的优化问题有两个变量需要确定：
    • 一个是分类的分群值 SmS_mSm​ ，每一类可表示为 S1,S2,⋯,SmS_1,S_2,⋯,S_mS1​,S2​,⋯,Sm​ ；
    • 另外一个是每一类对应的中心点 μ1,μ2,⋯,μmμ_1,μ_2,⋯,μ_mμ1​,μ2​,⋯,μm​ 。
  • 高质量聚类的要求：
    • high intra-class similarity：聚类内部样本高相似度
    • low inter-class similarity：不同聚类样本低相似度
    • 同时，好的聚类可以发现样本的隐藏特征（类似PCA？）。

MSE损失函数

Ein(S1,⋯ ,SM;μ1,⋯ ,μM)=1N∑n=1N∑m=1M⟦xn∈Sm⟧∥xn−μm∥2E_{i n}\left(S_{1}, \cdots, S_{M} ; \mu_{1}, \cdots, \mu_{M}\right)=\frac{1}{N} \sum_{n=1}^{N} \sum_{m=1}^{M} \llbracket \mathbf{x}_{n} \in S_{m} \rrbracket\left\|\mathbf{x}_{n}-\mu_{m}\right\|^{2} Ein​(S1​,⋯,SM​;μ1​,⋯,μM​)=N1​n=1∑N​m=1∑M​[[xn​∈Sm​]]∥xn​−μm​∥2

能够发现这是一个组合最佳化的问题，既要优化分群值 SmS_mSm​，又要求解每一类的中心点 umu_mum​。

MSE（Mean-Squared Loss）用了欧式距离的平方
Euclidean distance：向量对应元素的差的平方和的根号。
Manhattan distance：向量对应位置元素的差的和。

优化方式

对 SSS 和 μμμ 分别进行最优化求解：
首先，如果中心点 μ1,μ2,⋯,μmμ_1,μ_2,⋯,μ_mμ1​,μ2​,⋯,μm​ 是固定的，因为一个点只能属于一个群，所以点 xnx_nxn​ 一定属于距离自己最近的样本点的群。
然后，如果 S1,S2,⋯,SMS_1,S_2,⋯,S_MS1​,S2​,⋯,SM​ 是固定的，目标就是只要找出每个类的中心点 μμμ 。使用梯度下降算法，得到中心点 μmμ_mμm​ 就等于所有属于类群 SmS_mSm​ 的平均位置处：

μm=1Nm∑n∈Smxnμ_m = \frac{1}{N_m} \sum_{n \in S_m}x_n μm​=Nm​1​n∈Sm​∑​xn​

NmN_mNm​ 为该聚类内点的个数
SmS_mSm​ 为某一个聚类
xnx_nxn​ 为聚类内的点

经过以上的推导，我们得到了一个非常有名的一种unsupervised learning算法，叫做k-Means Algorithm。这里的k就是代表中心点个数和类群的个数，即上面的M。

流程

  1. 首先，随机选择 kkk 个中心点 μ1,μ2,⋯,μkμ_1,μ_2,⋯,μ_kμ1​,μ2​,⋯,μk​；
  2. 然后，再由确定的中心点得到不同的类群 S1,S2,⋯,SkS_1,S_2,⋯,S_kS1​,S2​,⋯,Sk​ （Assignment step）；
  3. 接着，再由确定的类群计算出新的不同的k个中心点 （Refitting step）；
  4. 继续循环迭代计算，交互地对μ和S值进行最优化计算，不断更新μ和S值，直到程序收敛，实现 EinE_inEi​n 最小化。这种方式叫做 交替最小化 Alternating minimization 。

  • K-means 算法的弊端：
    • 需要事先制定聚类数量
    • 不适合非凸形状的聚类
    • 对噪声和偏离对象敏感
  • K-means 提升方法：
    • Repeated K-means，多试几次择优
    • 更好初始化，使用一些启发式方法来分配向量起始分布

对比MLP

  • 相同点：
    • both layered feedforward networks to get nonlinear function mappings 非线形映射
    • both for universal approximators 都是通用逼近器
  • 不同点：
    • RBF只有一个隐藏层
    • MLP的隐藏层和输出层的节点都用相同的激活函数 activation functions
      RBF因为中心点和变量的不同，每个节点的激活函数都不同。
    • MLP的隐藏层和输出层都是非线性的 nonlinear
      RBF只有隐藏层是非线性的
    • RBF节点的激活函数计算的是欧氏距离
      MLP的激活函数计算内积
    • MLP得到的是全局近似 global approximations
      RBF得到的是局部最优 local approximations

应用

  • 上面所说的，多变量拟合/插值问题。感觉回归/分类都可以。
  • 降维： Φ\PhiΦ 其实可以看做降维的操作。这里用不到权重，中心点可以看做某一个维度的标准，而某一个样本到不同中心点的距离/相似性都可以看做一个新的维度。
