Boosting

  • 集成学习大致可分为两大类：Bagging和Boosting

    • Bagging一般使用强学习器，其个体学习器之间不存在强依赖关系，容易并行。
    • Boosting则使用弱学习器，其个体学习器之间存在强依赖关系，是一种序列化方法。
    • Bagging主要关注降低方差，而Boosting主要关注降低偏差。

  • Boosting是一族算法，其主要目标为将弱学习器“提升”为强学习器，大部分Boosting算法都是根据前一系列弱学习器的不足(shortcoming)对样本分布进行调整，再根据新的样本分布训练下一弱学习器，如此迭代M次，最后将一系列弱学习器组合成一个强学习器。

  • boosting算法中，损失函数改变，给不同的样本赋予不同的权重：

∑n=1N1NI[j(xn≠tn)]becomes      ∑n=1NwnI[j(xn≠tn)]where      wn>0  and  ∑n=1Nwn=1\begin{aligned} \sum_{n=1}^N \frac1N \mathbb I\left[j(x^n \ne t^n)\right]\\ becomes\;\;\; \sum_{n=1}^N w^n \mathbb I\left[j(x^n \ne t^n)\right]\\ where\;\;\; w^n>0\; and \;\sum_{n=1}^Nw^n=1 \end{aligned} n=1∑N​N1​I[j(xn​=tn)]becomesn=1∑N​wnI[j(xn​=tn)]wherewn>0andn=1∑N​wn=1​

  • Boosting有两大经典算法:
    • AdaBoost：shortcomings的表征就是权值高的样本点
      1997：A Decision-Theoretic Generalization of On-Line Learning and an Application to Boosting
    • Gradient Boosting：shortcomings的表征就是梯度
      2001：Greedy function approximation: A gradient boosting machine

AdaBoosting

AdaBoost是一个具有里程碑意义的算法，因为其是第一个具有适应性的算法，即能适应弱学习器各自的训练误差率，这也是其名称的由来（Ada为Adaptive的简写）。
主要解决二分类问题。

基分类器 base model

  • 要求：
    • 最小化带权损失
    • 模型数量可能很多，所以基分类器需要快速求解，一般使用弱学习器/分类器。
  • 弱分类器效果仅仅比猜测效果高一点点，比如二分类问题中，弱分类器的正确率只有0.51。
  • Decision Stump：一个只有两个叶节点的决策树，即决策树墩 Decision Stump。
    将区域分成上下或者左右两个空间。
  • 一个单独的弱分类器不能显著减小损失（准确率0.51），但是计算出来的损失值却小于1/2，因为在计算损失时样本权重不同。

  2. 每次根据带权重样本，训练一个基分类器。根据每个基分类器的表现赋予其权重，并放入融合模型集合中。
  3. 然后每次训练后，根据训练结果调整 WWW ：提高分错样本的权重，这样先前错误的样本在后续就能得到更多关注。

Gradient Boosting

公式推导

在Gradient Boosting中将先前累加模型的损失函数的负梯度作为上一轮基学习器犯错的衡量指标，在下一轮学习中通过 拟合负梯度（沿着负梯度的方向移动） 来纠正上一轮犯的错。

常见的单模优化时的梯度下降法，是将参数作为对象，求损失对参数的导数，然后调整参数使得损失变小。
现在是直接将直接把学习器在每个样本上的预测值F作为对象，求损失函数对预测值F的导数，然后调整输出值让输出值不断逼近标签值。
再者，全都用弱学习器，没什么参数可言，都是超参数，改不了。

具体做法：训练一个弱学习器，使加上该弱学习器后的累积模型损失往负梯度的方向减少。

Fm(x)=Fm−1(x)+arg⁡min⁡h∈HLoss⁡(yi,Fm−1(xi)+h(xi))(1.a)F_{m}(x)=F_{m-1}(x)+\underset{h \in H}{\arg \min } \operatorname{Loss}\left(y_{i}, F_{m-1}\left(x_{i}\right)+h\left(x_{i}\right)\right) \tag{1.a} Fm​(x)=Fm−1​(x)+h∈Hargmin​Loss(yi​,Fm−1​(xi​)+h(xi​))(1.a)

定义

MSE(y^1,...,y^N)=1N∑i=1N(yi−y^i)2MSE(\hat y_1,...,\hat y_N) = \frac 1N\sum_{i=1}^N(y_i-\hat y_i)^2 MSE(y^​1​,...,y^​N​)=N1​i=1∑N​(yi​−y^​i​)2

∇MSE=[∂MSE∂y^1,...,∂MSE∂y^N]=−2[y1−y^1,...,yN−y^N]=−2[r1,...,rN]\begin{aligned} \nabla{MSE}&=\left[ \frac{\partial MSE}{\partial{\hat y_1}} ,...,\frac{\partial MSE}{\partial{\hat y_N}} \right]\\ &=-2\left[y_1- \hat y_1, ..., y_N- \hat y_N\right] \\ &=-2\left[ r_1,...,r_N\right] \end{aligned} ∇MSE​=[∂y^​1​∂MSE​,...,∂y^​N​∂MSE​]=−2[y1​−y^​1​,...,yN​−y^​N​]=−2[r1​,...,rN​]​

更新的值就是残差的一次项。同理，残差也等比例减小。

结束条件

  1. 限制梯度下降的迭代数量，即控制基模型的数量。
    这种方法不能保证结果的表现能力。
  2. 还可以根据损失来停止，损失达到某个阀值即可退出。

权重选择

可以使用定值，用交叉验证来选择。
也可以使用变量，为：

ρ=h(∣∣∇f(x)∣∣)\rho = h(||\nabla f(x)||) ρ=h(∣∣∇f(x)∣∣)

∣∣∇f(x)∣∣||\nabla f(x)||∣∣∇f(x)∣∣ 表示梯度大小，当梯度很小时，模型权重也很小，步长变小，避免震荡；当梯度很大时，模型权重也很大，更快拟合。

算法流程

  1. 根据最小化损失函数训练一个基模型
  2. 依次训练M个模型
  3. 得到损失函数对输出值的负梯度，这个就是下一个基模型需要去拟合的部分
  4. 求模型权重（就是求模型），在上面的推导中，我们直接默认下一个模型的输出值就是这个负梯度。但是在这里我们用最小二乘法去求模型，就是让模型和负梯度的差距最小化。可以直接对步骤4的表达式求导，令等于0，可求得新模型的参数a；
    a为模型参数，b为放缩因子。
  5. 模型解出后，组合新模型之后得到的模型应该也满足损失函数最小，最小化误差求分类器权重。
  6. 融合模型

以上 Gradient Boosting 的算法流程具有一般性，根据其中的损失函数和弱学习器的不同可以演变出多种不同的算法。
如果是指数损失，则算法演变成 AdaBoost？？？？；弱学习器如果使用决策树，则算法成为 GBDT（Gradient Boosting Decision Tree）…
