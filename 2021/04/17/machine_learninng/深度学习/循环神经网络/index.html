循环神经网络 Recurrent Neural Network / RNN
是一种具有记忆力的网络，允许信息长久地保存在神经网络中。
适用于数据具有前后依赖性。

介绍

  • 先前的神经网络，层与层之间相互连接，但是每层上的各个节点是无连接的。
  • 循环神经网络的隐藏层的值不仅仅取决于当前这次的输入，还取决于上一次隐藏层的值。

Simple RNN

  • 激活函数原因：
    分解来看，假设输入x=0，则 st=st−1W=st−2W2=...=s0Wts_t=s_{t-1}W=s_{t-2}W^2=...=s_0W^{t}st​=st−1​W=st−2​W2=...=s0​Wt
    如果矩阵W的特征值略大或者略小于1，都会引起梯度消失/爆炸的问题。
  • 参数：
    矩阵A的尺寸：shape(h) * [shape(h)+shape(x)]，即h和x的长度。
    x为word embedding的输出词向量维度，h为rnn输出向量的维度。

参数更新方式

  • BPTT (Back propagation through time)：对每一个输出值的loss求和得到一个总的损失L，求导时每一个时间步都要对相同的W进行求导。
    然而使用BPTT，当序列很长时，对于单个参数的更新可能会带来很大的开销；并且存在梯度消失的问题(vanishing gradient problem)，梯度被近距离梯度主导，导致模型难以学到远距离的依赖关系。。

  • TBPTT（Truncated Back propagation through time）
    同样正常做后向传播，但是每前向传播 k1k_1k1​ 步，后向传播 k2k_2k2​ 步。如果 k2k_2k2​ 比较小，那么其计算代价将会降低。
    TensorFlow的TBPTT(k1k_1k1​, k2k_2k2​) 效果并不如 TBPTT(1, k2k_2k2​)，这表示TBPTT(1,k2k_2k2​) 可能不能学得更加全面的序列的信息(同上文分析的naive方式的不足)，因此，可以考虑采用 TBPTT(k1,k2)TBPTT(k_1, k_2)TBPTT(k1​,k2​) (其中 k1<k2<nk_1 < k_2 < nk1​<k2​<n )这种方式。

梯度消失/爆炸

可以看到 ∂Sj∂Sj−1\dfrac{\partial S_{j}}{\partial S_{j-1}}∂Sj−1​∂Sj​​ 的连乘操作可能会导致梯度消失/爆炸：

  • tanhtanhtanh 的导数总是小于1 的， 如果 WsW_sWs​ 也是一个大于0小于1的值， 那么随着 ttt 的增大， 上述公式的值越来越趋近于0， 这就导致了梯度消失问题。
  • 那么如果 WsW_sWs​ 很大， 上述公式会越来越趋向于无穷， 这就产生了梯度爆炸。

CEC（constant error carrousel）

Sepp Hochreiter 和 Jürgen Schmidhuber 在他们提出 Long Short Term Memory 的文章中引入了一种CEC的思想，即让t时刻的记忆因子直接和输入线性相加，组成t+1时刻的输入。这样求导时，梯度不变。
但是这种naive的做法太直白了，实际上就是个线性模型，在学习效果上不够好，因此LSTM引入了三个门。

引入三个门单元

ct=ft⊙ct−1+it⊙c^tc_t = f_t\odot c_{t-1} + i_t\odot \hat{c}_t ct​=ft​⊙ct−1​+it​⊙c^t​

导数为 ftf_tft​ ，是sigmoid的结果。输出要么接近于0 ， 要么接近于1。
当门为0时，说明上一时刻的信息对当前时刻没有影响， 我们也就没有必要传递梯度回去来更新参数了；当输出为1时，正好可以原封不动地传回去。

长短期记忆网络 Long Short Term Memory / LSTM

  • 解决梯度消失问题
  • 对于实际问题来说，有时需要更加久远的历史信息，有时仅需要临近的历史信息。
  • 传统的神经网路的每一个隐藏单元都只是执行一个简单的激活函数的操作，但是LSTM中每个模块内有四层结构：3个sigmoid层和1个tanh层。这四层结构构成了三种门单元，用于信息选择性通过。
  • 这三层门单元分别是遗忘门，输入门和输出门。
  • 简单的循环神经网络只需要传递一个隐藏状态 hth_tht​，而长短期记忆网络还需要传递候选状态 ctc_tct​，和候选内部状态 c^t\hat{c}_tc^t​。

遗忘门

  • sigmoid函数 σ\sigmaσ ，值域[0,1]，可以用来表示有多少量可以通过。
    在这里，根据当前的输入和上一时刻的隐藏输出来决定内部状态有多少被遗忘。
  • 数学原理：把t-1时的隐藏层向量 ct−1c_{t-1}ct−1​ 乘上一个遗忘因子 ftf_tft​ 。遗忘因子是由短期记忆ht−1h_{t-1}ht−1​以及事件信息 xtx_txt​ 来计算。
  • 遗忘因子计算公式：ft=σ(Wf[ht−1,xt]+bf)f_t = \sigma(W_f [h_{t-1}, x_t] + b_f)ft​=σ(Wf​[ht−1​,xt​]+bf​)

输入门

  • 一个sigmoid函数和一个tanh函数。根据当前的输入和上一时刻的隐藏输出决定什么样的新信息被存放在细胞状态。针对遗忘门中丢弃的属性信息，在本单元模块找到相应的新的属性信息，添加进去，以补充丢弃的属性信息。
  • 两个函数的作用：
    1. sigmoid层，决定什么值需要更新
    2. tanh层，创建一个新的候选值向量，生成候选记忆
  • 数学原理：根据遗忘因子和上一个隐藏层传递的值得到遗忘之后的保留值；根据上次输出和本次输入得到遗忘的补充值。
  • 计算公式：

it=σ(Wi[ht−1,xt]+bi)c^t=tanh(Wc[ht−1,xt]+bc)ct=ft⊙ct−1+it⊙c^t\begin{aligned} i_t &= \sigma( W_i [h_{t-1}, x_t]+b_i )\\ \hat{c}_t &= tanh( W_c [h_{t-1}, x_t]+b_c)\\ c_t &= f_t\odot c_{t-1} + i_t\odot \hat{c}_t \end{aligned} it​c^t​ct​​=σ(Wi​[ht−1​,xt​]+bi​)=tanh(Wc​[ht−1​,xt​]+bc​)=ft​⊙ct−1​+it​⊙c^t​​

⊙\odot⊙ 表示哈达玛积，对应位置相乘。
iti_tit​ ：输入门层，得到记忆因子，决定什么值我们要被补充。
c^t\hat{c}_tc^t​ ：创建的一个新的候选值状态
ctc_tct​ : 把旧状态与遗忘 ftf_tft​ 相乘，丢弃掉我们确定需要丢弃的信息;把候选状态和记忆因子 iti_tit​ 相乘，得到补充的信息;相加得到新的隐藏层向量 ctc_tct​。

输出门

  • 一个sigmoid函数。根据当前候选状态，输入和上一时刻的隐藏输出来决定该时刻的隐藏输出。
    首先我们使用一个Sigmiod函数来去确定细胞状态的哪部分需要输出，然后把细胞状态通过tanh层处理，两者相乘得到最终我们想要输出的信息。
  • 数学原理：通过一个Sigmoid函数得到遗忘门 oto_tot​ ， oto_tot​ 与 tanh(ct)tanh(c_t)tanh(ct​) 相乘得到最终的输出 hth_tht​ 。
  • 计算公式：

ot=σ(Wo[ht−1,xt]+bo)ht=ot⊙tanh(ct)\begin{aligned} o_t &= \sigma( W_o [h_{t-1}, x_t] + b_o )\\ h_t &= o_t\odot tanh( c_t ) \end{aligned} ot​ht​​=σ(Wo​[ht−1​,xt​]+bo​)=ot​⊙tanh(ct​)​

总结

优势

LSTM有进有出且当前的cell informaton是通过input gate控制之后叠加的，而RNN是叠乘，因此LSTM可以防止梯度消失或者爆炸，会比RNN好一点。

门控制循环单元 Gated Recurrent Unit / GRU

与LSTM效果差不多，但是计算量更低。

  • 输入：当前时刻的输入xtx^txt和上一层隐状态ht−1h^{t-1}ht−1
  • 输出：当前时刻的输出yty^tyt和传递给下一层的隐状态hth^tht，两者值相同。
  • 过程：

    本时刻输入和上一层隐状态经过重置门得到隐藏因子，然后本时刻输入和本时刻隐藏因子经过更新门得到本时刻隐藏层状态，同样也是本时刻输出。
