rL5-基于价值函数逼近 -- DQN

Posted on 2021-04-18 In Machine Learning , Reinforcement Learning
Symbols count in article: 1.5k Reading time ≈ 7 mins.

前面讲过：RL 基本方法为先估计值函数（关键），然后利用值函数改善当前的策略。

几种得到值函数的方法：

  • 模型已知：动态规划
  • 模型未知：Mc，TD

注意这里值函数是一个表格，状态值函数的索引是「状态」，行为值函数的索引为「状态-行为对」。值函数的更新其实就是这个表的更新，所以又称「表格型强化学习」。

这项方法有一个基本前提：状态空间和动作空间需要是离散的，并且数量不要特别大。

那么如果是连续的空间，或者数量很多，就无法用一张表格来表示了。这个时候就需要利用 函数逼近 的方法来表示值函数 。

就是用一个逼近函数 v^(s)\hat v(s)v^(s) 去得到当前状体 s 的值，也就可以上神经网络了，当然也可以使用非参数逼近的方法。

值函数更新

参数化逼近就是写成 v^(s,θ)\hat v(s,\theta)v^(s,θ) ， θ\thetaθ 为网络参数，那么如何得到实验数据进行监督学习呢？


前文中的更新方式为：

  • 蒙特卡罗方法, 值函数更新公式为

Q(s,a)←Q(s,a)+α(Gt−Q(s,a))Q(s, a) \leftarrow Q(s, a)+\alpha\left(G_t-Q(s, a)\right) Q(s,a)←Q(s,a)+α(Gt​−Q(s,a))

  • TD\mathrm{TD}TD 方法值函数更新公式为

Q(s,a)←Q(s,a)+α[r+γQ(s′,a′)−Q(s,a)]Q(s, a) \leftarrow Q(s, a)+\alpha\left[r+\gamma Q\left(s^{\prime}, a^{\prime}\right)-Q(s, a)\right] Q(s,a)←Q(s,a)+α[r+γQ(s′,a′)−Q(s,a)]

  • TD(λ)\mathrm{TD}(\lambda)TD(λ) 方法值函数更新公式为

Q(s,a)←Q(s,a)+α[Gtλ−Q(s,a)]Q(s, a) \leftarrow Q(s, a)+\alpha\left[G_t^\lambda-Q(s, a)\right] Q(s,a)←Q(s,a)+α[Gtλ​−Q(s,a)]

可以看到是利用当前的数据不断向目标函数靠近的，即 $G_t\ ;\ r+\gamma Q\left(s^{\prime}, a{\prime}\right) ; G_t\lambda $ 。
所以在监督学习的过程中，也是向一个目标值靠近，其数据和标签对为 (St,Ut)(S_t, U_t)(St​,Ut​) ，训练的目标函数：

arg⁡max⁡θ(q(s,a)−q^(s,a,θ))2\arg \max_{\theta} (q(s,a) - \hat q(s, a, \theta) )^2 argθmax​(q(s,a)−q^​(s,a,θ))2

基于 MC 方法的函数逼近

给定需要评估的策略，作为行动策略，产生实验，然后收集到 状态-价值对 列表 ：

<s1,G1>,<s2,G2>,....,<sr,Gr><s_1, G_1>,<s_2, G_2>, ...., <s_r, G_r> <s1​,G1​>,<s2​,G2​>,....,<sr​,Gr​>

然后更新函数：

Δθ=α(Gt−v^(St,θ))∇θv^(St,θ)\Delta \theta=\alpha\left(G_t-\widehat{v}\left(S_t, \theta\right)\right) \nabla_\theta \widehat{v}\left(S_t, \theta\right) Δθ=α(Gt​−v(St​,θ))∇θ​v(St​,θ)

基于 TD 方法的函数逼近

TD 目标为：

Rt+1+γv^(St+1,θ)R_{t+1}+\gamma \hat v(S_{t+1}, \theta) Rt+1​+γv^(St+1​,θ)

所以更新函数的方法为：

θt+1=θt+ΔθΔθ=α[R+γv^(St+1,θ)−v^(St,θt)]∇θv^(St,θt)\begin{aligned} &\theta_{t+1} = \theta_t + \Delta \theta \\ &\Delta \theta = \alpha \left[R+\gamma \hat v(S_{t+1}, \theta) - \widehat{v}\left(S_t, \theta_t \right)\right] \nabla_\theta \widehat{v}\left(S_t, \theta_t\right) \end{aligned} ​θt+1​=θt​+ΔθΔθ=α[R+γv^(St+1​,θ)−v(St​,θt​)]∇θ​v(St​,θt​)​

可以看到函数使用了两次，但是我们求导数值只对当前状态值 v^(St,θt)\widehat{v}\left(S_t, \theta_t\right)v(St​,θt​) 求导。

DQN

三维可视化助你直观理解DQN算法[DQN理论篇]

DQN = Deep learning + Q-learning

首先来看 Q-leaning 的伪代码：

最关键的两个点：

  1. 异策略：

    1. 行为策略：使用 ε−greedy\varepsilon - greedyε−greedy 行动策略 保证所有动作都能被选择。
    2. 目标策略，也就是第六行更新的策略，下一次选择让值函数最大的那个动作。

  2. 时间差分：不走完了，只走一步，得到奖励 rtr_trt​ 和下一步骤的最大价值 max⁡aQ(st+1,a)\max_a Q(s_t+1, a)maxa​Q(st​+1,a) 。

DQN 在此基础上，更新了三点：

  1. 使用 CNN 来逼近值函数 Q

  2. 「经验回放」的训练过程：
    训练神经网络时候，假设是训练数据为独立同分布的，但是通过强化学习采集到的数据存在关联性，利用这些数据进行顺序预测的话，效果很差。
    所以这里使用 random shuffle ，均匀随机采样，从数据库里抽取数据，然后训练。

  3. 独立设置了「目标网络」，来单独处理时间差分中的 TD 偏差
    训练时，目标为

θt+1=θt+α[r+γmax⁡a′Q(s′,a′;θ)−Q(s,a;θ)]∇Q(s,a;θ)\theta_{t+1}=\theta_t+\alpha\left[r+\gamma \max _{a^{\prime}} Q\left(s^{\prime}, a^{\prime} ; \theta\right)-Q(s, a ; \theta)\right] \nabla Q(s, a ; \theta) θt+1​=θt​+α[r+γa′max​Q(s′,a′;θ)−Q(s,a;θ)]∇Q(s,a;θ)

- 计算值函数的网路为 $\theta$ ，每一步骤都更新
- 计算 TD 目标 $r+\gamma \max _{a^{\prime}} Q\left(s^{\prime}, a^{\prime} ; \theta\right)$ 所用的网络为 TD 网络，这里采用一个单独的网络 $\theta^-$ ，用固定的步数间隔更新。

由此，目标变为

θt+1=θt+α[r+γmax⁡a′Q(s′,a′;θ−)−Q(s,a;θ)]∇Q(s,a;θ)\theta_{t+1}=\theta_t+\alpha\left[r+\gamma \max _{a^{\prime}} Q\left(s^{\prime}, a^{\prime} ; \theta^{-}\right)-Q(s, a ; \theta)\right] \nabla Q(s, a ; \theta) θt+1​=θt​+α[r+γa′max​Q(s′,a′;θ−)−Q(s,a;θ)]∇Q(s,a;θ)

相当于用Q网络估算出来的两个相邻状态的Q值，他们之间的距离，就是一个r的距离 。

Double DQN

DQN 无法避免 Q-learning 的一个缺点： 过估计

过估计是指需要被估计的函数值比真实函数值要大，根源来自于 Q-learning 中的 max 操作，同时每一个动作对应的价值的过估计量都是不同的，从而导致最终的策略并非最优，而是次优。

为了解决过估计的问题，Double DQN 被提出，即将 「动作的选择」 和 「动作的评估」 分别用不同的值函数来实现。

首先 TD 目标为：

YtQ=Rt+1+γmax⁡aQ(St+1,a;θt)Y_t^Q=R_{t+1}+\gamma \max _a Q\left(S_{t+1}, a ; \theta_t\right) YtQ​=Rt+1​+γamax​Q(St+1​,a;θt​)

动作选择：对每一个可能的动作 a ，用神经网络 Q 得到其价值，然后选择最大的那一个动作。
动作估计：拿到选择的动作之后，使用这个动作 a，带入到神经网络 Q 中，得到动作价值，构造 TD 函数。

以前的 DQN 这两个网络是相同的，而 Double DQN 使用到了两个网络来选择和评估动作价值，对应的 TD目标为：

YtDoubleQ =Rt+1+γQ( St+1,argmax⁡aQ(St+1,a;θt); θt′ )Y_t^{\text {DoubleQ }}=R_{t+1}+\gamma Q\left(\ S_{t+1}, \underset{a}{\operatorname{argmax}} Q\left(S_{t+1}, a ; \theta_t\right) ;\ \theta_t^{\prime}\ \right) YtDoubleQ ​=Rt+1​+γQ( St+1​,aargmax​Q(St+1​,a;θt​); θt′​ )

动作选择时，使用的动作值函数为

arg⁡max⁡aQ(St+1,a;θt)\arg \max _a Q\left(S_{t+1}, a ; \theta_t\right) argamax​Q(St+1​,a;θt​)

选出动作 a∗a^*a∗ 之后，动作评估函数为：

Rt+1+γQ( St+1,a∗; θt′ )R_{t+1}+\gamma Q\left(\ S_{t+1}, a^* ;\ \theta_t^{\prime}\ \right) Rt+1​+γQ( St+1​,a∗; θt′​ )
