简介

  1. Unsupervised pre-training：在大量的无标注数据集合上进行无监督预训练
  2. Supervised fine-tuning：然后在标注数据集上进行监督微调

adapt the model to a discriminative task with labeled data.

这种使用监督学习和无监督学习的方法也称为半监督学习 Semi-supervised learning。

模型结构

Unsupervised pre-training

  • 使用：
    使用类似Transformer中的decoder的结构作为特征抽取器，使用单向的模型结构，多层结构（12层）。
  1. 特征抽取器

    GPT Decoder包含一层Masked Multi-head Attention和一层前馈神经网络。并且在每层后面都进行了残差连接和归一化操作（norm）。
    使用decoder的原因是因为需要根据上文来预测下文，同时避免了多层双向传播的数据泄露问题。
  2. 单向的模型结构
    设置了窗口大小k，即在预测单词i时，仅仅根据i前面k个单词进行预测。
  3. 多层结构：12层decoder
  • 训练：使用最大似然的思想和随机梯度下降的方法来训练模型
    对于输入文本序列U=[u1,u2,...un]U=[u_1,u_2,...u_n]U=[u1​,u2​,...un​]，根据最大似然思想，基于过去窗口中的单词来预测当前单词。整个文本序列的损失函数为：

L1(U) = ∑i=k+1nlogP(ui∣ui−k,...,ui−1;θ)L_1(U)\ =\ \sum_{i=k+1}^{n} logP(u_i|u_{i-k},...,u_{i-1};\theta) L1​(U) = i=k+1∑n​logP(ui​∣ui−k​,...,ui−1​;θ)

k：窗口大小
θ\thetaθ：网络参数

  • 训练目标：每个位置输出的向量和正确答案向量（原词词向量）的相似度最大
    对于每个单词，其词向量由embedding和位置编码组成。

h0 = U^We+Wphi = transformerblock(hi−1)  ∀i∈[1,l]P(u) = softamx(hnWeT)\begin{aligned} &h_0\ =\ \hat UW_e+W_p\\ &h_i\ =\ transformer_{block}(h_{i-1})\ \ \forall i\in[1,l]\\ &P(u)\ =\ softamx(h_n W_e^T)\\ \end{aligned} ​h0​ = U^We​+Wp​hi​ = transformerblock​(hi−1​)  ∀i∈[1,l]P(u) = softamx(hn​WeT​)​

在训练好模型后，我们增加一些标签对模型进行微调来适应下游任务。
使用C^\hat CC^表示样本库，其中每个样本都包含一个序列X=[x1,x2,..,x3]X=[x_1,x_2,..,x_3]X=[x1​,x2​,..,x3​]，即输入的文本，同时对应一个标签yyy。
即C^=(X1,y1),(X2,y2)...,(XM,yM)\hat C={(X_1,y_1),(X_2,y_2)...,(X_M,y_M)}C^=(X1​,y1​),(X2​,y2​)...,(XM​,yM​)。

Task-specific input transformations

不同的任务需要不同结构的输入，

Previous work proposed learning task specific architectures on top of transferred representations.
Such an approach reintroduces a significant amount of task-specific customization and does not use transfer learning for these additional architectural components.

因为我们预先训练的模型是在连续的文本序列上训练的，所以我们需要一些转换来将它应用到这些任务中。
所有的转换都包括添加随机初始化的开始和结束标记(<s><s><s>，<e><e><e>)

  1. 对于分类问题，不用怎么动，加上一个起始和终结符号即可。
  2. 对于句子关系判断问题，比如 Entailment，两个句子中间再加个分隔符即可。
  3. 对文本相似性判断问题，把两个句子顺序颠倒下做出两个输入即可，这是为了告诉模型句子顺序不重要。
  4. 对于多项选择问题，则多路输入，每一路把文章和答案选项拼接作为输入即可。
