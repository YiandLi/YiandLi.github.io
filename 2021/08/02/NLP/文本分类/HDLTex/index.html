特征抽取

使用两种特征抽取方式：Glove embedding 和 N-gram TF-IDF

  1. GLove embedding：使用Glove 100维度的词向量
    则文档 djd_jdj​ 的向量表示为：dj=(w1,j,w2,j,...,wi,j,wlj,j)d_j = (w_{1,j}, w_{2,j}, ..., w_{i,j}, w_{l_j,j})dj​=(w1,j​,w2,j​,...,wi,j​,wlj​,j​)
    wi,jw_{i,j}wi,j​ 为文档j中单词i的Glove向量表示
    ljl_jlj​ 为文档j的单词个数。
  2. N-gram TF-IDF：

以文本 “In this paper we introduced this technique”为例
• Feature count (1): { (In, 1) , (this, 2), (paper, 1), (we, 1), (introduced, 1), (technique, 1) }
• Feature count (2): { (In, 1) , (this, 2), (paper, 1), (we, 1), (introduced, 1), (technique, 1), (In this, 1), (This paper, 1), (paper we, 1),…}

count(N)表示集合中最大的N-gram数量，这个特征的向量空间表示为：

fj=∑n=0Nfj,nf is the feature space of document j for n−grams of size nfj,n=[x(j,0),...,x(j,k−1),xj,{0,1},...,xj,{k−2,k−1},...,xj,{k−n,...,k−1}]\begin{aligned} &f_j = \sum_{n=0}^N f_{j,n} \\ &f\ is\ the\ feature\ space\ of\ document\ j\ for\ n-grams\ of\ size\ n \\ &f_{j,n} =[x_{(j,0)}, ..., x_{(j,k-1)}, x_{j,\{0,1\}}, ...,x_{j,\{k-2,k-1\}}, ..., x_{j,\{k-n,...,k-1\}}] \end{aligned} ​fj​=n=0∑N​fj,n​f is the feature space of document j for n−grams of size nfj,n​=[x(j,0)​,...,x(j,k−1)​,xj,{0,1}​,...,xj,{k−2,k−1}​,...,xj,{k−n,...,k−1}​]​

模型结构

作者首先介绍了了三种深度学习模型：DNN，RNN，CNN。

DNN

作为一个baseline，使用多层DNN拼接做层次标签任务。
输入：文章特征向量
输出：多/二分类器

RNN

xt=F(xt−1,ut,θ)which is xt=Wrecσ(xt−1)+Winut+b\begin{aligned} &x_t = F(x_{t-1}, u_t, \theta)\\ which\ is\ &x_t = W_{rec}\sigma(x_{t-1}) + W_{in}u_t+b \end{aligned} which is ​xt​=F(xt−1​,ut​,θ)xt​=Wrec​σ(xt−1​)+Win​ut​+b​

x表示输出/隐藏层状态
u表示输入

因为RNN中偏向保留尾部特征，所以作者使用了CNN中的最大池化。

CNN

作用：卷积层用于转换特征，池化层用于筛选重要特征。
问题：特征空间太大/管道数量太多。？？

Hierarchical Deep Learning for Text

  • DNN: 8 hidden layers with 1024 cells in each hidden layer.
  • RNN: GRU and LSTM are used in this implementation, 100 cells with GRU with two hidden layers.
  • CNN: Filter sizes of {3, 4, 5, 6, 7} and max-pool of 5, layer sizes of {128, 128, 128} with max pooling of {5, 5, 35}, the CNN contains 8 hidden layers.
