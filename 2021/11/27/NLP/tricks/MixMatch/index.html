MixMatch

介绍

传统半监督学习（Semi-supervised learning /SSL）都是通过在损失函数中添加与未标记数据相关的损失项，使模型可以泛化到未标注数据。
这个损失项一般为三种：
- entropy minimization：模型预测出低熵分布。
- consistency regularization：对扰动输入保持相同的概率分布。
- generic regularization：增强泛化能力，减少过拟合。
为未标记的数据引入统一的损失项，该项减少了熵，同时保持一致性并与传统正则化技术保持兼容。

相关工作

一个根据输入x和参数 θ\thetaθ 生成标签分布y的通用模型： pmodel⁡(y∣x;θ)\mathrm{p}_{\operatorname{model}}(y \mid x ; \theta)pmodel​(y∣x;θ)

Consistency Regularization 自洽正则化

对于一个无标注数据，原始的样本数据和增强后的样本数据经过模型，得到的预测值应该一致。
即新的损失项为：

∥pmodel (y∣ Augment (x);θ)−pmodel (y∣Augment⁡(x);θ)∥22\| \mathrm{p}_{\text {model }}(y \mid \text { Augment }(x) ; \theta)-\mathrm{p}_{\text {model }}(y \mid \operatorname{Augment}(x) ; \theta) \|_{2}^{2} ∥pmodel ​(y∣ Augment (x);θ)−pmodel ​(y∣Augment(x);θ)∥22​

注意数据增广是随机操作，两个 Augment(x)Augment(x)Augment(x) 的输出不同。
本文使用了对图像的随机左右翻转和剪切（Crop）。

Entropy Minimization 最小化熵

许多半监督学习方法都基于一个共识，即分类器的分类边界不应该穿过边际分布的高密度区域 (high-density regions of the marginal data distribution)。
传统做法就是，分类器给未标注数据输出低熵预测。实现方法为增加一个新的损失项：pmodel⁡(y∣x;θ)\mathrm{p}_{\operatorname{model}}(y \mid x ; \theta)pmodel​(y∣x;θ)
本文用了一种名为 “sharpening” 的方法来保证无标准数据的低熵预测。

Traditional Regularization 传统正则化

正则：对模型施加约束，避免其记忆训练样本从而只达到统计效果，从而让模型泛化性更高，即预测没有见过的样本。
本文使用了Adam实现权值衰减，并且将 Mixup 同时用在了标记数据和未标记数据中。

MixMatch

mixmatch集合上述三种方法

对带标签样本和无标签样本都进行样本增强：
对于 X\mathcal{X}X 中每一个带标签样本 xbx_bxb​ ，得到一个增强样本 x^b=Augment⁡(xb)\hat{x}_{b}=\operatorname{Augment}\left(x_{b}\right)x^b​=Augment(xb​) 。得到同尺寸的增强带标签样本数据集 X^\hat{\mathcal{X}}X^ 。
对于 U\mathcal{U}U 中每一个无标签样本 xbx_bxb​ ，得到k个增强样本 u^b,k=Augment⁡(ub),k∈(1,…,K)\hat{u}_{b, k}=\operatorname{Augment}\left(u_{b}\right), k \in(1, \ldots, K)u^b,k​=Augment(ub​),k∈(1,…,K) 。

Sharpening

熵的计算方式是 −∑i=1cpilog⁡pi-\sum_{i=1}^{c} p_{i} \log p_{i}−∑i=1c​pi​logpi​ ，表示信息的确定性。熵越低，一个事件的确定性也就更高。所以one-hot分布比均匀分布的熵要低很多。
使用了 Sharpen，来使得“伪”标签熵更低：

 Sharpen (p,T)i:=pi1T/∑j=1Lpj1T\text { Sharpen }(p, T)_{i}:=p_{i}^{\frac{1}{T}} / \sum_{j=1}^{L} p_{j}^{\frac{1}{T}}  Sharpen (p,T)i​:=piT1​​/j=1∑L​pjT1​​

ppp 在MixMatch中对应上面的 qˉb\bar{q}_{b}qˉ​b​ 。
TTT 是温度参数，趋向于0时，  Sharpen (p,T)\text { Sharpen }(p, T) Sharpen (p,T) 就更趋向于one-hot编码。所以降低T可以保证低熵分布。
最后，使用 qb=Sharpen⁡(qˉb,T)q_{b}=\operatorname{Sharpen}\left(\bar{q}_{b}, T\right)qb​=Sharpen(qˉ​b​,T) 来表示该无标注样本的标签。

经过数据增强和构造伪标签，我们得到了两个集合：

X^=((x^b,pb);b∈(1,…,B))U^=((u^b,k,qb);b∈(1,…,B),k∈(1,…,K))\begin{aligned} &\hat{\mathcal{X}}=\left(\left(\hat{x}_{b}, p_{b}\right) ; b \in(1, \ldots, B)\right) \\ &\hat{\mathcal{U}}=\left(\left(\hat{u}_{b, k}, q_{b}\right) ; b \in(1, \ldots, B), k \in(1, \ldots, K)\right) \end{aligned} ​X^=((x^b​,pb​);b∈(1,…,B))U^=((u^b,k​,qb​);b∈(1,…,B),k∈(1,…,K))​
