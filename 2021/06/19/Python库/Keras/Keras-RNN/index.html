Embedding Layer

文档：https://keras.io/zh/layers/embeddings/

简介

嵌入层 keras.layers.Embedding
将正整数表示的稀疏向量转换为固定尺寸的稠密向量。
只能用于模型的第一层。

参数：

  • input_dim：大于0的整数，表示词汇表的大小。
  • output_dim：大于0的整数，词向量的维度。
  • embeddings_initializer：矩阵初始化的方法。
  • embeddings_regularizer：矩阵的正则化方法。
  • embeddings_constraint
  • mask_zeros
  • input_length：输入序列的长度。

注意：input_dim和output_dim是调用构造函数必须要传的，初始化方式默认为uniform，其余都默认为None。

  • 输入尺寸：(batch_size, sequence_length)
  • 输出尺寸：(batch_size, sequence_length, output_dim)

SimpleRNN Layer

文档：https://keras.io/zh/layers/recurrent/#simplernn

  • 类：keras.layers.SimpleRNN
  • 常用参数：
    • units：必备参数，表示输出空间的维度
    • activation：激活函数
    • kernel_initializer：权值矩阵初始化器，用于输入的线性转换。
    • recurrent_initializer：权值矩阵 的初始化器，用于循环层状态的线性转换。
    • bias_initializer：偏置向量的初始化器。
    • return_sequences：是返回输出序列中最后一个元素，还是全部序列。
