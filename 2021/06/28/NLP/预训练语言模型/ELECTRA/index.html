“Efficiently Learning an Encoder that Classifies Token Replacements Accurately”

概述

  • BERT中的MLM（mask language model）使用[mask]标签来替换一些（15%）单词，然后训练一个模型去预测/重新生成这些单词。
    因为每次都只学习了15%的特征，所以训练代价很大。

  • 所以这里引入一个高效的方法RTD（replaced token detection）

    • 使用小型的生成模型生成单词，替换掉一些输入单词.
    • 然后使用一个判别模型判断句子中的每个单词是生成词还是原词。

    predict for every token whether it is an original or a replacement

  • 微调时只训练判别器

  • 这种判别的方式同时也可以解决BERT中mismatch的问题：训练时使用了[mask]，但是在微调时不会使用mask。

  • 在transformer的结果上继续训练。？？是否可以看作transformer的微调？？

we apply it to pre-train Transformer text encoders that can be fine-tuned on downstream tasks.

结构

Generator

  • 用于生成/预测一个单词，通常使用一个小型的神经网络。
    使用MLM模型来训练神经网络，bert是使用MLM训练atteneion。

MLM trains the network as a generator

  • 输入：
    一段文本x = [x1,x2...xn]x\ =\ [x_1, x_2 ... x_n]x = [x1​,x2​...xn​]
    • 经过一个encoder（w2v，trm…）得到的带有上下文信息的词向量列表 hidden representation。
      h(x) = [h1,h2..hn]h(x)\ =\ [h_1,h_2 .. h_n]h(x) = [h1​,h2​..hn​]，将h看作原词词向量列表（original）。
    • 同时随机初始化？列表中每个单词的word embedding，作为估计词词向量（guess）。
  • 实现：
    • 随机选取(15%)tokens，替换为mask token。

    Typically k=⌈0.15n⌉k = \lceil 0.15n \rceilk=⌈0.15n⌉, i.e., 15% of the tokens are masked out.

    • 然后对每一个mask token使用神经网络进行预测，得到对应的corrupted token，生成xcorruptx^{corrupt}xcorrupt序列。
      使用softmax的方式，找到和mask token的原词词向量最近似的估计词词向量。
      pG(xt∣x)=exp( e(xt)ThG(x)t ) ∑x′exp( e(x′)ThG(x)t )p_G(x_t|x) = \dfrac {exp(\ e(x_t)^T h_G(x)_t\ )}{\ \sum_{x'}exp(\ e(x')^Th_G(x)_t\ )}pG​(xt​∣x)= ∑x′​exp( e(x′)ThG​(x)t​ )exp( e(xt​)ThG​(x)t​ )​
      hG(x)th_G(x)_thG​(x)t​：位置t的原词词向量。
      e(xt)e(x_t)e(xt​)：位置t的word embedding，即估计词词向量。
      使用点积求词向量相似度，使用转置相乘表示点积。
    • 明确了最近似的token之后，使用原词词向量来表示预测词词向量（predict），这个过程称为sample。
  • 一些细节：
    • 生成器是一个MLM模型。

    The generator is trained to perform masked language modeling (MLM).

    • 这里准备工作有两个变换：
      输入可以看作one-hot编码，然后encode为原词向量，同时word emb为预测词。
    • 要训练的是word embedding

      因为softmax的判定，生成器可以训练到所有的word embedding，而以往计算BERT的MLM loss时会忽略没被mask的token。

Discriminator

predict which tokens in xcorruptx^{corrupt}xcorrupt match the original input x
predict whether each token in the corrupted input was replaced by a generator sample or not
predicts whether the token xtx_txt​ is “real”

  • 目标：预测 位置t的单词是否为原词
  • 实现：在得到位置t的词之后，将该词本来的原词词向量输入到一个神经网络中，输出的单值经过sigmoid输出层作二分类。

D(x,t) = sigmoid( wThD(x)t )D(x,t)\ =\ sigmoid(\ w^Th_D(x)_t\ ) D(x,t) = sigmoid( wThD​(x)t​ )

  • 要训练的是w表示的神经网络，点积后的相似度是否 >0.5>0.5>0.5。

过程中的序列变换

  1. [mask]的索引序列：m = [m1,m2,..,mk]m\ =\ [m_1, m_2,.., m_k]m = [m1​,m2​,..,mk​]
    mi ∼ unif{1,n} for i=1 to km_i\ \sim\ unif\{1,n\}\ for\ i=1\ to\ kmi​ ∼ unif{1,n} for i=1 to k
    ？？ unif：Uniform Distribution 均匀分布
  2. 根据m，用[mask]替换选定单词，得到新的序列 xmasked = REPLACE(x,m,[mask])x^{masked}\ =\ REPLACE(x,m,[mask])xmasked = REPLACE(x,m,[mask])
    mmasked = REPLACE(x,m.,[mask])m^{masked}\ =\ REPLACE(x,m.,[mask])mmasked = REPLACE(x,m.,[mask])
  3. 每个[mask]都要经过概率选择，替换为最有可能的单词x^\hat xx^
    x^i ∼ pG(xi∣xmasked) for i ∈ m\hat x_i\ \sim\ p_G(x_i|x^{masked})\ for\ i\ \in\ mx^i​ ∼ pG​(xi​∣xmasked) for i ∈ m
    经过替换后得到了新的替换词序列xcorruptx^{corrupt}xcorrupt，同时也是识别器的输入。
    xcorrupt = REPLACE(x,m,x^)x^{corrupt}\ =\ REPLACE(x,m,\hat x)xcorrupt = REPLACE(x,m,x^)

训练

单独的损失函数：

LMLM(x,θG) = E( ∑i∈m−logpG(xi∣xmasked) )LDisc(x,θD) = E(∑t=1n−1(xtcorrupt=xt)logD(xcorrupt,t)−1(xtcorrupt≠xt)log(1−D(xcorrupt,t)))\begin{aligned} &L_{MLM}(x,\theta_G)\ =\ E(\ \sum_{i\in m}- logp_G(x_i|x^{masked})\ )\\ &L_{Disc}(x,\theta_D)\ =\ E(\sum_{t=1}^{n}-1(x^{corrupt}_t=x_t)logD(x^{corrupt}, t)-1(x^{corrupt}_t \neq x_t)log(1-D(x^{corrupt}, t))) \end{aligned} ​LMLM​(x,θG​) = E( i∈m∑​−logpG​(xi​∣xmasked) )LDisc​(x,θD​) = E(t=1∑n​−1(xtcorrupt​=xt​)logD(xcorrupt,t)−1(xtcorrupt​=xt​)log(1−D(xcorrupt,t)))​

EEE：求数学期望，概率加权平均。

  • 生成器：LMLM(x,θG) = E( ∑i∈m−logpG(xi∣xmasked) )L_{MLM}(x,\theta_G)\ =\ E(\ \sum_{i\in m}- logp_G(x_i|x^{masked})\ )LMLM​(x,θG​) = E( ∑i∈m​−logpG​(xi​∣xmasked) )
    最大化m中所有位置的pG(xt∣x)p_G(x_t|x)pG​(xt​∣x)，因为是相互独立的，所以需要将m中所有位置的pG(xt∣x)p_G(x_t|x)pG​(xt​∣x)连乘起来。
    然后加上log方便计算，最后取其相反数作为损失函数，目标是最小化这个损失函数。
    E暂时没搞懂
    • 和GAN不同，生成器的训练目标是努力生成一个real样本，而不是努力生成一个fake样本来骗过识别器。
    • 使用最大似然训练的效果 比 强化训练生成神经网络的效果 更好。实际上也不能，因为sampling的过程。

    • we do not supply the generator with a noise vector as input, as is typical with a GAN.

  • 判别器：LDisc(x,θD) = E( ∑t=1n−1(xtcorrupt=xt)logD(xcorrupt,t)−1(xtcorrupt≠xt)log(1−D(xcorrupt,t)) )L_{Disc}(x,\theta_D)\ =\ E\bigg(\ \sum_{t=1}^{n}-1(x^{corrupt}_t=x_t)logD(x^{corrupt}, t)-1(x^{corrupt}_t \neq x_t)log(1-D(x^{corrupt}, t))\ \bigg)LDisc​(x,θD​) = E( ∑t=1n​−1(xtcorrupt​=xt​)logD(xcorrupt,t)−1(xtcorrupt​=xt​)log(1−D(xcorrupt,t)) )，是一个交叉熵损失函数。
    我们希望当xtcorrupt=xtx^{corrupt}_t=x_txtcorrupt​=xt​时，识别器输出的概率D(xcorrupt,t)D(x^{corrupt}, t)D(xcorrupt,t)越大越好，而反之不等时越小越好。
    因为使用了sigmoid函数，所以加入log方便计算，然后取反得到损失函数，目标是最小化这个损失函数。

联合损失函数：

minθG,θD = ∑x∈TLMLM(x,θG)+λLDisc(x,θD)min_{\theta_G,\theta_D}\ =\ \sum_{x\in T} L_{MLM}(x,\theta_G)+\lambda L_{Disc}(x,\theta_D) minθG​,θD​​ = x∈T∑​LMLM​(x,θG​)+λLDisc​(x,θD​)

T表示原始文本：在原始文本上选择单词序列/句子来进行训练。
最终是最小化联合损失（minimize the combined loss）。

  • 这两个神经网络是分别训练的，即判别器的损失不会传给生成器。
  • 生成器更新所有的token embedding，而判别器只更新被sample的embedding。
  • 预训练之后，继续单独对判别器进行微调。

模型拓展选择

  • small generator：
    • 生成器不能太厉害，否则一定会预测出正确的单词，这样判别器只需要判别正确即可。
      所以需要使用一个小型的生成器，当生成器的大小为判别器大小的1/2-1/4时，模型效果最好。
      换一种理解，出题人一定不能太厉害，否则训练答题人的难度会特别大。
    • 这里我理解的小型是考虑的词向量叠加的层数，即论文中写的是layer size (number of hidden units).
      同时也解释了联合损失函数中的minθG,θDmin_{\theta_G,\theta_D}minθG​,θD​​

  • Weight Sharing：
    让生成器和判别器共享一些参数可以加快预训练的速度，但是并不是共享所有的参数。

    为什么会加速：生成器更新所有的token embedding，而判别器只更新被sample的embedding。训练一个，可以同时更新另一个。

训练算法

上述的训练方式是训练联合损失。此外，论文中提到了两种其他的训练方式，但是都不如原本的训练方式好：

  • two stage training:
    这种方式只适用于生成器和判别器大小相同的情况。
    • 首先只训练MLM模型n次
    • 然后使用MLM的参数初始化判别器，冻结MLM，单独训练判别器n次。
      如果不初始化，则会导致出题人太强而答题人训练失败的情况。
  • 借鉴GAN的思想，生成器的目标改为生成一个最有可能的赝品。
