Keras — 基本

Posted on 2021-06-16 Edited on 2021-06-25

搭建网络（Dense层）

全连接层

from keras.models import Sequential
from keras.layers import Dense

  • 创建全连接层Dense层：keras.layers.Dense()
    参数为：
    • units：单元数量，即该层中神经元的数量
    • activation=None：激活类型
    • use_bias=True
    • kernal_initializer=‘glorot_uniform’：内核初始化
    • bias_initializer=‘zeros’：偏差初始化
    • input_dim：需要为第一层定义输入形状，后续层只需要定义神经元的数量即可。
    • kernel_regularizer：正则化方式
    • …

初始化器

初始化器定义了设置Keras各层权重随机初始化的方法。
用来将初始化器传入Keras层的参数名取决于具体的层，通常为kernel_initializer和bias_initializer。

  • 初始化器及其名称：
    • zeros： keras.initializers.Zeros()，张量初始值设置为0。
    • ones：keras.initializers.Ones()，张量初始值设置为1。
    • Constant：keras.initializers.Constant(value=0)，张量初始值设为一个常数。
    • RandomNormal：keras.initializers.RandomNormal(mean=0.0, stddev=0.05, seed=None)
      按照正态分布生成随机张量。
    • RandomUniform：keras.initializers.RandomUniform(minval=-0.05, maxval=0.05, seed=None)
      按照均匀分布生成随机张量的初始化器。
      …
  • 调用方式：
    • 作为字符串传递，需要把驼峰命名法改为下划线命名法，将使用默认参数。
      model.add(Dense(64, kernel_initializer='random_normal'))
    • 作为可用函数传，可以传入参数。
      model.add(Dense(64, kernel_initializer=initializers.random_normal(stddev=0.01)))
    • 自定义初始化器

正则化器

  • 文档：https://keras.io/zh/regularizers/
  • 可用的正则化器：
    • l1：keras.regularizers.l1(0.)
    • l2：keras.regularizers.l2(0.)
    • l1l2：keras.regularizers.l1_l2(l1=0.01, l2=0.01)
  • 例子：kernel_regularizer=regularizers.l2(0.01),

激活函数

  • 文档：https://keras.io/zh/activations/
  • 在构造层对象时传递 activation 参数实现。
  • 分类：
    • softmax：keras.activations.softmax(x, axis=-1)
    • relu：keras.activations.relu(x, alpha=0.0, max_value=None, threshold=0.0)
    • tanh：keras.activations.tanh(x)
    • sigmoid：sigmoid(x)

Dropout层

将正则化（regularization）和泛化（generalization）功能引入模型，来减少帮助过拟合。
本层会丢弃一些神经元或者将他们设置为0，减少计算量。
keras.layers.Dropout(rate, noise_shape=None, seed=None)

编译模型

model.compile()

损失函数 loss

回归

  • 均方差：实际值和误差值之差的平方的均值
    用于在差值更大时更严厉地惩罚模型
    keras.losses.mean_squared_error(y_actual, y_pred)
  • 平方绝对误差
    keras.losses.mean_absolute_error(y_actual, y_pred)

分类

将分类结果转化为概率，并且基于预测值的概率来估计损失

  • 二元交叉熵：定义分类结果为二元变量
    Loss = -[ y*log p + (1-y)*log(1-p) ]
    keras.losses.binary_crossentropy(y_actual, y_pred)
  • 分类交叉熵：用于分类结果是非二元的
    keras.losses.categorical_crossentropy(y_actual, y_pred)

优化方法 optimizer

文档：https://keras.io/zh/optimizers/

  • 调用方式：
  1. 传入类：optimizer= optimizers.SGD(lr=0.01, clipnorm=1.)，可以后面自定义参数。
  2. 传入名称：optimizer=‘sgd’

随机梯度下降法 sgd

keras.optimizers.SGD(lr=0.01, momentum=0.0, decay=0.0, nesterov=False)

  • 为了降低SGD优化中的高波动，比较好的方法是通过提供小分批来减少迭代次数，批尺寸通常设置为2的幂。
  • 参数：lr 学习率>=0

自适应矩估计 Adam

为每个参数计算一个自适应学习率，定义了损失梯度的动量(momentum)和方差(variance)，并且利用组合效应来更新权重参数。
权重 -= 动量和方差的组合
keras.optimizers.Adam(lr=0.01, beta_1=0.9, beta_2=0.999, epsilon=None, decay=0.0, amsgrad=Flase)

RMSprop

通常用于训练RNN

  • 参数：lr

评估指标 metrics

用于判断模型在另一个未接触过的数据集上的性能，与误差函数类似，但是不被用于模型优化。

  1. 二元准确率 keras.metrics.binary_accuracy
  2. 分类准确率 keras.metrics.categorical_accuracy
  3. 稀疏分类准确率 keras.metrics.sparse_categorical_accuracy

model的evaluate方法会接受测试数据和测试标签集合，并且返回结果。

另外可以使用predict方法来得到实际的预测结果:

训练模型

Keras提供了fit函数，用于根据给定的训练数据训练模型对象。
model.fit(x_train, y_train, batch_size=64, epochs=3, validation_data=(x_val, y_val))

测试模型

使用evaluate函数，传入样本和标签，函数返回loss和accuracy。
loss, acc = model.evaluate(x_test, y_test)

  • 参数：
    • x=None y=None: 测试数据和标签。
    • batch_size: 整数或 None。每次评估的样本数。如果未指定，默认为 32。
