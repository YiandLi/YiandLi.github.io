argmax() / argmin()

tensor.argmax()，传入维度
torch.argmax()，传入tensor和维度
返回的是每个维度对应的序号

max() / min()

返回位置和具体的值，返回的是元组形式。

使用a.max(1)[1]来得到最大值结果。

sort()

tensor.sort() / torch.sort()
排序并且返回一个namedtuple:(values, indices)，即排序后的tensor，和tensor元素的原始位置。

参数：

  • dim (int, optional)
  • descending = False (bool, optional)：是否降序，默认为False（升序）

使用a.sort(1)[0]得到排序后结果。
