Numpy: N维数组容器

引入了ndarray-多维数组的概念，可用于储存和处理大型矩阵。

一些基本操作：

导入numpy库：

import numpy as np

创建ndarray对象：

  1. np.array(列表)
    输入列表，并返回其ndarray形式

  1. np.arange()
    参数：
    a. 1个参数a，生成0到a-1的列表
    b. 2个参数a,b，生成a到b-1的列表
    c. 3个参数a,b,c，生成a到b-1的列表，步长为c

  2. np.linspace(起始值，终结值，元素数量)可以将元素均匀分布

  3. zeros(),ones(),empty()
    参数：
    a. 长度，必需
    b. dtype=“”，非必选
    empty()生成的其元素是任意值，等待被赋值。

ndarray对象的一些基本属性：

  1. shape：尺寸
  2. dtype：内部元素的数据类型
  3. T：自身的转置矩阵
  • 批量运算：

一些常用的方法：

  1. sum()： 返回所有元素的和

  2. ndarray数组截取：

    • 索引截取：被截取数组[起始索引:结尾索引:步长]
    • 注意：
      不包含结尾索引。
      numpy的截取是一个指针指向原数组，不会重新开辟内存。所以改变截取数组也会改变原数组。可以使用copy()来操作，不影响原数组。
    • 多维数组的截取：被截取数组[行切片，列切片]
    • 布尔型截取：传入一个相同长度的布尔形式的数组索引，过滤，返回对应值为true的值的数组.
      可以使用批量处理和布尔型截取来筛选数据，注意多个条件使用：并且->(条件1)&(条件2) ；或->(条件1)|(条件2)
    • 花式索引：传入一个索引的列表 (列表套列表)
    • 可以使用这三种索引方式组合来索引多维数组

  3. copy()：复制某个数组，对复制数组修改不会影响原数组。

  4. reshape(size的元祖数量)：改变矩阵尺寸

  5. np.abs()：返回ndarray每个元素绝对值的数组

  6. np.sqrt()：返回开方，负数开方为nan

  7. np.round()/ np.rint()：每个元素四舍五入

  8. np.trunc()：向0取整

  9. np.floor() 和 np.ceil()：向下和向上取整

  10. np.modf()：返回整数和小数部分数组，共返回2个元素

  11. np.isnan：判断是否为一个数字

  12. np.inf表示最大的数，通常是被除数为0的结果。

  1. np.var()方差 np.std()标准差
  2. np.max() np.min(), np.argmax()返回最大数的下标, np.argmin()返回最小值的下标

staticmethod & classmethod

Posted on 2021-02-02 In Coding , Python
Symbols count in article: 511 Reading time ≈ 3 mins.

一般来说，要使用某个类的方法，需要先实例化一个对象再调用方法。
而使用 @staticmethod 或 @classmethod，就可以不需要实例化，直接 类名.方法名() 来调用。

区别

一般来说，Class 内部的 Method 首个参数必须是表示类本身的 self（cls） 参数。

  • @staticmethod 为静态方法装饰器，不需要表示自身对象的 self 和自身类的 cls 参数，就跟使用函数一样。
    • 如果内部需要调用这个类的一些属性方法，则只能直接 类名.属性名 或 类名.方法名() 。
    • 但是如果调用正常的类方法的话，不会默认传入 self 参数，则需要额外参数 类名 作为第一个参数。

  • @classmethod 类方法装饰器，也不需要 self 参数，但第一个参数需要是表示自身类的 cls 参数。
    • 如果内部需要调用这个类的一些属性方法，除了直接 类名.属性名 或 类名.方法名() 外，还可以用 cls() 先初始化类，再来调用类的属性，类的方法，实例化对象等，避免硬编码。

self 和 cls

cls 和 self 其实只是一种命名习惯习惯, 是在 PEP8中定义的。
cls 表示类本身，所以需要实例化。
self 表示类的实例化，直接调用就好。

文件夹作为包

对于一个python项目，里面的每一个文件夹都可以认为是一个 package ，而每一个 .py 文件被认为是一个 module 。
如果一个 python 项目文件夹下含有 __init__.py 文件，那么这个文件夹便是一个 python 库。

当我们从一个 package 里面调用东西的时候，其内部 __init__.py 文件内的代码会被首先执行。
无论是直接 import package_1 这样直接导入整个包，还是通过 from package_1 import module 这样仅仅导入一个方法，都会被执行。

于是通常我们需要导入 package_1 文件夹内 pyfile1 文件中的 method1 方法时，我们需要写下 from package_1.pyfile1 import method1，多个外部文件中我们都需要这样写，比较繁琐。
我们可以在其对应的 __init__.py 文件内写下 from package_1.pyfile1 import method1，然后外部文件直接调用 from packaged_1 import method1 即可，逻辑就是执行了 from package_1.pyfile1 import method1 ，所以现在 __init__ 有 method1，然后我们再通过 __init__ 直接调用 method1。
__init__.py相当于 一个接口，我们在其内部注册一些方法，然后package 外面只能调用这个接口返回的某些方法和类别。

更重要地，该接口还定义了哪些方法是外部不可调用的，比如只注册方法 a_function_1() ：

  4. 一般情况下，在目录很深时，我们使用 init，同时在 __init__ 中定义导入别名 as 来简化。

迭代求解：随机梯度下降 Stochastic Gradient Descent

上面讲的梯度下降是针对整个训练集合的样本的，即对整个训练集作后向传播求到总损失，然后做平均，再求参数更新量。
这个参数更新量同样可以看做每个样本对应的更新量的平均：

∂J∂w=1N∑i=1N∂Li∂w\frac{\partial \mathcal J}{\partial w} = \frac 1N\sum_{i=1}^N \frac{\partial \mathcal L^i}{\partial w} ∂w∂J​=N1​i=1∑N​∂w∂Li​

这种整体求损失得到梯度的方式我们称之为batch training。
但是当我们的样本数量很多时，这种方式很不实用。

所以我们引入 随机梯度下降（SGD） 方式，根据 一个 随机样本更新参数。
SGD可以在遍历整个数据集之前就很好地拟合模型。

同时如果样本是随机选择的，那么这些随机样本梯度的期望就是整个batch的梯度。

E[∂Li∂w]=1N∑i=1N∂Li∂w=∂J∂wE\left[ \frac{\partial \mathcal L^i}{\partial w} \right] = \frac 1N\sum_{i=1}^N \frac{\partial \mathcal L^i}{\partial w} = \frac{\partial \mathcal J}{\partial w} E[∂w∂Li​]=N1​i=1∑N​∂w∂Li​=∂w∂J​

问题和解决：
只使用一个样本会导致方差变大，所以可以选择一个小一点的minibatch。
类似地，minibatch过小也会导致方差过大。

字符串函数

MySQL 内置了很多字符串函数：

  1. concat(s1, s2, ..., sn)
  2. lower()，upper()：转换大小写
  3. lpad(str,n,pad)，rpad(str,n,pad)：左/右填充，用字符 pad 对 str 的左/右进行填充，填充到长度为 n。
  4. trim(str)：去掉头尾空格
  5. substring(str, start, len)

数值函数

  1. ceil(x) 向上取整
  2. floor(x) 向下取整
  3. mod(x,y) 表示 x/y 的余数，即 x%y
  4. rand() 返回(0,1)内的随机数
  5. round(x,y) 求 x 的四舍五入的值，保留 y 位小数

日期函数

curdate()：返回当前日期
curtime()：返回当前时间
now()：返回当前日期和时间
year(date),month(date),day(date)：获取传入的date的年月日
date_add(date, interval [num] type)：返回date之后interval [num] type的日期。interval [num] type是固定写法，[num]是数值，type为单位，可以是day / month / year / ...
datediff(date1, date2)：返回两个日期之间的差值 date1-date2，返回的单位是 天。

流程控制函数

if(value, t, f)：如果 value==true，则返回 t，否则返回 f。
ifnull(value1, value2)：如果 value1!=Null，则返回 value1；否则返回 value2。
case when [val1] then [res1] ... else [default] end：如果 val1==True，返回 res1，否则返回 default 默认值。
case [expr] when [val1] then [res1] ... else [default] end：如果 expr==val1，返回 res1，否则返回 default 默认值。
