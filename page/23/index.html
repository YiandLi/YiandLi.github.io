python 多进程

Posted on 2021-02-14 Edited on 2022-06-30

实例化进程

通过 python demo.py 命令开始执行 demo.py 这个程序时，程序被赋予了声明，成为一个进程，这个进程是主进程。
Process() 方法构造的进程都是子进程。

进程启动

上一步初始化进程后，通过 p.start() 方法启动进程。
有三种启动方式：

  • spawn：子进程仅继承有限的资源。适用于 Unix/Linux 和 Windows，Windows 和 MacOS 的默认方式。
  • fork：子进程会继承父进程中所有的资源，仅适用于。Unix/Linux，Unix/Linux 默认方式。
  • forkserver：创建一个单进程的服务进程，专门用来处理子进程的创建，仅适用于 Unix/Linux。

阻塞主进程

p.join() 方法会使主线程进入阻塞，直到调用 p.join() 方法的子线程执行完毕。

例1：进程1，2同步进行

例2：进程1先开始，结束后进程2才开始

Pool() 方法

用于一次性创建多个进程
构造进程池：p=Pool(3)，参数表示每次执行的子进程个数的限制。
Pool的默认大小是你所用的电脑CPU的核数，CPU核数可通过os.cpu_count()获得。
进程池也有阻塞函数p.join()，但是在调用p.join()前需要调用p.close()方法。

进程间通信

每个子进程享有独立的内存空间，接收进程产生的数据不能马上同步到转发进程中

pandas: 表格容器

基于NumPy的一种工具，该工具是为了解决数据分析任务而创建的。Pandas纳入了大量库和一些标准的数据模型，提供了高效地操作大型数据集所需的工具。

导入pandas库：

import pandas as pd

Series对象

Series对象的创建

使用pd.Series()方法
参数：1. 数组 2. index=数组，非必选

Series对象的支持的特性

  1. 索引方式： a. 下标索引/键值索引/花式索引

  3. index属性返回标签的列表，values返回值的列表
  4. 可以和标量，或者series对象批量运算。注意这里牵扯到数据对齐：两个series对象运算时，是按照标签来对应的，而不是索引。如果存在标签不同的，则结果为NaN。

    a. sr1.add(sr2, fill_value=默认值)：添加并将NaN设置为默认值。
    b. sr.isnull()/notnull()：如果值为NaN，返回True/False，否则返回false/True。
    c. sr.dropna()：返回删除所有缺失值的Series对象
    d. sr.fillna(参数)：填充NaN并返回。

    4. fill_value:指定移位后的填充值，fill_value=0，即表示移位后缺失值填充为0

numpy库的简单讲解

Posted on 2021-02-08 Edited on 2021-12-09

Numpy: N维数组容器

引入了ndarray-多维数组的概念，可用于储存和处理大型矩阵。

一些基本操作：

导入numpy库：

import numpy as np

创建ndarray对象：

  1. np.array(列表)
    输入列表，并返回其ndarray形式

  1. np.arange()
    参数：
    a. 1个参数a，生成0到a-1的列表
    b. 2个参数a,b，生成a到b-1的列表
    c. 3个参数a,b,c，生成a到b-1的列表，步长为c

  2. np.linspace(起始值，终结值，元素数量)可以将元素均匀分布

  3. zeros(),ones(),empty()
    参数：
    a. 长度，必需
    b. dtype=“”，非必选
    empty()生成的其元素是任意值，等待被赋值。

ndarray对象的一些基本属性：

  1. shape：尺寸
  2. dtype：内部元素的数据类型
  3. T：自身的转置矩阵
  • 批量运算：

一些常用的方法：

  1. sum()： 返回所有元素的和

  2. ndarray数组截取：

    • 索引截取：被截取数组[起始索引:结尾索引:步长]
    • 注意：
      不包含结尾索引。
      numpy的截取是一个指针指向原数组，不会重新开辟内存。所以改变截取数组也会改变原数组。可以使用copy()来操作，不影响原数组。
    • 多维数组的截取：被截取数组[行切片，列切片]
    • 布尔型截取：传入一个相同长度的布尔形式的数组索引，过滤，返回对应值为true的值的数组.
      可以使用批量处理和布尔型截取来筛选数据，注意多个条件使用：并且->(条件1)&(条件2) ；或->(条件1)|(条件2)
    • 花式索引：传入一个索引的列表 (列表套列表)
    • 可以使用这三种索引方式组合来索引多维数组

  3. copy()：复制某个数组，对复制数组修改不会影响原数组。

  4. reshape(size的元祖数量)：改变矩阵尺寸

  5. np.abs()：返回ndarray每个元素绝对值的数组

  6. np.sqrt()：返回开方，负数开方为nan

  7. np.round()/ np.rint()：每个元素四舍五入

  8. np.trunc()：向0取整

  9. np.floor() 和 np.ceil()：向下和向上取整

  10. np.modf()：返回整数和小数部分数组，共返回2个元素

  11. np.isnan：判断是否为一个数字

  12. np.inf表示最大的数，通常是被除数为0的结果。

  1. np.var()方差 np.std()标准差
  2. np.max() np.min(), np.argmax()返回最大数的下标, np.argmin()返回最小值的下标

np中的random库

作用：可以用于生成随机数组
参数：前两个参数为随机区间，最后一个为数组的尺存(可以是一个元组)
方法：

  1. randint(x,y,size) 生成随机整数，[x, y)
  2. rand() 生成0到1的随机数，rand(x)生成生成0到1的x个元素的一维数组，rand(x,y)生成x*y的数组
  3. randn()以给定的形状创建一个数组，数组元素来符合标准正态分布N(0,1)
  4. choice(x,y) 从数组x中随机选择元素，生成y尺寸大小的数组

numpy.random.RandomState()

为一个伪随机数生成器

伪随机数是用确定性的算法计算出来的似来自[0,1]均匀分布的随机数序列。并不真正的随机，但具有类似于随机数的统计特征，如均匀性、独立性等。(摘自《百度百科》)

直接传入数字，只要数字相同，则生成数相同。

线性回归

问题

输入：每个样本 x∈Xx\in \mathcal Xx∈X ，对应标签 t∈Tt\in \mathcal Tt∈T 。
目标：找到函数 f:X→Tf:\mathcal X \rightarrow \mathcal Tf:X→T ，即 t≈y=f(x)t \approx y=f(x)t≈y=f(x) 。

模型

样本： x=(x1,...,xD)∈RDx=(x_1,...,x_D)\in \mathbb{R}^Dx=(x1​,...,xD​)∈RD ，标签值 t∈Rt\in \mathbb{R}t∈R
目标：找到函数 y=f(x)≈ty=f(x) \approx ty=f(x)≈t

y=f(x)=∑jwjxj+by = f(x) =\sum_j w_jx_j + b y=f(x)=j∑​wj​xj​+b

y为预测值(prediction)
w和b为参数，w为权重(weights)，b为偏置(bias)

损失函数

单个样本的平方差损失函数(Squared error loss function)：

L(y,t)=12(y−t)2\mathcal L(y,t) = \frac 12(y-t)^2 L(y,t)=21​(y−t)2

y−ty-ty−t 为残差 residual 。

整个训练集的损失函数(Cost function)为：

J(w,b)=12N∑i=1N(yi−ti)2=12N∑i=1N(wTxi+b−ti)2\begin{aligned} \mathcal J(w,b)=&\frac 1{2N}\sum_{i=1}^{N}(y^i-t^i)^2\\ = &\frac 1{2N}\sum_{i=1}^{N}(w^Tx^i+b-t^i)^2 \end{aligned} J(w,b)==​2N1​i=1∑N​(yi−ti)22N1​i=1∑N​(wTxi+b−ti)2​

向量化表示 yyy ：

Xw+b1=(wTx1+b...wTxN+b)=(y1...yN)=yXw+b1 = \begin{pmatrix}w^Tx^1+b\\...\\w^Tx^N+b\end{pmatrix}=\begin{pmatrix}y^1\\...\\y^N\end{pmatrix}=y Xw+b1=⎝⎛​wTx1+b...wTxN+b​⎠⎞​=⎝⎛​y1...yN​⎠⎞​=y

可以将b放入w中，

X  =  [1[x1]T1[x2]T1...]∈RN∗(D+1)    and    w=[bw1w2...]∈RD+1X\;=\;\begin{bmatrix}1&{\lbrack x^1\rbrack}^T\\1&{\lbrack x^2\rbrack}^T\\1&...\end{bmatrix}\in\mathbb{R}^{N\ast(D+1)}\;\;and\;\;w=\begin{bmatrix}b\\w_1\\w_2\\...\end{bmatrix}\in\mathbb{R}^{D+1} X=⎣⎢⎡​111​[x1]T[x2]T...​⎦⎥⎤​∈RN∗(D+1)andw=⎣⎢⎢⎡​bw1​w2​...​⎦⎥⎥⎤​∈RD+1

于是变为

y=Xwy=Xw y=Xw

求解

使用最小化损失函数的方法求解
从数学角度来讲，有代数角度和微积分角度。
从方式角度来讲，有直接求解和迭代求解。

直接求解：代数 Algebraic

对一个样本来说，我们想最小化 ∣∣Xw−t∣∣2||Xw-t||^2∣∣Xw−t∣∣2 或者是 ∣∣Xw−t∣∣||Xw-t||∣∣Xw−t∣∣ 。
将 XwXwXw 看做一个D维的超平面，将 ttt 看做一个点。
目标变为：最小化超平面上到点t的距离。即在平面上找到点 y∗=Xw∗y^*=Xw^*y∗=Xw∗ 使得 (y∗−t)⊥Xw(y^*-t) \perp Xw(y∗−t)⊥Xw ，于是求解：

XT(y∗−t)=0XTXw∗−XTt=0w∗=(XTX)−1XTtX^T(y^*-t)=0\\ X^TXw^*-X^Tt=0\\ w^*=(X^TX)^{-1}X^Tt XT(y∗−t)=0XTXw∗−XTt=0w∗=(XTX)−1XTt

直接求解：微积分 Calculus

迭代求解：随机梯度下降 Stochastic Gradient Descent

上面讲的梯度下降是针对整个训练集合的样本的，即对整个训练集作后向传播求到总损失，然后做平均，再求参数更新量。
这个参数更新量同样可以看做每个样本对应的更新量的平均：

∂J∂w=1N∑i=1N∂Li∂w\frac{\partial \mathcal J}{\partial w} = \frac 1N\sum_{i=1}^N \frac{\partial \mathcal L^i}{\partial w} ∂w∂J​=N1​i=1∑N​∂w∂Li​

这种整体求损失得到梯度的方式我们称之为batch training。
但是当我们的样本数量很多时，这种方式很不实用。

所以我们引入 随机梯度下降（SGD） 方式，根据 一个 随机样本更新参数。
SGD可以在遍历整个数据集之前就很好地拟合模型。

同时如果样本是随机选择的，那么这些随机样本梯度的期望就是整个batch的梯度。

E[∂Li∂w]=1N∑i=1N∂Li∂w=∂J∂wE\left[ \frac{\partial \mathcal L^i}{\partial w} \right] = \frac 1N\sum_{i=1}^N \frac{\partial \mathcal L^i}{\partial w} = \frac{\partial \mathcal J}{\partial w} E[∂w∂Li​]=N1​i=1∑N​∂w∂Li​=∂w∂J​

问题和解决：
只使用一个样本会导致方差变大，所以可以选择一个小一点的minibatch。
类似地，minibatch过小也会导致方差过大。
