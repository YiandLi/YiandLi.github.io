三种 ： KFold、StratifiedKFold、GroupKFold

KFold

直接随机的将数据划分为k折，不考虑 train set 和 eval set 的 class 和 group 分布情况。
只需要传入数据X即可，模型会动态返回两个数组train, test，分别对应 train example 和 test example 的索引。

模型会递归输出 n_splits 次，其实每次相当于单拎出来一个 fold 作为 test 集合，将其他的 fold 拼接起来，这样就看不到具体是如何划分 fold 的了，即每次只能显示输出一个 fold 的索引，即 test。

更常用的是动态根据每次生成的单个 test 索引集合设置 fold_id ：

这样后续只需要 topic_df_non_source[topic_df_non_source["fold"]=1] 即可多次拿出对应的fold。

StratifiedKFold

根据数据集的分布来划分，使得划分后的数据集的目标比例和原始数据集近似，即 train set 和 eval set 的类别分布都和原始数据集分布近似。
这里需要同时传入标签列表 y ，

GroupKFold

保证同一个group的数据不会同时出现在训练集和测试集上。
因为如果训练集中包含了每个group的几个样例，可能训练得到的模型能够足够灵活地从这些样例中学习到特征，在测试集上也会表现很好。但一旦遇到一个新的group它就会表现很差。


可以将 group 理解成 domain，这样让同一个 domain 的数据只出现在 train set 或者 dev set，是想评价模型的迁移能力。
需要同时传入数据集合的 group信息。

时间序列模型

Posted on 2021-12-12 In Machine Learning
Symbols count in article: 847 Reading time ≈ 4 mins.
  • Time Series：时间序列就是一段按照时间排序，等时间间隔的观测值 yty_tyt​ 。
    Time Series Model：连续时间指标的值是相关的，否则无法做出预测。
  • 机器学习中，时间序列被描述为依靠时间t的一个向量/标量的序列: {x(t0),x(t1),...,x(ti−1),x(ti),x(ti+1)}\{x(t_0), x(t_1), ..., x(t_{i-1}), x(t_i), x(t_{i+1})\}{x(t0​),x(t1​),...,x(ti−1​),x(ti​),x(ti+1​)} 。
    设时间间隔为 Δt\Delta tΔt ，则可以写作 {x[t]}={x(0),x(Δt),x(2Δt),...}\{x[t]\} = \{x(0), x(\Delta t), x(2\Delta t), ...\}{x[t]}={x(0),x(Δt),x(2Δt),...} 。
  • 问题定义：已知时间序列 {x[t], x[t-1], …}，我们想预测未来时间的x值。
    x^[t+s]=f(x[t],x[t−1],...)\hat x[t+s] = f(x[t], x[t-1], ...)x^[t+s]=f(x[t],x[t−1],...) ，s为预测范围horizon of prediction。
  • 解决方法为，构造一个生成模型，对每一点 x[ti]x[t_i]x[ti​] ，将先于 tit_iti​ 之前的值当作输入，后于 tit_iti​ 的值作为目标值进行训练。
  • 难点：
    • 数据量有限：观测序列太短
    • 噪声：错误的样本
    • 非平稳性(Non-stationarity)：标准(fundamentals)随着时间改变
    • 预测方法(forecasting method)：统计 or AI
  • 使用神经网络，使用大小为 sss 的窗口在输入序列上滑动 （sliding window），将窗口内的时间序列映射到[0,1]区间，用做输入 Xt−s,Xt−s+1,...,XtX_{t-s}, X_{t-s+1},..., X_tXt−s​,Xt−s+1​,...,Xt​ ，使用MLP得到一个输出 Xt+1X_{t+1}Xt+1​ 。
  • 我们需要模型储存一个状态或者记忆(maintains a state or memory)，或者是在内存中储存过去的时间步。MLP和RBF都是静态网络(static networks)：从单独的输入得到单独的输出。
    动态网络可以学习到映射关系：一个单独的输入信号到一个响应信号序列。就是模型从序列的一个样本得到剩下的其他样本。
  • 可以使用两种方式额外传入记忆因子 memory effects：
    • 隐式 implicit：时间信号(time lagged signal) 作为输入，传入一个静态网络/循环网络。
      假设样本的环境是静止的。
    • 显式 explicit：时间后向传播算法(temporal backpropagation method)
      假设样本的环境是非静止的。
      模型可以追踪样本的结构变化。

Time Delayed Network 时延网络

使用了隐式表示 implicit representation approach的方式进行序列学习 sequence learning：组合了静态网络 static network（MLP/RBF）和记忆结构 memory structure。

记忆信号被额外缓存 buffered externally，作为额外的输入节点/输入缓存 additional input nodes。

Dynamical Neural Networks 动态神经网络

循环网络（Recurrent networks） 可以显示地处理序列。
和前向传播网络不同的是：在处理输入之外，还会处理一个状态。
网络有一个 工作记忆 working memory ，即隐状态，支持模型进行回忆并且维持活动能力。

Elman Network

将上一时间步的隐藏层状态 zzz 缓存起来，和输入 xxx 一起得到现在的隐藏层状态，然后从隐藏层得到输出 yyy 。

y=Fw(z)zt=Fv(zt−1,x)\begin{aligned} y &= F_w(z)\\ z^t &= F_v(z^{t-1}, x) \end{aligned} yzt​=Fw​(z)=Fv​(zt−1,x)​

  • 四层结构：
    • 输入层
    • 上下文层：作为输入层的额外补充，从隐藏层到上下文层的权重为1，因为要精准copy。
    • 隐藏层：生成内部表征。即合并输入层和上下文层的信息，并且复制到上下文层为了下一个时间步复用，并且传输到输出层。
    • 输出层：线性组合

径向基神经网络

Symbols count in article: 2.5k Reading time ≈ 13 mins.

机器学习的任务大多都是输入输出的映射问题，网络就是需要学习这种映射函数。
所以神经网络的设计可以看做一种曲线拟合问题 curve-fitting或者是函数近似问题 function approximation。
曲线拟合问题就像是寻找一个插值平面（interpolation surface） 在高维空间中拟合样本，即插值问题interpolation problem。

径向基函数 Radial Basis Function

定义：函数的值依赖于空间中点 xxx 到中心点 ccc 之间距离。是一个单调函数，可记为 ϕ(x,c)=ϕ(∣∣x–c∣∣)\phi(x,c)=\phi(||x–c||)ϕ(x,c)=ϕ(∣∣x–c∣∣) 。
径向基函数用于度量某点到指定中心点c的距离，用距离来表示相似性。
取值一般和距离呈反比：当 xxx 远离 ccc 时函数取值很小；当 xxx 靠近 ccc 时，取值变大，来保证只有中心区域被少部分激活。
