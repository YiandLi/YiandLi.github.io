概率密度函数

probability density function，通常用小写pdf表示。

  • 连续型随机变量的概率密度函数：描述这个随机变量的输出值，在某个确定的取值点附近的可能性的函数。
  • 点 (x,y)(x,y)(x,y) 的意义为取值x时的概率为y。
  • 随机变量的取值落在某个区域之内的概率则为概率密度函数在这个区域上的积分。
    用累积分布函数(Cumulative Distribution Function)表示，通常取大写 CDF。
    • 累积分布函数用来表示概率分布的累积分布

    FX(x)=P(X<x)F_X(x) = P(X \lt x) FX​(x)=P(X<x)

Hard K-means

  • 要选择代表点，因为如果某些样本点很接近，那么就可以用一个中心点来代表它们。
    这就是 聚类（cluster） 的思想，从所有N个样本点中选择少数几个代表作为中心点。

  • 聚类（clustering）问题是一种典型的非监督式学习（unsupervised learning）。
    它的优化问题有两个变量需要确定：

    • 一个是分类的分群值 SmS_mSm​ ，每一类可表示为 S1,S2,⋯,SmS_1,S_2,⋯,S_mS1​,S2​,⋯,Sm​ ；
    • 另外一个是每一类对应的中心点 μ1,μ2,⋯,μmμ_1,μ_2,⋯,μ_mμ1​,μ2​,⋯,μm​ 。

  • 找到最优聚类是一个NP-hard问题。

损失函数

Ein(S1,⋯ ,SM;μ1,⋯ ,μM)=1N∑n=1N∑m=1M⟦xn∈Sm⟧∥xn−μm∥2E_{i n}\left(S_{1}, \cdots, S_{M} ; \mu_{1}, \cdots, \mu_{M}\right)=\frac{1}{N} \sum_{n=1}^{N} \sum_{m=1}^{M} \llbracket \mathbf{x}_{n} \in S_{m} \rrbracket\left\|\mathbf{x}_{n}-\mu_{m}\right\|^{2} Ein​(S1​,⋯,SM​;μ1​,⋯,μM​)=N1​n=1∑N​m=1∑M​[[xn​∈Sm​]]∥xn​−μm​∥2

能够发现这是一个组合最佳化的问题，既要优化分群值 SmS_mSm​，又要求解每一类的中心点 umu_mum​。

优化方式

对 SSS 和 μμμ 分别进行最优化求解：
首先，如果中心点 μ1,μ2,⋯,μmμ_1,μ_2,⋯,μ_mμ1​,μ2​,⋯,μm​ 是固定的，因为一个点只能属于一个群，所以点 xnx_nxn​ 一定属于距离自己最近的样本点的群。
然后，如果 S1,S2,⋯,SMS_1,S_2,⋯,S_MS1​,S2​,⋯,SM​ 是固定的，目标就是只要找出每个类的中心点 μμμ 。使用梯度下降算法，得到中心点 μmμ_mμm​ 就等于所有属于类群 SmS_mSm​ 的平均位置处：

μm=1Nm∑n∈Smxnμ_m = \frac{1}{N_m} \sum_{n \in S_m}x_n μm​=Nm​1​n∈Sm​∑​xn​

NmN_mNm​ 为该聚类内点的个数
SmS_mSm​ 为某一个聚类
xnx_nxn​ 为聚类内的点

经过以上的推导，我们得到了一个非常有名的一种unsupervised learning算法，叫做k-Means Algorithm。这里的k就是代表中心点个数和类群的个数，即上面的M。

流程

  1. 首先，随机选择 kkk 个中心点 μ1,μ2,⋯,μkμ_1,μ_2,⋯,μ_kμ1​,μ2​,⋯,μk​；
  2. 然后，再由确定的中心点得到不同的类群 S1,S2,⋯,SkS_1,S_2,⋯,S_kS1​,S2​,⋯,Sk​ （Assignment step）；
  3. 接着，再由确定的类群计算出新的不同的k个中心点 （Refitting step）；
  4. 继续循环迭代计算，交互地对μ和S值进行最优化计算，不断更新μ和S值，直到程序收敛，实现 EinE_{in}Ein​ 最小化。

这将同时考虑多个参数来最小化损失转化为分别考虑参数最小化损失的方式叫做 交替最小化 Alternating minimization 。

讨论

  • K-means 算法的弊端：
    • 需要事先指定聚类数量
    • 不适合非凸形状的聚类
    • 对噪声和偏离对象敏感
  • K-means 提升方法：
    • Repeated K-means，多试几次择优
    • 更好初始化，使用一些启发式方法来分配向量起始分布

GMM

混合高斯模型

对于单高斯模型，由于可以明确训练样本是否属于该高斯模型：
  对于任意输入x，使用 p(x∣z=k)=N(x∣μk,Σ)p(x|z=k)=\mathcal N(x|\mu_k,\Sigma)p(x∣z=k)=N(x∣μk​,Σ) 表示该样本为正样本/负样本的概率。然后根据提前设置的阈值来进行分类。

混合高斯模型（Gaussian Mixture Model）假设每个聚类分布都是符合高斯分布的，当前数据分布为多个高斯分布的叠加。
通过使⽤⾜够多的⾼斯分布，并且调节它们的均值和⽅差以及线性组合的系数，⼏乎所有的连续概率密度都能够以任意的精度近似。

竞争学习

Lecture 14 竞争学习1

生物学中，神经元相互存在竞争关系。

竞争学习 Competitive Learning：
属于无监督学习。
在 Hebbian Learning 中，所有的神经元都可以被激活。
但是在竞争学习中，每次训练时，每一组/层只有一个神经元会激活。
输出单元和其他单元竞争 competitve，最出色的一个我们称之为赢家 Winner-takes-all（WTA）unit，赢家通吃。

无监督竞争学习

  • 层与层之间神经元相互刺激 excitatory
  • 层内神经元相互阻碍的 inhibitory
    • 层内会有多个神经元簇 cluster，聚类内部神经元相互抑制
    • 在每个神经元簇中，只有一个神经元可以被激活

竞争学习中的聚类问题

  1. 随机初始化K个原型聚类中心（prototype vector
  2. 随机找到一个样本点（a single example
  3. 找到距离这个样本点最近的聚类中心（closest prototype），就是我们所说的赢家。
  4. 使这个聚类中心（赢家）靠近样本点

更新权重规则

在聚类问题中，权重 www 可以理解为聚类点的向量（我们使用点积表示相似度时，点积就是相乘，所以也可以理解为权重）。

hjh_jhj​ 表示样本点和聚类中心j的相似度。

加强竞争公平性

  • 初始化的聚类中心建议在样本点的区域内部 (in region with few patterns
  • 一些聚类中心/单位永远不可能成为赢家，进而成为了死亡神经元（dead unit
  • 为了更有效地保证竞争公平性，每个单位应该有平等的机会，最终都能代表训练数据。

上图反映了侧抑制机制，具体为同层不同神经元相互影响，表现为权重改变，可以看成一个横截面去理解。
输出层神经元之间相互抑制，每个神经元都倾向于抑制跟他同一层的其他神经元，即其他神经元的权值会减小。
通过抑制作用，Maxnet经过若干次迭代，使除了最大输入外的所有输入值变为负值，并使得对应输出为0。最终剩下的一个输出节点恰好是输入最大的节点。

Mexican Hat Network

根据到赢家神经元的距离，侧连接会产生不同的积极作用(excitatory effect)或者抑制作用(inhibitory effect)。
墨西哥草帽图模拟了输出层神经元之间的突触权值的分布。

  • close neighbours: cooperative (mutually excitatory)
  • distant neighbours: competitive (mutually inhibitory)
  • too far away neighbours: irrelevant

获胜神经元对其邻近神经元的影响是由近及远，由兴奋逐渐转变为抑制，因此其学习算法中不仅获胜神经元本身要调整权向量，它周围的神经元在其影响下也要程度不同地调整权向量。

自组织映射(Self-organizing map, SOM)

生物动机

Biological Motivation
功能相近的神经元分布在一起。
人脑在处理信息时，会将外部的 高维 信号(external multidimensional representation)映射到脑内的二维空间(2-dimensional internal representation)。
即 大脑的自组织机制self-organization ：大脑会生成线性或者平面拓扑的特征 linear or planar topology。

网络特征

  • 无监督学习，竞争学习
  • 通过学习输入空间中的数据，生成一个 低维、离散的映射(Map) ，是一种矢量量化算法，从某种程度上也可看成一种降维算法。
  • 融合了竞争机制和协作机制 (incorporation of competitive and cooperative mechanism)。
  • 使用 近邻关系函数(neighborhood function) 来维持输入空间的拓扑结构：二维映射包含了数据点之间的相对距离。
    输入空间中相邻的样本会被映射到相邻的输出神经元。
    输出空间维持了输入空间的近邻关系，输出空间中近邻的神经元表示了相似的输入。

如果训练集大小为N，对于每棵树而言，随机且有放回地从训练集中的抽取N个训练样本（bootstrap sample），作为该树的训练集。
每棵树的训练集都是不同的，而且里面包含重复的训练样本，不容易过拟合。

如果样本量无限大，则某个样本不会被抽到的概率为 (1−1n)n(1-\frac 1n)^n(1−n1​)n ，这个值的极限是e，为0.36。
所以整个样本约有36.8%的数据不会被抽到，这部分数据被称为外包数据(Out of Bag, OOB)。同样地，每一个样本不会参与36.8%的树的训练。

训练树

设每个样本有M个属性，在决策树的每个节点需要分裂时，随机从这M个属性中选取出m个属性，满足条件 m << M。
然后从这m个属性中采用某种策略（比如说信息增益）来选择1个最优属性作为该节点的分裂属性，直到无法继续分裂。
即每棵树都尽最大程度的生长，没有剪枝过程。

随机性

  1. 随机采样，选择训练集。
  2. 随机选择属性

参数

决策树的数量 和 抽取的特征数量

  • 随机森林分类效果（错误率）与两个因素有关：
    • 森林中任意两棵树的相关性：相关性越大，错误率越大；
    • 森林中每棵树的分类能力：每棵树的分类能力越强，整个森林的错误率越低。
  • 减小特征选择个数m，树的相关性和分类能力也会相应的降低；增大m，两者也会随之增大。
    所以关键问题是如何选择最优的m（或者是范围）

包外误差

训练每棵树时，都有一部分样本没有参加训练，这部分样本可以作为测试集，统计预测误差，称为包外误差。
和交叉验证类似，这部分验证的误差可以作为泛化误差的估计。

简化流程

根据实验，随着决策树的数量增加，包外误差和测试误差都会下降。
1）对每个样本，计算它作为oob样本的树对它的分类情况（约1/3的树）；
2）然后以简单多数投票作为该样本的分类结果；
3）最后用误分个数占样本总数的比率作为随机森林的oob误分率。

这个包外误差可以用来确定决策树的数量，即通过观察误差何时停止下降来决定何时终止训练。

特征重要性

随机森林可以在训练过程中计算特征的重要性，即置换法。
原理：如果某个特征对分类很重要，那么对于某个样本，改变这个特征值之后，样本的预测结果就会出现错误。
即分类结果对这个特征值很敏感。反之，如果某特征值对分类不重要，则随便改变也不会影响分类结果。

对于每棵决策树，使用相应的OOB(袋外数据)数据来计算它的袋外数据误差,记为 err1err_1err1​ 。
随机地对袋外数据OOB所有样本的特征X加入噪声干扰(就可以随机的改变样本在特征X处的值)，再次计算它的袋外数据误差,记为 err2err_2err2​ 。

一些总结

  • 随机森林属于Bagging算法。
  • 超参数为：
    • 训练单棵树的特征数量
    • 决策树的数量
    • 每棵决策树的设置，如最少的叶节点数量
  • 当数据总特征很多，但是有效特征（relevant predictors）比较少的时候，随机森林效果不好。
    因为每棵树的都会弱化这些有效特征，导致每棵树都是弱分类器。
  • 增加树的数量不会降低方差，即不会导致过拟合。
    但是如果树的数量太多，会导致树木相关性增强，从而增大方差。
  • 融合模型的鲁棒性不会低于单棵决策树。

Boosting

  • 集成学习大致可分为两大类：Bagging和Boosting

    • Bagging一般使用强学习器，其个体学习器之间不存在强依赖关系，容易并行。
    • Boosting则使用弱学习器，其个体学习器之间存在强依赖关系，是一种序列化方法。
    • Bagging主要关注降低方差，而Boosting主要关注降低偏差。

  • Boosting是一族算法，其主要目标为将弱学习器“提升”为强学习器，大部分Boosting算法都是根据前一系列弱学习器的不足(shortcoming)对样本分布进行调整，再根据新的样本分布训练下一弱学习器，如此迭代M次，最后将一系列弱学习器组合成一个强学习器。

  • boosting算法中，损失函数改变，给不同的样本赋予不同的权重：
