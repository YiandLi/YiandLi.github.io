Binning/Discretization 数据分箱/离散化

  • 将多个连续值分组为较少数量的“分箱”。
  • 分箱的数据不一定必须是数字，它们可以是任何类型的值，如“狗”，“猫”，“仓鼠”等。 分箱也用于图像处理，通过将相邻像素组合成单个像素，它可用于减少数据量。
  • 优点：
    • 对连续变量离散化，特征离散化后，模型会更稳定，降低了模型过拟合的风险。
    • 对异常数据有很强的鲁棒性
    • 逻辑回归中，单变量离散化为N个后，每个变量有单独的权重，相当于为模型引入了非线性，能够提升模型表达能力，加大拟合。
    • 模型更加稳定，降低单变量的轻微扰动造成的偏差。
  • 方法：
    • 有监督分箱：
      • 卡方分箱
      • 最小熵法分箱
    • 无监督分箱：
      • 等距分箱
      • 等频率分箱
      • 使用决策树分箱

等距分箱 Equal Width Discretization

从最小值到最大值之间，均分为 N 等份，只考虑边界，不考虑区域内样本数量。
如果样本的分布有偏差/不准确，则方法无效。

pd.cut(x, bins, right=True, labels=None, retbins=False, precision=3, include_lowest=False, duplicates='raise')

  • x ：需要被分箱的一维数组
  • bins ：整数，标量序列或者间隔索引，是进行分组的依据
    • 整数：则表示将x中的数值分成等宽整数份
    • 标量序列：序列中的数值表示用来分档的分界值，eg.[1、2、3、4] 。
    • 间隔索引：间隔索引不能重叠
  • right ：默认情况下， 每个分组的左边为开区间，右边为闭区间。
    把参数right设置成False，左右的开闭区间便互相调换了位，为左闭右开。
  • labels ：数组或布尔值，控制分箱之后数组中的值。
    • None默认情况，数组中的值为区间形式。
    • False：返回分箱的整数指示符，即x中的数据在第几个箱子里。
    • 数组：长度要与分箱个数一致，表示分箱区域的值。

等频率分箱 Equal Frequency Discretization

官方文档
使用频率来选择边界，使得每个区间包含大致相等的实例数量。
通常使用百分位数，或者是四分间隔。

代码

pd.qcut(x, q, labels=None, retbins=False, precision=3, duplicates='raise')

  • q : int or list-like of float ：进行分组的依据
    10表示十分位，4表示四分位。
    [0, 0.5, 1]表示二分，[0, 0.25, 0.5, 0.75, 1]表示四分。

决策树分箱 Discretization with decision tree

按照决策树的叶节点给观测值分类，不保证区间中实例数量相等。
可以提前得到对一些变量的预测能力。

noise

noise 本真噪音，也称Bayes error，表示样本的固有的不可预测性，是任何学习算法在该学习目标上的期望误差的下界，刻画了学习问题本身的难度。
和预测y无关，不是模型能过决定的。

bias

bias 偏差，刻画了学习算法本身的拟合能力。偏差越大，模型越欠拟。
（独立于训练样本的误差，刻画了匹配的准确性和质量：一个高的偏置意味着一个坏的匹配）

variance

variance 方差，度量了同样大小的训练集的变动所导致的学习性能的变化，刻画了数据扰动所造成的影响。即加入扰动后，模型还能否精准预测。模型越过拟合，方差越大。

Bagging：Bootstrap aggregating

Bootstrap

由原始数据经过再抽样所获得的与原始数据集含量相等的样本称为再抽样样本(resamples)或自助样本(bootstrapsamples)。
如果将由原始数据集计算所得的统计量称为观察统计量(observed statistic)的话，那么由再抽样样本计算所得的统计量称为自助统计量(bootstrap statistic)。
核心思想：自助统计量：：观察统计量<=>观察统计量：：真值 ： 自助统计量和观察统计量之间的关系等价于观察统计量和真值之间的关系。所以可以通过在训练集合中采样来估计总体数据集的分布。

再抽样：有返还的抽样(sampling with replacement)。

重采样的小样本计算出的统计量近似于正态分布。

Hard K-means

  • 要选择代表点，因为如果某些样本点很接近，那么就可以用一个中心点来代表它们。
    这就是 聚类（cluster） 的思想，从所有N个样本点中选择少数几个代表作为中心点。

  • 聚类（clustering）问题是一种典型的非监督式学习（unsupervised learning）。
    它的优化问题有两个变量需要确定：

    • 一个是分类的分群值 SmS_mSm​ ，每一类可表示为 S1,S2,⋯,SmS_1,S_2,⋯,S_mS1​,S2​,⋯,Sm​ ；
    • 另外一个是每一类对应的中心点 μ1,μ2,⋯,μmμ_1,μ_2,⋯,μ_mμ1​,μ2​,⋯,μm​ 。

  • 找到最优聚类是一个NP-hard问题。

损失函数

能够发现这是一个组合最佳化的问题，既要优化分群值 SmS_mSm​，又要求解每一类的中心点 umu_mum​。

优化方式

对 SSS 和 μμμ 分别进行最优化求解：
首先，如果中心点 μ1,μ2,⋯,μmμ_1,μ_2,⋯,μ_mμ1​,μ2​,⋯,μm​ 是固定的，因为一个点只能属于一个群，所以点 xnx_nxn​ 一定属于距离自己最近的样本点的群。
然后，如果 S1,S2,⋯,SMS_1,S_2,⋯,S_MS1​,S2​,⋯,SM​ 是固定的，目标就是只要找出每个类的中心点 μμμ 。使用梯度下降算法，得到中心点 μmμ_mμm​ 就等于所有属于类群 SmS_mSm​ 的平均位置处：

μm=1Nm∑n∈Smxnμ_m = \frac{1}{N_m} \sum_{n \in S_m}x_n μm​=Nm​1​n∈Sm​∑​xn​

NmN_mNm​ 为该聚类内点的个数
SmS_mSm​ 为某一个聚类
xnx_nxn​ 为聚类内的点

经过以上的推导，我们得到了一个非常有名的一种unsupervised learning算法，叫做k-Means Algorithm。这里的k就是代表中心点个数和类群的个数，即上面的M。

流程

  1. 首先，随机选择 kkk 个中心点 μ1,μ2,⋯,μkμ_1,μ_2,⋯,μ_kμ1​,μ2​,⋯,μk​；
  2. 然后，再由确定的中心点得到不同的类群 S1,S2,⋯,SkS_1,S_2,⋯,S_kS1​,S2​,⋯,Sk​ （Assignment step）；
  3. 接着，再由确定的类群计算出新的不同的k个中心点 （Refitting step）；
  4. 继续循环迭代计算，交互地对μ和S值进行最优化计算，不断更新μ和S值，直到程序收敛，实现 EinE_{in}Ein​ 最小化。

这将同时考虑多个参数来最小化损失转化为分别考虑参数最小化损失的方式叫做 交替最小化 Alternating minimization 。

讨论

  • K-means 算法的弊端：
    • 需要事先指定聚类数量
    • 不适合非凸形状的聚类
    • 对噪声和偏离对象敏感
  • K-means 提升方法：
    • Repeated K-means，多试几次择优
    • 更好初始化，使用一些启发式方法来分配向量起始分布

Lecture 14 竞争学习1

生物学中，神经元相互存在竞争关系。

竞争学习 Competitive Learning：
属于无监督学习。
在 Hebbian Learning 中，所有的神经元都可以被激活。
但是在竞争学习中，每次训练时，每一组/层只有一个神经元会激活。
输出单元和其他单元竞争 competitve，最出色的一个我们称之为赢家 Winner-takes-all（WTA）unit，赢家通吃。

无监督竞争学习

  • 层与层之间神经元相互刺激 excitatory
  • 层内神经元相互阻碍的 inhibitory
    • 层内会有多个神经元簇 cluster，聚类内部神经元相互抑制
    • 在每个神经元簇中，只有一个神经元可以被激活

竞争学习中的聚类问题

  1. 随机初始化K个原型聚类中心（prototype vector
  2. 随机找到一个样本点（a single example
  3. 找到距离这个样本点最近的聚类中心（closest prototype），就是我们所说的赢家。
  4. 使这个聚类中心（赢家）靠近样本点

Hebbian Learning

Hebbian提出：突触前神经元向突触后神经元的持续重复的刺激可以导致 突触传递效能(synaptic weight) 的增加，或者导致突触后神经元的 阀值(threshold) 减小。
反映到神经网络就是，当输入和输出相同时，该权重就会增加，反之不同则权重不变或者下降：
