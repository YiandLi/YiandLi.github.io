论文：When in Doubt: Improving Classification Performance with Alternating Normalization

使用一种无参数的（但是有超参数）后处理方法 (a non-parametric post-processing) 应用于分类问题。
具体地，在验证集上，使用那些高置信度样本的预测概率调整低置信度样本的预测概率。

步骤

  1. 针对样本预测概率输出，用一个指标来划分 （简单样本）高置信度样本 和 （难样本）低置信度样本 。
    use ambiguity level to identify high-confidence validation examples and challenging examples
  2. 迭代进行 交替归一化 alternating normalization
    每一次迭代过程中先进行 列归一化/行间归一化，然后进行 行归一化/行内归一化。
    normalizing across rows (examples) and columns (target labels)
  3. 输出：上一步迭代过程中，高置信样本和低置信样本的概率分布都会改变。我们只保留最终的低置信度样本的概率分布，作为输出；高置信样本的概率分布还是迭代前的概率分布。

Identify Hard Examples

一共有 mmm 个类别
最简单的划分指标肯定是信息熵:

H(b0)=−∑ib0,ilog⁡m(b0,i)\mathcal{H}\left(\mathbf{b}_{0}\right)=-\sum_{i} \mathbf{b}_{0, i} \log _{m}\left(\mathbf{b}_{0, i}\right) H(b0​)=−i∑​b0,i​logm​(b0,i​)

概率分布的信息熵越大，置信度越低。但是有时，信息熵并不符合我们分类的目标，比如两个样本:

b01=[0.5  0.25  0.25]b02=[0.5  0.5  0]H(b01)  >  H(b02)\begin{aligned} b^1_0 &= [0.5\; 0.25\; 0.25]\\ b^2_0 &= [0.5\; 0.5\; 0]\\ \mathcal{H}\left(b^1_0\right)\; &\gt\; \mathcal{H}\left(b^2_0\right) \end{aligned} b01​b02​H(b01​)​=[0.50.250.25]=[0.50.50]>H(b02​)​

可以看到 b01b^1_0b01​ 的信息熵大于 b02b^2_0b02​ 的信息熵，所以按理说 b01b^1_0b01​ 的置信度会小于 b02b^2_0b02​ ，但是对于分类问题，明显是 b01b^1_0b01​ 更容易做出预测。

One Iteration of AN

每次都是使用全部的高置信样本去纠正一个低置信样本的概率分布。
首先行间标准化（row Normalization），然后行内标准化（column Normalization）。两者的目标是冲突的，所以可以不断迭代进行单独归一。

  • 定义样本表示：
    类别数量为 mmm
    高置信样本数量为 nnn ，拼接为 A0∈Rn×mA_0\in \mathbb{R}^{n\times m}A0​∈Rn×m

实验

  1. 低置信样本的不确定性越高/信息熵越大，并且用于纠正的高置信样本的确定性越高/信息熵越低，最终的提升效果越好。
  2. 对高置信样本 A0A_0A0​ 具有良好的鲁棒性。
  3. 分类任务的类别数量越多，提升越明显。
  4. 超参数 (τ,α,d)(τ,\alpha,d)(τ,α,d) ，一般 d=1d=1d=1 效果最好，实验中 α>10\alpha>10α>10 ， τ=0.75τ=0.75τ=0.75 效果最好（但是论文的归一结果明显小于1，就不一样了呢）。
    基本 alphaalphaalpha 越大，效果越好。并且随着 α\alphaα 的增加，τττ 和 ddd 的影响越来越小。

苏神博客中指出 α\alphaα 纯粹是为了调参而引入的（参数多了，总能把效果调到有所提升），没有太多实际意义。同时， α=1\alpha=1α=1 时，基本已经是最优选择了。

Hopfield Network

Posted on 2022-01-02 In Machine Learning , Competitive Learning
Symbols count in article: 822 Reading time ≈ 4 mins.

Hopfield Network (霍普菲尔德网络)，是 Hopfield 在1982年提出的一种基于能量的模型。

  • 用于Associated Memories和Combinatorial Optimization。
    • 离散Hopfield网络主要用于联想记忆。
    • 连续Hopfield网络主要用于优化计算功能。
  • 能量模型：可以看作一种代价函数。这个概念最先在热力学中被提出，用来描述系统的能量值。当能量函数求得的能量值达到最小值的时候，整个热力学系统达到稳定状态。在深度学习之中，引入这个概念也是为了使模型达到稳定的状态。
  • 流程：
    • 首先利用一个完整的pattern来训练网络得到所有的权值，这样这个网络本身就是我们的output。
    • 以后当我们只有pattern的一部分数据的时候，我们可以把这一部分数据输入到网络中，通过达到最小的 E(𝑥)E(𝑥)E(x) 来复原完整的pattern。

网络结构

有连续型模型和离散型模型，这里介绍后者。

  • 全连接
  • 单层
  • 权重对称：

wij=wjiwii=0\begin{aligned} w_{ij}&=w_{ji}\\ w_{ii}&=0 \end{aligned} wij​wii​​=wji​=0​

  • 每一个神经元既是输入节点，也是输出节点。在离散模型中，输入值和输出值只能为1或-1。
  • x为外部输入，可以是短暂的transient或者是永久的permanent。

Associated Memory network

  • 有两种联想记忆的方式：
    • 自联想 (Auto-Association)
      返回一个和当前样本接近的(resemble)，以前存储的样本。
      通过输入模式的一部分信息/被扰动的表示即可恢复模式的全部信息
    • 异联想 (Hetero-Association)
      和输入样本内容(content)不一样，甚至连形式(format)都不一样。
      即网络存贮了两个模式对之间的对应关系。异联想记忆实际上确定了输入模式与输出模式之间的对应关系。

Binning/Discretization 数据分箱/离散化

  • 将多个连续值分组为较少数量的“分箱”。
  • 分箱的数据不一定必须是数字，它们可以是任何类型的值，如“狗”，“猫”，“仓鼠”等。 分箱也用于图像处理，通过将相邻像素组合成单个像素，它可用于减少数据量。
  • 优点：
    • 对连续变量离散化，特征离散化后，模型会更稳定，降低了模型过拟合的风险。
    • 对异常数据有很强的鲁棒性
    • 逻辑回归中，单变量离散化为N个后，每个变量有单独的权重，相当于为模型引入了非线性，能够提升模型表达能力，加大拟合。
    • 模型更加稳定，降低单变量的轻微扰动造成的偏差。
  • 方法：
    • 有监督分箱：
      • 卡方分箱
      • 最小熵法分箱
    • 无监督分箱：
      • 等距分箱
      • 等频率分箱
      • 使用决策树分箱

等距分箱 Equal Width Discretization

从最小值到最大值之间，均分为 N 等份，只考虑边界，不考虑区域内样本数量。
如果样本的分布有偏差/不准确，则方法无效。

  • x ：需要被分箱的一维数组
  • bins ：整数，标量序列或者间隔索引，是进行分组的依据
    • 整数：则表示将x中的数值分成等宽整数份
    • 标量序列：序列中的数值表示用来分档的分界值，eg.[1、2、3、4] 。
    • 间隔索引：间隔索引不能重叠

    • False：返回分箱的整数指示符，即x中的数据在第几个箱子里。
    • 数组：长度要与分箱个数一致，表示分箱区域的值。

给定学习目标和训练集规模，它可以把一种学习算法的期望误差分解为三个非负项的和，即noise、bias和 variance。

noise

noise 本真噪音，也称Bayes error，表示样本的固有的不可预测性，是任何学习算法在该学习目标上的期望误差的下界，刻画了学习问题本身的难度。
和预测y无关，不是模型能过决定的。

bias

bias 偏差，刻画了学习算法本身的拟合能力。偏差越大，模型越欠拟。
（独立于训练样本的误差，刻画了匹配的准确性和质量：一个高的偏置意味着一个坏的匹配）

variance

variance 方差，度量了同样大小的训练集的变动所导致的学习性能的变化，刻画了数据扰动所造成的影响。即加入扰动后，模型还能否精准预测。模型越过拟合，方差越大。

Bagging：Bootstrap aggregating

Bootstrap

由原始数据经过再抽样所获得的与原始数据集含量相等的样本称为再抽样样本(resamples)或自助样本(bootstrapsamples)。
如果将由原始数据集计算所得的统计量称为观察统计量(observed statistic)的话，那么由再抽样样本计算所得的统计量称为自助统计量(bootstrap statistic)。
核心思想：自助统计量：：观察统计量<=>观察统计量：：真值 ： 自助统计量和观察统计量之间的关系等价于观察统计量和真值之间的关系。所以可以通过在训练集合中采样来估计总体数据集的分布。

再抽样：有返还的抽样(sampling with replacement)。

重采样的小样本计算出的统计量近似于正态分布。

算法

使用bootstrap采样m个数据集（有放回重采样，和原本数据集大小相同），然后用于训练每一个模型，最后平均这些模型的预测结果。

Bayes error 和 Bias 不会改变。

概率密度函数

probability density function，通常用小写pdf表示。

  • 连续型随机变量的概率密度函数：描述这个随机变量的输出值，在某个确定的取值点附近的可能性的函数。
  • 点 (x,y)(x,y)(x,y) 的意义为取值x时的概率为y。
  • 随机变量的取值落在某个区域之内的概率则为概率密度函数在这个区域上的积分。
    用 累积分布函数(Cumulative Distribution Function) 表示，通常取大写 CDF。
