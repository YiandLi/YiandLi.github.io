大模型小模型碎碎念

Posted on 2024-04-20 In NLP , LLM Model
Symbols count in article: 394 Reading time ≈ 3 mins.

Loss 才是涌现的关键

https://arxiv.org/abs/2403.15796

Pre-training Loss vs 性能 ：训练了三个参数为1.5B、6B和32B的模型：

- 随着预训练损失降低，任务性能会提高。但是三种大小的模型在预训练损失降至约2.2之前都表现为随机水平，之后随着损失的减少，性能逐渐提高，也就是所谓的涌现能力。

- 模型在下游任务上的性能在很大程度上与预训练损失相关，而与模型大小无关。

> 在未过拟合前提下，如果用 loss 来衡量模型的能力，loss 相近的模型，我把它理解成智力水平差不多，在常规的任务下表现差不多；更感兴趣的是，不通参数规模达到相近的 loss，需要多少 tokens。

Tokens vs 性能 ：用不同的tokens 量分别训练了 28 个相对较小的模型。

- 无论 tokens 数量和模型大小如何，具有相同 pre-training loss 的模型在 12 个下游任务上表现出相同的性能。

是不是有一种情况：在相同的 loss 下，小模型已经拟合了，换言之增加 token 数量没办法继续提高，但是大模型却可以通过增加 token 量继续让效果变好？

涌现 vs 连续性指标 ：

- 不管是非连续指标（Accuracy），还是连续指标（CorrectChoiceProb），当 pre-training loss 达到某个转折点时，语言模型的涌现能力就会出现。连续指标无法消除观察到的转折点。
