  • The os module represents paths as strings with which you cannot do much.
  • The pathlib module represents paths as special objects <class 'pathlib.PosixPath'> with useful methods and attributes.

完全切分

在一段文本中，找到字典匹配的所有单词，也称分词。
完全切分会找到所有的单词（“北京大学”会被分为“北京大学”，“北”，“北京”，“大”，“学”，“大学”）。
而我们只希望得到一整个单词，不需要这么零碎。所以，考虑越长优先级越高。

匹配方式

分类：正向最长匹配，逆向最长匹配，双向最长匹配

双向最长匹配

正向匹配，和逆向匹配都会出现切分的不到位的情况。取长补短：

  1. 同时执行两种匹配方式，词数不同则返回词更少的一个。
  2. 词数相同则返回单字更少的一个。
  3. 单字数量相同则返回逆向匹配的结果。

Value based

Deep Q network

最优动作价值函数：

Q⋆(st,at)=max⁡πE[Ut∣St=st,At=at]Q^{\star}\left(s_t, a_t\right)=\max _\pi \mathbb{E}\left[U_t \mid S_t=s_t, A_t=a_t\right] Q⋆(st​,at​)=πmax​E[Ut​∣St​=st​,At​=at​]

和 π\piπ 没有关系了，可以根据当前状态 sts_tst​ 对所有动作打分，分数就能反应每个动作的价值上限。

但是我们无法直接定义，价值学习的目标就是直接学习一个函数去模拟动作价值函数。

使用一个神经网络 Q(s,a;w)Q(s,a;w)Q(s,a;w) 来近似 Q∗(s,a)Q^*(s,a)Q∗(s,a) 函数，得到动作 a 的期望价值。

对于离散动作，直接根据最大期望价值选择动作即可。

Temporal Difference(TD) Learning

Ut=Rt+γUt+1U_t = R_t+\gamma U_{t+1}Ut​=Rt​+γUt+1​

然后用 DQN 去近似两个奖励值：

  • Q(st,at;w)Q\left(s_t, a_t ; \mathbf{w}\right)Q(st​,at​;w) is estimate of E[Ut]\mathbb{E}\left[U_t\right]E[Ut​].
  • Q(st+1,at+1;w)Q\left(s_{t+1}, a_{t+1} ; \mathbf{w}\right)Q(st+1​,at+1​;w), is estimate of E[Ut+1]\mathbb{E}\left[U_{t+1}\right]E[Ut+1​].

于是得到 Bellman equation:

Q(st,at;w)≈rt+γ⋅Q(st+1,at+1;w)Q\left(s_t, a_t ; \mathbf{w}\right) \approx r_t+\gamma \cdot Q\left(s_{t+1}, a_{t+1} ; \mathbf{w}\right)Q(st​,at​;w)≈rt​+γ⋅Q(st+1​,at+1​;w)

在还没有进行完游戏时，

  • rtr_trt​ 表示进行一半时，观测到的真实奖励。
  • $Q\left(s_t, a_t ; \mathbf{w}\right) $ 表示开始之前的预估分数。
  • Q(st+1,at+1;w)Q\left(s_{t+1}, a_{t+1} ; \mathbf{w}\right)Q(st+1​,at+1​;w) 表示未来的预估分数。

左边为 Prediction， 右边为 TD target ，可以写为：

yt=rt+γ⋅Q(st+1,at+1;wt)=rt+γ⋅max⁡aQ(st+1,a;wt)\begin{aligned} y_t & =r_t+\gamma \cdot Q\left(s_{t+1}, a_{t+1} ; \mathbf{w}_t\right) \\ & =r_t+\gamma \cdot \max _a Q\left(s_{t+1}, a ; \mathbf{w}_t\right) \end{aligned} yt​​=rt​+γ⋅Q(st+1​,at+1​;wt​)=rt​+γ⋅amax​Q(st+1​,a;wt​)​

即根据 t+1 时刻的状态，选择分数最大的动作得到分数（即max）。

损失函数为：

Lt=12[Q(st,at;w)−yt]2L_t = \frac 12 \left [ Q(s_t, a_t;\mathbf{w}) -y_t\right ]^2 Lt​=21​[Q(st​,at​;w)−yt​]2

总流程

Algorithm: One iteration of TD learning.

  1. Observe state St=stS_t=s_tSt​=st​ and action At=atA_t=a_tAt​=at​.
  2. Predict the value: qt=Q(st,at;wt)q_t=Q\left(s_t, a_t ; \mathbf{w}_t\right)qt​=Q(st​,at​;wt​).
  3. Differentiate the value network: dt=∂Q(st,at;w)∂w∣w=wt\mathbf{d}_t=\left.\frac{\partial Q\left(s_t, a_t ; \mathbf{w}\right)}{\partial \mathbf{w}}\right|_{\mathbf{w}=\mathbf{w}_t}dt​=∂w∂Q(st​,at​;w)​∣∣∣∣​w=wt​​.
  4. Environment provides new state st+1s_{t+1}st+1​ and reward rtr_trt​.
  5. Compute TD target: yt=rt+γ⋅max⁡aQ(st+1,a;wt)y_t=r_t+\gamma \cdot \max _a Q\left(s_{t+1}, a ; \mathbf{w}_t\right)yt​=rt​+γ⋅maxa​Q(st+1​,a;wt​).
  6. Gradient descent: wt+1=wt−α⋅(qt−yt)⋅dt\mathbf{w}_{t+1}=\mathbf{w}_t-\alpha \cdot\left(q_t-y_t\right) \cdot \mathbf{d}_twt+1​=wt​−α⋅(qt​−yt​)⋅dt​.

这里是将损失进行求导，拆开分别计算每一项的。

Policy based

策略函数 π(a∣s)\pi(a|s)π(a∣s) 是一个条件的概率密度函数，输出是一个概率分布，给每一个动作一个概率值。

每一个状态对应了n个动作的概率值，但是状态是无限多的，没有办法统计出来，所以还是使用一个函数来近似策略函数。

这里使用一个神经网络来近似，该神经网络被称为 Policy Network π(a∣s;θ)\pi(a|s;\theta)π(a∣s;θ) ， θ\thetaθ 为参数。

状态价值函数：

Vπ(s)=EA[Qπ(s,A)]=∑aπ(a∣st)⋅Qπ(st,a)V_\pi(s)=\mathbb{E}_A\left[Q_\pi(s, A)\right] = \sum_a \pi \left(a \mid s_t\right) \cdot Q_\pi\left(s_t, a\right) Vπ​(s)=EA​[Qπ​(s,A)]=a∑​π(a∣st​)⋅Qπ​(st​,a)

可以用来评价策略函数的好坏：对于一个状态，策略函数越好，状态价值越高。

使用上面的 policy network 替代策略函数，则得到：

Vπ(s;θ)=∑aπ(a∣s;θ)⋅Qπ(s,a)V_\pi(s; \theta) = \sum_a \pi \left(a \mid s;\theta \right) \cdot Q_\pi\left(s, a\right) Vπ​(s;θ)=a∑​π(a∣s;θ)⋅Qπ​(s,a)

这个式子自变量为 状态 和 网络参数 ，然后求对变量状态 SSS 的期望

J(θ)=ES[V(S;θ)]J(\boldsymbol{\theta})=\mathbb{E}_S[V(S ; \boldsymbol{\theta})] J(θ)=ES​[V(S;θ)]

把状态变成一个随机变量，给去掉，这样就只剩下一个网络参数 θ\thetaθ 了。

这个公式可以用来评价策略函数，所以目标就是改进 θ\thetaθ ，让这个期望越大越好。

过程

给定一个观测的状态 s ，我们需要求 Vπ(s,θ)V_\pi(s,\theta)Vπ​(s,θ) 对 θ\thetaθ 的导数，然后梯度上升法更新 θ\thetaθ 。

π\piπ 是关于 aaa 的概率密度函数，于是公式变为求关于随机变量动作 A 的期望：
(这里假设 QQQ 和 θ\thetaθ 独立，其实他们是相关联的: θ→π→Qπ\theta\rightarrow\pi\rightarrow Q_\piθ→π→Qπ​ )

∂V(s;θ)∂θ=∑a∂π(a∣s;θ)∂θ⋅Qπ(s,a)=∑aπ(a∣s;θ)⋅∂log⁡π(a∣s;θ)∂θ⋅Qπ(s,a)=EA∼π(⋅∣s;θ)[∂log⁡π(A∣s;θ)∂θ⋅Qπ(s,A)]\begin{aligned} \frac{\partial V(s ; \boldsymbol{\theta})}{\partial \boldsymbol{\theta}} & =\sum_a \frac{\partial \pi(a \mid s ; \boldsymbol{\theta})}{\partial \boldsymbol{\theta}} \cdot Q_\pi(s, a) \\ & =\sum_a \pi(a \mid s ; \boldsymbol{\theta}) \cdot \frac{\partial \log \pi(a \mid s ; \boldsymbol{\theta})}{\partial \boldsymbol{\theta}} \cdot Q_\pi(s, a) \\ & =\mathbb{E}_{A \sim \pi(\cdot \mid s ; \boldsymbol{\theta})}\left[\frac{\partial \log \pi(A \mid s ; \boldsymbol{\theta})}{\partial \boldsymbol{\theta}} \cdot Q_\pi(s, A)\right] \end{aligned} ∂θ∂V(s;θ)​​=a∑​∂θ∂π(a∣s;θ)​⋅Qπ​(s,a)=a∑​π(a∣s;θ)⋅∂θ∂logπ(a∣s;θ)​⋅Qπ​(s,a)=EA∼π(⋅∣s;θ)​[∂θ∂logπ(A∣s;θ)​⋅Qπ​(s,A)]​

于是我们得到了两种求导形式：

  1. ∑a∂π(a∣s;θ)∂θ⋅Qπ(s,a)\sum_a \frac{\partial \pi(a \mid s ; \boldsymbol{\theta})}{\partial \boldsymbol{\theta}} \cdot Q_\pi(s, a)∑a​∂θ∂π(a∣s;θ)​⋅Qπ​(s,a)
    适用于离散的形式，遍历每一种可能的动作，得到最终的和。

  1. EA∼π(⋅∣s;θ)[∂log⁡π(A∣s;θ)∂θ⋅Qπ(s,A)]\mathbb{E}_{A \sim \pi(\cdot \mid s ; \boldsymbol{\theta})}\left[\frac{\partial \log \pi(A \mid s ; \boldsymbol{\theta})}{\partial \boldsymbol{\theta}} \cdot Q_\pi(s, A)\right]EA∼π(⋅∣s;θ)​[∂θ∂logπ(A∣s;θ)​⋅Qπ​(s,A)]
    适用于连续的动作，这时有无穷种策略。
    使用蒙特卡洛近似，从 π\piπ 中进行抽样，得到 a^\hat aa^ ，然后计算中括号内部的值，得到近似的无偏估计：

g(a^,θ)=∂log⁡π(a^∣s;θ)∂θ⋅Qπ(s,a^)\boldsymbol g(\hat a , \boldsymbol \theta) = \frac{\partial \log \pi(\hat a \mid s ; \boldsymbol{\theta})}{\partial \boldsymbol{\theta}} \cdot Q_\pi(s, \hat a ) g(a^,θ)=∂θ∂logπ(a^∣s;θ)​⋅Qπ​(s,a^)

  1. Observe the state sts_tst​.
  2. Randomly sample action ata_tat​ according to π(⋅∣st;θt)\pi\left(\cdot \mid s_t ; \boldsymbol{\theta}_t\right)π(⋅∣st​;θt​).

近似 action value function: Q

如何求 QπQ_\piQπ​ ？ 两种方法：

  1. REINFORCE
    用策略网络 π\piπ 生成动作，得到完整轨迹： s1,a1,r1,....,sT,aT,rTs_1,a_1,r_1,....,s_T,a_T,r_Ts1​,a1​,r1​,....,sT​,aT​,rT​
    然后得到最终的「累计折扣回报 /return」 ut=∑k=tTγk−trku_t = \sum_{k=t}^T \gamma^{k-t}r_kut​=∑k=tT​γk−trk​
    然后用这个 utu_tut​ 去近似 QπQ_\piQπ​
  2. 使用另一个神经网络来近似 QπQ_\piQπ​ ，这样就有了两个神经网络，这就是 actor-critic method 。

actor-critic method

状态价值函数为：

Vπ(s)=EA[Qπ(s,A)]=∑aπ(a∣s)⋅Qπ(s,a)V_\pi(s) = \mathbb{E}_A\left[Q_\pi(s, A)\right] = \sum_a \pi \left(a \mid s\right) \cdot Q_\pi\left(s, a\right) Vπ​(s)=EA​[Qπ​(s,A)]=a∑​π(a∣s)⋅Qπ​(s,a)

其中需要 π\piπ 和 QπQ_\piQπ​ ，全都使用神经网络进行近似：

  1. policy network π(a∣s;θ)\pi(a|s; \theta)π(a∣s;θ) 决定下一步执行的动作，所以被称为 actor
  2. value network q(s,a;w)q(s,a;w)q(s,a;w) 近似 QπQ_\piQπ​ ，估计当前动作的预期价值，所以被称为 critic

于是就得到了：

V(s;θ;w)=∑aπ(a∣s;θ)⋅q(s,a;w)V(s;\theta;w) = \sum_a \pi \left(a \mid s; \theta\right) \cdot q\left(s, a;w\right) V(s;θ;w)=a∑​π(a∣s;θ)⋅q(s,a;w)

同时训练两个网络，

回到 RL

某个状态到最终状态的序列：

τ:st,st+1⋯ ,sT\tau: \quad s_t, s_{t+1} \cdots, s_T τ:st​,st+1​⋯,sT​

从形式上看，是一个 贯序决策问题 。
目标：找到最优策略 π:s→a\pi: s\rightarrow aπ:s→a ，让累计回报 R(τ)R(\tau)R(τ) 的期望最大。

累计回报是随机变量，无法求，这里转化为最大化期望的策略，即 max⁡π∫R(τ)pπ(τ)dτ\max _\pi \int R(\tau) p_\pi(\tau) d \taumaxπ​∫R(τ)pπ​(τ)dτ 。

累计回报：评价使用策略来完成任务的总回报。
所以直观目的为： 找到最优策略 ，让总回报最大，即更好地完成任务。

回报 reward 由回报函数 R 定义，对应了具体的任务。

分类

根据转移概率 P 是否已知，可以分为：

  1. 基于模型的动态规划方法
    模型可以告诉我们采用某种策略，即选定某个动作后，下一个状态是什么。这个性质可以让智能体无需采用确切的动作就可以知道下一个状态是什么，即不需要探索。
  2. 基于无模型的强化学习方法：只能使用各种总方法来评估当前行为的好坏，见下一章节。

两种方法都包含了三个子方法：

  1. 策略迭代
  2. 值迭代
  3. 策略搜索

动态规划的核心是找到最优策略，让值函数最大。

策略迭代算法

包含策略评估和策略改善，先用已知初始化的默认策略求状态价值，作为对当前策略的评估，然后利用状态价值来调整现在的策略，循环往复。

而给定一个 策略，怎么求值函数呢 ？ -> 策略评估

vπ(s)=∑a∈Aπ(a∣s)(Rsa+γ∑s′∈SPss′avπ(s′))v_\pi(s)=\sum_{a \in A} \pi(a \mid s)\left(R_s^a+\gamma \sum_{s^{\prime} \in S} P_{s s^{\prime}}^a v_\pi\left(s^{\prime}\right)\right) vπ​(s)=a∈A∑​π(a∣s)(Rsa​+γs′∈S∑​Pss′a​vπ​(s′))

在已知策略 π\piπ 时，状态 s 的价值可以利用后续所有可能状态 s’ 进行计算。
而 s’ 的价值也是未知的，可以通过 s’’ 进行计算，这就是一种 bootstrap 。

可以根据策略求出来值函数了，那么怎么找到最优策略呢？-> 策略改善
最直观的方法就是贪心搜索，将状态值函数最大的状态作为下一个目标状态，进而得到 act 。

不断进行「策略评估」 + 「策略提升」，
停止条件为预先设置的超参数 k 轮或者策略不再改变。

值函数迭代算法

存在一个问题：我们在进行策略评估时，使用了迭代法求解，目的是让状态价值收敛，来逼近真实状态价值。但是我们是否真的需要每一次都迭代很多次，来进行这个逼近的过程呢？
答案是不，一般迭代十次和无穷次得到的贪婪策略都是一样的。所以我们不需要等到策略评估算法完全收敛。

如果，只评估一次，就进行策略改善，则称 值函数迭代算法 ，是动态规划 最一般的计算框架 。

在每次迭代时，仍然需要遍历一次状态空间，得到每个状态的新的状态价值，然后更新策略。
