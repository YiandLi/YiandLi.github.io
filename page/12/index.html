A Hybrid Convolutional Variational Autoencoder for Text Generation

本文讨论了 architectural choices 对于 VAE text generation 产生的影响。
在生成文本时，训练VAE模型的核心困难是 KL Vanish，导致 decoder 退化为一个普通的语言模型。

模型核心：是一个由「一维卷积层 one-dimensional convolutional」和「反卷积层 deconvolutional layer」组成的前馈结构。
效果：更快收敛，更好地处理长序列的能力，避免在文本数据上训练VAE模型的一些主要困难（KL vanish）。

yaml 传入参数

  • 基本语法规则：

    • 大小写敏感
    • 使用缩进表示层级关系
    • 缩进时不允许使用 Tab键，只允许使用空格。
    • 缩进的空格数目不重要，只要相同层级的元素左侧对齐即可
    • # 表示注释，从这个字符一直到行尾，都会被解析器忽略，这个和python的注释一样

  • 读取方式：通过open()方法读取文件数据，再通过yaml.load(open_file ,Loader=yaml.FullLoader)函数将数据转化为列表或字典

  • 保存方式：yaml.dump(data,f,encoding='utf-8',allow_unicode=True)

纯量

  1. 数字默认 int 和 float 类型
  2. 布尔值 用 true 和 false 表示
  3. None 用 ~ 表示
  4. 使用两个感叹号，强制转换数据类型。
    n6: !!str 123 --> {'n6': '123'}
