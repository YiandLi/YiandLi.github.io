Meta Learning Adaptive Hyper-parameters

Posted on 2022-11-12 In Machine Learning , Meta Learning
Symbols count in article: 932 Reading time ≈ 5 mins.

Paper: NeurIPS 2020 - Meta-Learning with Adaptive Hyper-parameters
Code: https://github.com/baiksung/ALFA

Model-agnostic meta-learning (MAML) encodes the prior knowledge into a trainable initialization, which allowed for fast adaptation to few examples.
But MAML have an effectiveness problem of initialization especially when test tasks are different from training tasks.

Instead of searching for a better initialization, the author propose the Adaptive Learning of hyperparameters for Fast Adaptation (ALFA), an inner-loop optimization (or fast adaptation), which is a new weight update rule to accelerate the adaption process.

Specifically, a small meta-network is used to adaptively generate per-step hyper-parameters, including learning rate and weight decay coefficients.

Surprisingly, fast adaptation from random initialization with ALFA can already outperform MAML.

Introduction

Meta-learning, or learning-to-learn, investigates common prior knowledge from previous tasks for a rapid adaption ability with new tasks.
Recent meta-learning algorithms can be divided into three main categories:

  1. metric-based: learning feature space
  2. network-based: designing network architecture
  3. gradient-based: adjusting the optimization algorithm itself
    there are two major directions: learning the initialization and learning the update rule.

Model-agnostic meta-learning (MAML) aims to find a good starting location for the network parameters, from which new tasks are learned with few updates.
Most work focused on learning a better initialization. And the strategy of fast adaptation within task is only dependent on conventional optimizers (e.g. SGD). In other words, better learning algorithms for inner-loop optimization is relatively overlooked.

This work proposes to improve the adaptation property in weight updates, which is to enable training to be more effective with task-conditioned inner-loop updates.
The algorithm dynamically generates two important hyperparameters for optimization: learning rates and weight decay coefficients.
For each step, the current weight and gradient values are used to dynamically generate these hyperparameters, enabling each inner-loop iteration to be adaptive to the given task.

MAML has the problem of overfitting , for its sample inner-loop update rules. So regularization could be added to prevent overfitting during fast adaptation.
