Meta Re-weighting Self-Augmentation NER

Posted on 2022-12-11 In NLP , Named entity recognition
Symbols count in article: 1.3k Reading time ≈ 6 mins.

Paper:Robust Self-Augmentation for Named Entity Recognition with Meta Reweighting

NER任务也有一些数据增强的方法，但是因为该任务侧重于 token-level，所以一些构造出来的增强样本有可能会带来噪音。所以我们最好可以衡量一下构造出来的样本到底靠不靠谱。

本文作者首先介绍了适用于NER任务的两种数据增强方法，然后介绍一种衡量样本重要性（受到meta-learning启发）的方法。通过这种方法，我们可以得到每个增强样本对总损失的权重（重要性）。

Base model

Bert + BiLSTM + CRF

Self-Augmentation

Token Substitution (TS)

替换原原始序列的 token，作者先构造同义词字典，同时包含实体和非实体：

  • 实体同义词：相同标签的实体均为同义词。
  • 非实体同义词：训练 word2vec，得到距离最接近的前 k 个词作为同义词。

使用同义词字典进行替换，替换真实实体操作称为 entity mention substitution (EMS)，替换非实体操作称为 normal word substitution (NWS)。

同时作者引入超参数 γ\gammaγ 来控制 EMS 的比重。

Mixup for CRF

使用 mixup 融合两个真实样本 (X1,Y1),(X2,Y2)(X_1, Y_1), (X_2, Y_2)(X1​,Y1​),(X2​,Y2​) 。
首先用 bert 进行编码得到 e1,1⋯e1,n1\boldsymbol{e}_{1,1} \cdots \boldsymbol{e}_{1, n_1}e1,1​⋯e1,n1​​ 和 e2,1⋯e2,n2\boldsymbol{e}_{2,1} \cdots \boldsymbol{e}_{2, n_2}e2,1​⋯e2,n2​​ 。
然后使用线性相加的方式进行融合：

Xˉ:{BERT⁡(Xˉ)=e‾1⋯e‾ne‾i=λe1,i+(1−λ)e2,i,i∈[1,n]Yˉ:s(Yˉ∣Xˉ)=λs(Y1∣Xˉ)+(1−λ)s(Y2∣Xˉ)\begin{aligned} & \bar{X}:\left\{\begin{array}{c} \operatorname{BERT}(\bar{X})=\overline{\boldsymbol{e}}_1 \cdots \overline{\boldsymbol{e}}_n \\ \overline{\boldsymbol{e}}_i=\lambda \boldsymbol{e}_{1, i}+(1-\lambda) \boldsymbol{e}_{2, i}, i \in[1, n] \end{array}\right. \\ & \bar{Y}: s(\bar{Y} \mid \bar{X})=\lambda s\left(Y_1 \mid \bar{X}\right)+(1-\lambda) s\left(Y_2 \mid \bar{X}\right) \end{aligned} ​Xˉ:{BERT(Xˉ)=e1​⋯en​ei​=λe1,i​+(1−λ)e2,i​,i∈[1,n]​Yˉ:s(Yˉ∣Xˉ)=λs(Y1​∣Xˉ)+(1−λ)s(Y2​∣Xˉ)​

这里的 (Y1∣Xˉ)\left(Y_1 \mid \bar{X}\right)(Y1​∣Xˉ) 和 (Y2∣Xˉ)\left(Y_2 \mid \bar{X}\right)(Y2​∣Xˉ) 其实就是使用融合的 Xˉ\bar{X}Xˉ 做解码得到 Y1Y_1Y1​ 和 Y2Y_2Y2​ 的分数，然后再线性相加。

Meta Reweighting

重头戏来了，得到增强样本之后，我们怎么确定每个增强样本的重要性呢？
首先重要性值我们表示成一个权重，越重要，对 loss 的贡献权重越大。

数据集

  1. 原始样本集合： D={(Xim,Yim)}i=1M\mathcal{D}=\left\{\left(X_{i}^{m}, Y_{i}^{m}\right)\right\}_{i=1}^{M}D={(Xim​,Yim​)}i=1M​
    该样本集合也被称之为 meta example set ， 因为它不是全量的标注集，只是一个很小的(M≪NM\ll NM≪N) 子集合。是用来找后文的 wiw_iwi​ 的，同时使用了该集合上的损失，所以被称为 “meta” 。

  2. 增强样本集合：D^={(Xi,Yi)}i=1N\mathcal{\hat{D} }=\left\{\left(X_{i}, Y_{i}\right)\right\}_{i=1}^{N}D^={(Xi​,Yi​)}i=1N​

过程

  1. 使用 增强样本集合 D^\mathcal{\hat{D} }D^ 更新模型 Θ\ThetaΘ ：

Θ∗(w)=arg⁡min⁡Θ∑i=1NwiL(f(Xi;Θ),Yi)\Theta^{*}(w)=\underset{\Theta}{\arg \min } \sum_{i=1}^{N} w_{i} \mathcal{L}\left(f\left(X_{i} ; \Theta\right), Y_{i}\right) Θ∗(w)=Θargmin​i=1∑N​wi​L(f(Xi​;Θ),Yi​)

这里 wiw_iwi​ 表示每个样本对损失的贡献权重，即重要程度，是可训练的参数。

  1. 使用这个新模型在 meta example set D\mathcal{D}D 上推理，得到损失 L(f(Xim;Θ∗(w)),Yim)\mathcal{L}\left(f\left(X_{i}^{m} ; \Theta^{*}(w)\right), Y_{i}^{m}\right)L(f(Xim​;Θ∗(w)),Yim​) 。

  2. 用该损失更新 wiw_iwi​ ：

w∗=arg⁡min⁡w1M∑i=1ML(f(Xim;Θ∗(w)),Yim)w^{*}=\underset{w}{\arg \min } \frac{1}{M} \sum_{i=1}^{M} \mathcal{L}\left(f\left(X_{i}^{m} ; \Theta^{*}(w)\right), Y_{i}^{m}\right) w∗=wargmin​M1​i=1∑M​L(f(Xim​;Θ∗(w)),Yim​)

其实很好理解，可以用meta learning的方式理解，把 D\mathcal{D}D 看作outer loop set，把 D^\mathcal {\hat{D}}D^ 看作 inner loop set。

或者就是以神经网络的方法，把权重看作 MLP 的边，自然可以通过梯度进行拟合了。

个人觉得有点像之前的一些「根据梯度判别样本重要程度」的方法。
