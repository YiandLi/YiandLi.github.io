shell 脚本命令

Shell 脚本（shell script），是一种为 shell 编写的脚本程序。
本教程关注的是 Bash，也就是 Bourne Again Shell，由于易用和免费，Bash 在日常工作中被广泛使用。同时，Bash 也是大多数Linux 系统默认的 Shell。

source 命令

也称为“点命令”，通常用命令 . 来替代。
用于重新执行刚修改的初始化文件，使之立即生效，而不必注销并重新登录。
与sh命令的区别：

  • 当shell脚本具有可执行权限时，都用于执行某一个脚本或者命令，即sh filename与source filename没有区别。
  • sh filename 会重新建立一个「子shell」，在「子shell」中执行脚本里面的语句，该「子shell」继承「父shell」的环境变量，但「子shell」是新建的，其改变的变量不会被带回「父shell」，除非子shell中使用 export 命令设置环境变量。
  • source filename 读取脚本里面的语句依次在当前shell里面执行，没有建立新「子shell」。那么脚本里面所有新建、改变变量的语句都会保存在当前shell里面。

echo 打印输出命令

  • help echo：查看帮助
  • /user/bin/echo --version：查看版本，前面是全路径，可以使用which echo得到。
  • echo [Option][字符串]：打印字符串，加'单引号。
    • 想打印单双引号，则需要相互嵌套。
    • 打印环境变量 echo $USER
    • 还可以指定打印字体的样式（大小，颜色，粗细 …）
    • 当有转译字符(\n，\t，\t，\r)时，需要加 -e 才生效。
  • 区分 单引号，双引号，反引号
    • 单引号：不能解析$变量名，即全部当成字符串解析
    • 双引号：可以解析变量的值
    • 反引号：内容会作为系统命令执行，打印返回值。

变量

  • 定义：变量名=变量值，不要加空格

    • 定义时不需要加美元符号 $
    • 变量名只包含 [英文,数字,字母,下划线] ，不能以数字开头
    • 变量默认类型都是 字符串类型，无法直接进行数值运算

  • 引用，即使用定义过的变量：$变量名/${变量名}。

设置默认参数

  • ${a-b}：变量为null时，即命令行没有传入该变量时，设置默认值为 b。如果传入的是空字符串 ""，那么还会赋值变量a=""。

  • ${a:-b}：变量为null或者为 空字符串 "" 时，设置默认值为 b。

  • ${变量名:=新值}：如果「变量」存在，则返回「变量值」，否则返回「新值」，同时将「新值」赋给「变量」

使用示例

括号特殊用法

()：子shell完成，内部变量不影响当前shell中的变量，可以看作一个方法。
{}：当前shell完成，会影响当前变量。

逻辑符号

  • ;：隔开每个命令, 每个命令按照从左到右的顺序执行，彼此之间不关心是否失败，所有命令都会执行。
  • &：任务在后台执行，这样该命令在执行时，我们仍然可以操作控制台。
    • e.g., python3 run.py &
  • &&：表示前一条命令执行成功时，才执行后一条命令
    • e.g., echo '1‘ && echo ‘2’
  • |：表示管道，上一条命令的输出，作为下一条命令参数
    • e.g., ls | grep "aa"：在ls中的结果中搜索含有 "aa" 的字符串
    • e.g., echo 'yes' | wc -l >>> 1
  • ||：上一条命令执行失败后，才执行下一条命令
    • e.g., cat nofile || echo "fail"

条件判断

[ condition ]（注意 condition 前后要有空格）

  1. 整数之间比较
    -eq 等于（equal）
    -ne 不等于（not equal）
    -lt 小于（less than）
    -le 小于等于（less equal）
    -gt 大于（greater than）
    -ge 大于等于（greater equal）
    = 字符串相等，!=字符串不相等
  1. 文件权限
    -r：可以读
    -w：可以写
    -x：可以执行
  1. 按照文件类型进行判断
    -e 文件存在（existence）
    -f 文件存在并且是一个常规的文件（file）
    -d 文件存在并且是一个目录（directory）

比较 for 循环中的 $* 和 $@

$* 和 $@ 都表示传递给函数或脚本的所有参数
不被双引号 "" 包含时，都以 $1 $2 …$n 的形式输出所有参数。
而被双引号包围时，$*会将所有的参数作为一个整体，以“$1 2…2 …2…n”的形式输 出所有参数；$@会将各个参数分开，以“$1” “2”…“2”…“2”…“n”的形式输出所有参数。
