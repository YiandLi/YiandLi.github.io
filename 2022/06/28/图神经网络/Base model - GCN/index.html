图卷积：从GCN到GAT、GraphSAGE
图卷积神经网络笔记——第二章：谱域图卷积介绍（2）

两种方式来提取图的特征：

  • 空间域卷积（spatial domain），由于每个顶点提取出来的neighbors不同，处理上比较麻烦，效果没有频域卷积效果好。
  • 频域卷积（spectral domain）

GNN 网络：

  • 和普通的 DNN 网络不同，DNN 网络架构和输入输出无关。
  • 而一般 GNN 网络架构和输入的图有关，它把给定的图结构作为一层，每个时间步会经过一次图上的信息传递/聚合/卷积。

每一层输入节点集合： hl={h→1,h→2,…h→N},h→i∈RF\mathrm{h}^l=\left\{\overrightarrow{\mathrm{h}}_1, \overrightarrow{\mathrm{h}}_2, \ldots \overrightarrow{\mathrm{h}}_{\mathrm{N}}\right\}, \overrightarrow{\mathrm{h}}_{\mathrm{i}} \in \mathrm{R}^{\mathrm{F}}hl={h1​,h2​,…hN​},hi​∈RF ，每个节点隐藏层维度为 FFF 。

每一层输出节点集合： hl+1={h′→1,h′→2,…h′→N},h′→i∈RF′\mathrm{h}^{l+1}=\left\{\overrightarrow{\mathrm{h'}}_1, \overrightarrow{\mathrm{h'}}_2, \ldots \overrightarrow{\mathrm{h'}}_{\mathrm{N}}\right\}, \overrightarrow{\mathrm{h'}}_{\mathrm{i}} \in \mathrm{R}^{\mathrm{F'}}hl+1={h′1​,h′2​,…h′N​},h′i​∈RF′ ，每个节点隐藏层维度为 F′F'F′ 。

则这一层的权重矩阵为 W∈RF×F′W\in \mathrm R^{F\times F'}W∈RF×F′ 。

  • GCN 类型和 deepwalk 类型网络区别
    可以迁移到 word2vec 和 bert 类型，所区别的静态建模和动态建模（注意区分静态/动态词向量，建模和词向量不是一回事）
    • deepwalk ：建模时候只考虑了结构关系进行建模，节点特征只在计算时用到，即只在计算损失时使用节点特征。
    • GCN 类：建模时考虑了节点特征，用节点特征计算了融合的权重。
      其实从这一点上看 gcn 并没有用到节点特征，因为是根据邻接矩阵图进行平权融合；反观 gtn 就使用到了节点特征计算权重

基于 随机游走 的方法在获取节点的 embedding 过程中只考虑了图的结构信息，而节点的特征是在后期加入的，特征并没有经过模型进行抽取或变化。
而 GCN 等卷积类模型除了考虑图本身结构的信息之外，还加入了节点的特征进行计算，因此可以直接完成端到端的任务。

GCN

深度之眼Paper带读笔记GNN.08.GCN
代码：怎么搭好一个GCN？只需这四步

公式为：

Hl+1=σ(D−1/2AD−1/2HlWl)H^{l+1}=\sigma\left(D^{-1 / 2} A D^{-1 / 2} H^l W^l\right) Hl+1=σ(D−1/2AD−1/2HlWl)

  • DDD 为度矩阵， AAA 为邻接矩阵 + I ：

邻接矩阵 AAA 控制应该融合哪些点, HHH 表示上一层的embedding参数， AHAHAH 起到了邻居特征的汇聚效果。
度矩阵 DDD 控制节点权重。其中，度越大，表示该节点的邻居节点越多，则其信息量就越少，所以权重 D−1/2D^{-1 / 2}D−1/2 就越小。

  • 实现上：
    D−1/2AD−1/2D^{-1 / 2} A D^{-1 / 2}D−1/2AD−1/2 每一行的和为 1，起到了 normalization 的效果。
    D−1/2AD−1/2D^{-1 / 2} A D^{-1 / 2}D−1/2AD−1/2 相当于对邻接矩阵 A 进行了标准化，比如 [1,1,1,0,0,0] 就成为了 [1/3, 1/3, 1/3, 0,0,0]。
    然后乘以特征矩阵 HHH ，相当于目标节点的新特征等于了该节点和邻居节点的 mean pooing 。

  • WlW^lWl 是这一层训练的权重，相当于对特征矩阵进行投影变换，进行维度转换。

  • σ\sigmaσ 一般为 ReLu 激活函数。

优点和缺点

优点：
之前基于随机游走的方法都 只使用了结构信息 ，也就是这里的 D−1/2AD−1/2D^{-1 / 2} A D^{-1 / 2}D−1/2AD−1/2 进行建模。
而 GCN 额外使用了 WlW^lWl ，即节点的特征

缺点：

  1. 融合时 边权值 是 固定 的，都是邻居节点的倒数 ，不够灵活。
  2. 可扩展性差，因为它是 全图卷积融合 ，全图做梯度更新，当图比较大时，这样的方式就太慢了，不合适。
  3. 层数加深时，结果会极容易平滑， 每个点的特征结果都十分相似。

GAT

Graph Attention Networks

GCN 邻居节点的权重都是相同的，GAT 想让模型更灵活点，加个 attention ，让模型自己去学习权重。

节点 i，j 之间的注意力权重 αi,j\alpha_{i,j}αi,j​ 为：

αij=exp⁡(LeakyReLU⁡(a→T[Wh⃗i∥Wh⃗j]))∑k∈Niexp⁡(LeakyReLU⁡(a→T[Wh⃗i∥Wh⃗k]))\alpha_{i j}=\frac{\exp \left(\operatorname{LeakyReLU}\left(\overrightarrow{\mathbf{a}}^T\left[\mathbf{W} \vec{h}_i \| \mathbf{W} \vec{h}_j\right]\right)\right)}{\sum_{k \in \mathcal{N}_i} \exp \left(\operatorname{LeakyReLU}\left(\overrightarrow{\mathbf{a}}^T\left[\mathbf{W} \vec{h}_i \| \mathbf{W} \vec{h}_k\right]\right)\right)} αij​=∑k∈Ni​​exp(LeakyReLU(aT[Whi​∥Whk​]))exp(LeakyReLU(aT[Whi​∥Whj​]))​

其中 W,a→\mathbf{W} , \overrightarrow{\mathbf{a}}W,a 都是可训练参数。

得到注意力分数之后，进行融合：

h⃗i′=σ(∑j∈NiαijWh⃗j)\vec{h}_i^{\prime}=\sigma\left(\sum_{j \in \mathcal{N}_i} \alpha_{i j} \mathbf{W} \vec{h}_j\right) hi′​=σ⎝⎛​j∈Ni​∑​αij​Whj​⎠⎞​

还可以扩展为 multi-head attention（K代表K个attention head：

h⃗i′=∥k=1Kσ(∑j∈NiαijkWkh⃗j)\vec{h}_i^{\prime}=\|_{k=1}^K \sigma\left(\sum_{j \in \mathcal{N}_i} \alpha_{i j}^k \mathbf{W}^k \vec{h}_j\right) hi′​=∥k=1K​σ⎝⎛​j∈Ni​∑​αijk​Wkhj​⎠⎞​

其实就是分别对应到不同的 $\mathbf{W}^k $ 矩阵就可以了， $\mathbf{W}^k $ 决定了 h⃗i′\vec{h}_i^{\prime}hi′​ 最终的维度。
∥\|∥ 表示拼接。

使用这种拼接的方式输出的 向量维度 太大了，所以可以改成累加平均的方式：

h⃗i′=σ(1K∑k=1K∑j∈NiαijkWkh⃗j)\vec{h}_i^{\prime}=\sigma\left(\frac{1}{K} \sum_{k=1}^K \sum_{j \in \mathcal{N}_i} \alpha_{i j}^k \mathbf{W}^k \vec{h}_j\right) hi′​=σ⎝⎛​K1​k=1∑K​j∈Ni​∑​αijk​Wkhj​⎠⎞​

Graph-SAGE

Inductive Representation learning on Large Graphs
深度之眼Paper带读笔记GNN.07.GraphSAGE

SA: sample
GE: aggregate

GCN 和 GAT 做卷积融合是 全图 的，梯度是基于全图更新，若是图比较大，每个点邻居节点也较多，这样的融合效率必然很低。

  • transductive learning 和 inductive learning
    • transductive learning ：要预测的数据在训练时模型也能看到，即推理时使用的图就是训练时的图，和 GCN/GAT 一；不太能兼顾到图的拓展性。
    • inductive learning ：要预测的数据在训练时模型不用看到，即正常任务的形式，训练集和测试集相互独立。这样的训练方式支持图的拓展。

GraphSAGE 就是 inductive learning 的模式，GraphSAGE 提出随机采子图的方式去采样。
具体地，有一个超参数定一个采样子图的大小，即对于某一个中心节点，会采样多少个邻居节点。
然后将采样出来的 节点子图 作为样本组成 batch 中，通过子图更新 node embedding 。
这样采出的子图结构本身就是变化，从而让模型学到的是一种 聚合函数 。

优点：

  • 解决 unseen node 的情况
  • 不需要采样全图，大大提高训练效率
  • 增强图的拓展性


过程：

  • 采样子图：对于每一个节点采用切子图的方法，随机sample出部分的邻居点，作为聚合的feature点。
  • 聚合：采出子图后，做feature聚合，这里可以选择不同聚合方式。
  • 任务预测：使用最后一层的 node embedding 进行下游任务的预测，训练。

直推式+归纳式学习

无监督（直推式）

目标：nearby nodes 具有相似的embedding表示，disparate nodes 有不同的embedding表示。

Jg(zu)=−log⁡(σ(zu⊤zv))−Q⋅Evn∼Pn(v)log⁡(σ(−zu⊤zv))\mathrm{J}_{\mathrm{g}}\left(\mathrm{z}_{\mathrm{u}}\right)=-\log \left(\sigma\left(\mathrm{z}_{\mathrm{u}}^{\top} \mathrm{z}_{\mathrm{v}}\right)\right)-\mathrm{Q} \cdot \mathrm{E}_{\mathrm{v}_{\mathrm{n}} \sim \mathrm{P}_{\mathrm{n}}(\mathrm{v})} \log \left(\sigma\left(-\mathrm{z}_{\mathrm{u}}^{\top} \mathrm{z}_{\mathrm{v}}\right)\right) Jg​(zu​)=−log(σ(zu⊤​zv​))−Q⋅Evn​∼Pn​(v)​log(σ(−zu⊤​zv​))

目标节点为 vvv ，使用 fixed-length random walk 的方法选择和节点 vvv 共现的节点 uuu 作为正样本。
Pn(v)\mathrm{P}_{\mathrm{n}}(\mathrm{v})Pn​(v) 是 negative sampling distribution， QQQ 表示负采样中的负样本数量。

这个无监督的损失函数和 random walk 损失函数思想是一样的，都是正样本的表征要越近越好，负样本的表征要越远越好，而且这里也是用到了负采样技术。

唯一不同的是节点向量的生成方式：

  • deep walk 直接随机初始化
  • 这里使用邻居节点的向量进行聚合得到当前节点的向量表示（generated from the features contained within a node’s local neighborhood）

有监督（归纳式）

使用最后一层的节点表征进行下游任务的监督训练，即替换了上面的损失函数，不再需要负采样了。

算法步骤

N\mathcal NN ：使用 1/0 来表示邻居节点
第四步表示聚合 邻居节点 的方法，比如 max/mean/ …
第五步表示将自己的信息和邻居节点的聚合信息再次整合
第七步表示归一化操作

Nk\mathcal N_kNk​ 表示随机采样某个节点的邻居节点 ，这里的下标 kkk 表示每一层的随机采样率都是不同的，独立的。文中给了一个例子，当 K=2，并且 k=1,k=2 时对应的采样个数为 S1, S2S_1,\ S_2S1​, S2​ ，那么其实就采样了中心节点的 S2S_2S2​ 个 one-hop neighbor， $S_1S_2 $ 个 two-hop neighbor 。

总结

通用流程

  1. 定义一个 聚合方式 ，即如何使用上一层的「邻居 embedding」和「self embedding」得到这一层新的「self embedding」，一般这一步需要使用到可学习的参数。

hvk=σ([Ak⋅AGG⁡({huk−1,∀u∈N(v)}),Bkhvk−1])\mathbf{h}_v^k=\sigma\left(\left[\mathbf{A}_k \cdot \operatorname{AGG}\left(\left\{\mathbf{h}_u^{k-1}, \forall u \in N(v)\right\}\right), \mathbf{B}_k \mathbf{h}_v^{k-1}\right]\right) hvk​=σ([Ak​⋅AGG({huk−1​,∀u∈N(v)}),Bk​hvk−1​])

AGG 表示使用什么聚合方式将邻居节点进行聚合，e.g. mean, max, attn, lstm
Ak,Bk\mathbf{A}_k, \mathbf{B}_kAk​,Bk​ 为聚合邻居和自己的 mapping 学习参数。每一层 kkk 有自己专属的权重。
∀u∈N(v)\forall u \in N(v)∀u∈N(v) 表示点 vvv 的邻居节点

  1. 定义一个损失函数值，使用最后一层的「node embedding」得到损失并且更新网络。
  2. 选择节点（GraphSAGE，batch的算法），对于每一个点来说都有一个对应的多跳计子算图。每个计算子图是一个样本，组成 batch。
  3. 训练得到第一步聚合方式中的 网络权重，这里的权重表示模型学习到了聚合信息的能力。
  4. 推理阶段：使用网络权重对新的子图直接进行推理。
