A Unified View Of Parameter Efficient Transfer Learning

Posted on 2022-08-09 In NLP , Prompt
Symbols count in article: 2.2k Reading time ≈ 11 mins.

论文：TOWARDS A UNIFIED VIEW OF PARAMETER-EFFICIENT TRANSFER LEARNING

Abstract and Introduction

parameter-efficient transfer learning methods：冻结 PLM 的参数，只微调一小部分额外的参数。模型可以避免灾难性遗忘，并且快速迁移到新的 task 上。
但是典型的三种方法（adapter，prefix-tuning，low-rank matrices）的可解释性都比较低，并且他们之间看似没有什么联系。

所以作者

  1. 提出了一个统一的视角，找到他们之间的联系：全都是对模型内部特定层进行处理
  2. 通过实验，分析不同的 design element 的表现
  3. 提出一个更有效的方法

OVERVIEW OF PREVIOUS PARAMETER-EFFICIENT TUNING METHODS


三种主流方法，都是冻结 PLM 参数，仅仅微调新增参数。

Adapters

  • 机制：将 Adapter 插入在不同层（FFN，ATTN）之间
  • 计算方式：使用一个 bottlenet 处理输入，然后激活函数，然后残差连接。
    f(⋅)f(\cdot)f(⋅) 为激活函数， Wdown ∈Rd×r\boldsymbol{W}_{\text {down }} \in \mathbb{R}^{d\times r}Wdown ​∈Rd×r ， Wup ∈Rr×d\boldsymbol{W}_{\text {up }} \in \mathbb{R}^{r\times d}Wup ​∈Rr×d ， rrr 为 bottleneck 中的降维维度。

Houlsby et al. (2019) 将两个 Adapter 分别放到 multi-head attention 和 FFN 层之后。
Pfeiffer et al. (2021) 将一个 Adapter 放到 FFN “add & layer norm” 之后。

Prefix Tuning

  • 机制：将 lll 个 prompt vectors 放在每个multi-head attention 的 key 矩阵和 value 矩阵之前。
  • 计算方式：相当于每个原始 token 要多和 lll 个 soft prompt token 计算相似度，然后聚合。

Prompt-tuning (Lester et al., 2021) 直接在 embedding 层插入 soft prompt，相当于多了 prompt 的 attention 变换和 FFN 变换。

LoRA

  • 机制：将 trainable low-rank matrices，插入到模型层中，使用「类似残差连接的方式」模拟「参数的更新」。

W+ΔW=W+Wdown Wup \boldsymbol{W}+\Delta W=\boldsymbol{W}+\boldsymbol{W}_{\text {down }} \boldsymbol{W}_{\text {up }} W+ΔW=W+Wdown ​Wup ​

  • 计算方式：LoRA 对 multi-head attention 层中的 query 和 value 矩阵进行上述操作，将其对 embedding 的变换输出转化为：

h←h+s⋅xWdown Wup \boldsymbol{h} \leftarrow \boldsymbol{h}+s \cdot \boldsymbol{x} \boldsymbol{W}_{\text {down }} \boldsymbol{W}_{\text {up }} h←h+s⋅xWdown ​Wup ​

s≥1s \ge 1s≥1 ，是一个超参数。

The Difference from Adapters

Prefix Prompt 中引入了门控变量 λ\lambdaλ ，带来了一些新特性：

  • Prefix Prompt 是直接对输入序列即 attention layer 的输入做变换，但是 Adapter 是对 layer 的输出做变换。
    所以，Prefix Prompt 可以看作是一种 parallel (to layer) 的计算方式，而 Adapter 是一种 sequential 的计算方式。
  • Prefix Prompt 只能对 attention layer 做变换，但是 Adapter 对 attention layer 和 FFN layer 都可以做变换。
  • Prefix Prompt 可以对 attention layer 的每个注意力头做变换，但是 Adapter 只能对整体做变换（as single-headed）。
    这使得 Prefix Prompt 更容易地进行 full-rank update，即只要保证 l≥d/Nhl\ge d/N_hl≥d/Nh​ 即可，但是 Adapter 必须保证 r≥dr\ge dr≥d 。

THE UNIFIED FRAMEWORK

本质上，上述三种方法全都是学习一个 modification vector Δh\Delta \boldsymbol hΔh ，作用在 h\boldsymbol hh 上：

  • attention 输入：对应 parallel 模式，典型的 Prefix Prompt。
  • attention 输出：对应 sequential 模式，典型的 Adapter。
  • FFN 输入：一般不适用，因为 prefix prompt 是被直接映射的，不会改变原始序列。但是可以进行抽象化： h=h+Δh\boldsymbol h = \boldsymbol h + \Delta \boldsymbol hh=h+Δh 。
  • FFN 输出：对应 sequential 模式，典型的 Adapter。

其实就是 改变一个变换 ：

  • 在计算变换的同时进行改变：parallel，使用输入计算 Δh\Delta \boldsymbol hΔh
  • 在计算变换后进行改变：sequential，使用输出计算 Δh\Delta \boldsymbol hΔh

同时，在计算 Δh\Delta \boldsymbol hΔh 时，不同方法也有一些不同，但是计算 Δh\Delta \boldsymbol hΔh 时都是 projdown→nonlinear→projup{ proj_{down} → nonlinear → proj_{up}}projdown​→nonlinear→projup​ ，作者归纳为：

Multi-head Parallel Adapter 是将 Parallel Adapter 作用在 每个注意力头 上，实现方法类似 prefix tuning。

EXPERIMENTS

比较繁琐，不赘述，主要说下结论。

THE RESULTS OF EXISTING METHODS


尽管在 GLUE 上，很多 paper 中提出的方法效果都很好，仅微调很少的参数就能比肩 fine-tune，但是在其他数据集上这些方法还是比不上 fine-tune，表示其对任务的泛化性不佳。

WHICH INSERTION FORM – SEQUENTIAL OR PARALLEL?

prefix tuning 在 keys 和 values 向量前拼接 l 个向量，引入参数量为 2×l×d2\times l\times d2×l×d 。
adapter 需要引入两个矩阵 Wdown \boldsymbol{W}_{\text {down }}Wdown ​ 和 Wup \boldsymbol{W}_{\text {up }}Wup ​ ，引入参数量为 2×r×d2\times r\times d2×r×d 。

  • prefix > attn sequential adapter，parallel adapter > sequential adapter：parallel 方法比 sequential 方法好。
    可能有疑问 Adapter 不是 sequential 的吗，还能 parallel ？
    这里做个区分，这里的 adapter 仅表示 +Δh+ \Delta \boldsymbol h+Δh ，而非传统的 sequential 方法。

WHICH MODIFIED REPRESENTATION – ATTENTION OR FFN?


  • 微调参数量比较大时，FFN modification > ATTN modification：作者认为是因为 FFN 学习到的是 task-specific patterns
  • 微调参数量很小时(eg. 0.1%)，ATTN modification > FFN modification。

WHICH COMPOSITION FUNCTION?

三种方法：

  • simple addition (adapter)
  • gated addition(prefifix tuning)
  • scaled addition (LoRA)

结论：scaling composition > additive，没讨论 softmax gating 的情况

AN EFFECTIVE INTEGRATION BY TRANSFERRING FAVORABLE DESIGN ELEMENTS

首先是三个结论

  1. Scaled parallel adapter is the best variant to modify FFN;
  2. FFN can better utilize modification at larger capacities;
  3. Modifying head attentions like prefix tuning can achieve strong performance with only 0.1% parameters.

然后基于结论，作者提出他们自己的模型 Mix-And-Match adapter (MAM Adapter)：

  1. 在 attention sub-layer 上使用 prefix tuning (l=30)
  2. 在 FFN 上使用 scaled parallel adapter (r=512)
